## 1. 직무윤리 (5)

### 01 직무윤리

**강조의 이유 :** 윤리경쟁력의 시대 (시대의 변화, 거래비용, 생산성, 신종자본, 인프라, 사회적 비용감소)

**금융 투자업** : 산업의 고유속성(이해상충), 상품의 특성(원본손실), 소비자의 변화, 안전장치

**성격 :** 사전예방기능, 강제적 제재수단

**적용대상**

- 투자권유자문인력 등 관련 전문자격증 보유자
- 자격증은 없으나 관련 업무에 실질적으로 종사하는 자
- 업무와 관련되어 있는 자
- 잠재적 고객을 대상으로 투자관련업무를 수행하는 자

### **02 금융투자회사 표준윤리준칙**

**분류 :** 고객 / 본인 / 회사 / 사회에 대한 의무

### 03 금융투자업 직무윤리

4**가지 의무**

- 기본원칙 (금융투자회사 표준윤리원칙)
- 이해상충방지 의무 (관련 법률)
- 금융소비자 보호의무 (관련 법률)
- 본인, 회사, 사회에 대한 윤리 (금융투자회사 표준윤리원칙)

**A. 기본원칙**

고객우선원칙, 신의성실원칙

**B. 이해상충방지의무 (법률)**

개념 : 자본시장법 37조 2항 (투자자의 이익을 해하며 이익을 얻으면 안됨)

원인 : 사적업무에서 취한 정보를 공적업무에 사용 시 이해상충 발생

방지 : 이해상충관리의무, 공시 또는 회피, 정보교류차단, 조사분석자료 이해상충방지의무, 자기거래금지의무

❗자기거래금지 예외 : 장내거래, 다자간매매채결회사 통한 거래

**C. 금융소비자 보호의무 (법률)**

개념 : 정보의 비대칭으로 금융소비자가 불이익을 얻지 않게 하는 것

의무 : ‘그때마다의 구체적인 상황에서 전문가로서의 주의를 기울여야’ 함

기관 : 이사회, 대표이사, 금융소비자보호총괄책임자 (CCO) - 위험관리 업무는 담당이 아님

단계

상품개발단계 : 사전협의절차, 금융상품개발 관련점검, 외부의견 청취

상품판매이전 : 교육체계 마련, 판매자격 관리 (펀트, 증권, 파생)

상품판매단계 : KYC rule → 적합성 원칙 → 적정성 원칙→ 설명의무 (일반금융소비자만)

(손실보전 홍보 금지, 요청하지 않은 투자권유 금지 - 증권/장내파생은 허용)

상품판매이후 : 보고 및 기록의무, 누설 및 부당이용 금지, 모니터링제도

ex) 해피콜 서비스, 고객의 소리, 위법계약해지권, 미스터리 쇼핑

***** KYC rule (know your customer rule)

투자권유 여부 확인 → 일반/전문금융소비자 확인 → 고객 정보 파악 → 서명 → 투자 성향 파악 → 예금성 제외

**D. 본인, 회사, 사회에 대한 윤리 (금투사 표준윤리준칙)**

본인에 대한 윤리

- 법규준수, 자기혁신의무 (임직원은 최소한의 지식을 갖추고 준칙을 준수)
- 품위유지의무, 공정성 및 독립성유지의무, 사적이익추구금지의무, 수수료 및 성과보수 제한

회사에 대한 윤리

- 상호존중의무, 회사재산의 사적사용금지, 경영진의 책임, 비밀정보 관리
- 위반행위의 보고, 고용종료 후의 의무, 대외활동 준수사항 준수

사회에 대한 윤리

- 시장질서 존중 (교란행위 금지), 주주가치극대화, 사회적 책임

**직무윤리위반 시 제재**

내부통제 : 내부통제기준(업무의 분장과 조직구조 관련사항) 설립, 준법감시인 활용

내부통제위원회 설치

- 면제  - 자산총액 기준 : 상호저축은행 7천억, 금융투자회사 - 보험회사 - 여신전문 5조
- ❗단 금융투자회사가 총합 20조 이상인 경우는 제외

준법감시인

- 임명 : 이사회의결, 해임 : 이사회 2/3이상 의결
- 업무 : 이사회의 지휘를 받아 내부통제업무 수행, 재무경영과 무관한 보수 지급, 명확한 업무 위임 가능 (영업관리자), 임기 2년

외부통제 : 자율규제, 행정제제, 민형사책임, 시장통제

1억 이하 과태로 : 내부통제기준 x, 준법감시인x, 이사회 없이 준법감시인

3천 이하 과태료 : 준법감시인 보수지급 기준 x

2천 이하 과태료 : 준법감시인 임면 미보고

## 2. 자본시장과 금융투자업에 관한 법률 (7)

패러다임 : 포괄주의 규제, 기능별 규제, 업무범위 확대, 투자자보호의 선진화

### 01 정의

**감독기관**

- 공적규제 : 금융위원회, 증권선물위원회, 금융감독원
- 자율규제 : 한국거래소, 한국금융투자협회
- 기타 : 예탁결제원, 증권금융, 금융투자상품 거래청산회사

**금융투자상품**

1. 투자성이 있는 것으로 증권 vs 파생상품으로 구분
2. 명시적 포함

    <증권>

    - 채무증권(채권) : 지급청구권이 표시된 것
    - 지분증권(주식) : 주권, 신주인수권 포함
    - 수익증권(펀드) : 수익권이 표시 - 집투기구는 투자신탁만 발행
    - 투자계약증권 : 공동사업에 대한 손익
    - 파생결합증권 : 기초자산 혹은 자산의 지수 변동과 연계된 증권
    - 증권예탁증권 : 위 5개 증권을 예탁받아 발행국 외에서 발행

    <파생상품>

    - 선도 : 기초자산(의 가격)을 특정시점에 인도할 것을 계약
    - 옵션 : 기초자산의 거래를 성립할 수 있는 권리를 계약
    - 스왑 : 일정기간동안 정한 가격으로 교환할 것을 계약
3. 명시적 배제 : CD, 관리신탁 수익권, 주식매수선택권(스톡옵션) 제외

**금융투자업 :** 투자매매업, 투자중개업, 집합투자업, 투자자문업, 투자일임업, 신탁업

**투자자**

- 전문투자자 : 위험감수능력이 있는 자
    - 절대적 : 국가, 금융기관, 관련기관
    - 상대적 : 원하면 일반투자자 대우 가능
    - 자발적 : 100억 이상 잔고 보유 법인, 특정 조건 개인 (5천 이상 + 소득1억/자산5억/종사이력)
- 일반투자자 : 전문투자자가 아닌 자

**인가/등록**

- 인가 : 투자매매업, 투자중개업, 집합투자업, 신탁업
- 등록 : 투자자문업, 투자일임업 + 온라인소액중개, 일반사모집합투자
- 금융위원회 접수 후 3개월 이내 결정됨

**건전성 규제**

자산건전성 5단계 : 정상 - 요주의 - 고정 - 회수의문 - 추정손실 (0.5 / 2 / 20 / 75 / 100)

순자본비율 : 금융투자회사의 파산을 사전에 예방, 100% 이상 유지

(영업용순자본 - 총위험액) / 필요유지자기자본

- 경영개선 권고 ( < 100%, 2년적자 + 900%, 1100%)
    - 점포관리 효율화
    - 자산 처분
    - 신규업무진출 제한
- 경영개선 요구 ( < 50%, 2년적자 + 1100%,  1300%, 4등급)
    - 점포 폐쇄 및 통합, 신설제한
    - 축소
    - 영업의 일부정지
- 경영개선 명령  ( < 0%)
    - 임원진 직무집행 정지
    - 6개월 이내의 영업 전부정지

- 금융투자회사는 위험관리체제 구축 필요, 외환건전성 관리

**업무보고서 :** 분기는 45일, 매월은 익일말일 내 제출

**경영공시 :** 부실채권 10%, 금융사고 2% 소송패소 1%, 적기시정조치, 회계기간 변경

(단, 10억원 이하는 제외)

### 02 금융투자업 종류별 영업행위 규칙

**1,2. 투자매매업, 투자중개업**

- 매매형태 명시의무, 최선집행
- 금지 : 자기계약, 불건전 영업행위, 선행매매, 조사분석자료 공표 후 매매
- 신용공여 : 자기자본 한도, 담보비율 140%까지 융자/대주하여 공여 가능, 위반 시 행정조치

* 종합금융투자사업자 : 3조 이상 자기자본을 가진 주식회사

**3. 집합투자업**

금지

- 집투 투자 시 동일기구 총액 20% 이상,  동일업자기구 총액 50% 이상
- 전체/각 집합투자기구에서 한 법인의 총주식 20%/10% 이상 투자
- 적격요건 없는 장외파생상품 매매, 위험평가액이 순자산을 넘은 파생상품 투자
- 국내 부동산 취득 후 1년, 국외 부독산 취득 후 정해진 기간 내 처분
- 집합투자기구의 재산으로 금전차입
    - 불가피한 경우 대량환매청구, 대량매수청구에서 10% 차입가능
    - 부동산펀드의 경우 순자산액 200%까지 차입, 100%까지 대여 가능
    - 부동산 70%이상 보유펀드는 70%까지 차입

의무

- 소유한 주식의 의결권을 충실하게 행사해야 함 (관련된 경우는 중립투표)
- 자산운용보고서보고 per 3개월, 필요 시 수시공시 (인력변경, 환매변경, 부실자산 발생, 총회내용)
- 집합투자재산의 산정된 기준가액이 적정한지 신탁업자가 확인
- 이익금 분배 : 증권으로, 유보(MMF는 불가), 초과분배

특징

- 투자신탁은 수익증권만 발생 가능 (무액면, 기명식, 균등분할)

**4,5. 투자자문업, 투자일임엄**

금지 : 보관, 예탁, 대여, 대여 중개, 수수료 이외 받음, 선행매매

(일임업 추가 - 본인이 인수한 증권을 일임재산으로 매수, 3개월마다 보고, 증권의결권 불가)

예외 - 인수 3개월 후 매수, 인수주권 증권시장에서 매수

예외 - 국채, 지방채, 통안채, 특수채, 사채권 (주식관련, 상각형 제외)

* 유사투자자문업 : 불특정다수인을 대상으로 하는 것, 대가를 받는 경우 해당

**6. 신탁업**

종류 : 종합신탁, 금전신탁, 재산시탁, 부동산신탁

금지 : 불특정금전신탁 운용제한, 불건전영업행위 금지대상

투자신탁

- 신탁계약서를 통해 집합투자업자가 신탁업자와 계약을 체결
- 무액면 기명식의 수익증권 발행, 수익자총회 의결정족수는 1/4
- 임의해지 : 수익자 전원 동의, 1년 뒤 원본액 50억 미만
- 법정해지 : 기간종료, 해지결의, 피흡수, 신탁등록취소, 수익자 1인

### 03 공시제도

**발행시장 - 증권신고서**

모집 : 50인 이상, 신주 청약 권유

매출 : 50인 이상, 기존주 청약 권유

간주모집 : 50인 미만이지만 사실상 50인 이상이면 증권신고서 제출

소액공모 : 10억 미만은 증권신고서 제출의무 없음, 재무관련은 공시

* 특수

일괄신고서 : 동일 증권을 지속발행하는경우 면제

정정신고서 : 정정하고자 하거나 요구받은경우 제출

**유통시장**

**정기공시 - 사업보고서, 반기 및 분기보고서** (각 90일, 45일 내)

**수시공시 - 주요사항보고서**

대상 : 부도 및 거래정지, 영업정지, 회생절차개시, 해산사유 발생, 중요한 영업, 자기주식취득, 경영 주요사항

**공개매수제도** : 5%이상 보유하게 되면 공개매수(증권시장 외, 불특정 다수 대상)해야 하며, 5일 내 보고해야 함

**시세조종행위 금지** : 위장거래(통정매매, 가장매매), 가격조작행위 (안정조작, 시장조성은 제외)

규정

- 금융투자업규정

- 자본시장조사 업무규정

### **01 금융투자업규정**

**금융투자업의 인가**

인가 : 투자매매업, 투자중개업, 집합투자업, 신탁업

(3개월 소요, 요건 - 사업계획, 대주주 요건, 이해상충방지체계 구비)

등록 : 그 외

**자산건전성**

- 정상 - 요주의 - 고정 - 회수의문 - 추정손실
- 분류에 따라 대손충당급 적립 필요

**순자본규제** (중복)

용도

- 영업용순자본을 총위험액 이상으로 유지해야 함
- 장부가액 기준 산정, 부외자산과 부외부채도 위험액 평가
- 1개월마다 보고, 순자본비율 100% 붕괴 시 즉시 보고

계산

- 영업용순자본 = 자산 - 부채 - 차감항목 + 가산항목
- 총위험액 = 시장위험액 + 신용위험액 + 운영위험액
- 순자본비율 = (영업용순자본 - 총위험액) / 필요유지자기자본

평가

- 경영개선권고 : 순자본비율 100% 미만 → 인력 및 운용개선, 관리효율화
- 경영개선요구 : 순자본비율 50% 미만 → 고위험자산처분, 점포제한
- 경영개선명령 : 순자본비율 0% 미만 → 주식소각, 양도, 합병

**대주주와의 거래 제한**

- 대주주가 발행한 증권 소유 금지, 신용공여 금지
- 계열회사 증권 - min(자기자본 0.1% or 10억)보다 작으면 전원찬성 시

### **02 증권의 발행 및 공시 등에 관한 규정**

**증권신고서 효력 발생**

- 정정신고서 수리 - 당일
- 일괄신고서의 정정신고서 - 수리 3일 후
- 당초 신고한 총액의 +-20% 변동 - 영향 없음

**증권분석기관**

업무 : 증권의 모집가액, 매출가액 적정성 평가

대상 : 투자매매업자, 신용평가업자, 회계법인, 채권평가회사

제한 : 3% 이상 지분을 가짐, 동시에 5% 지분을 가진 자 있음

**주요사항보고대상 제외**

- 일상적인 영업활동으로 인한 자산 양수도
- 교체주기 1년 내의 설비관련 자산취득 및 처분
- 자기주식 취득 및 처분, 자산유동화, 공개매수

### 03 대외 발표 관련

**조사분석 자료 작성 및 공표**

의의 : 금융투자회사의 명의로 제 3자에게 제공되는 특정 상품의 가치에 대한 평가

원칙 : 선관주의의무 준수, 금융투자분석사의 확인

공표 : IPO 담당 시 1년 2회 이상 무료 공개

심의 : 구체적 내용 (2년간 투자등급, 변동추이), 인용자료 출처명기

교육 : 금융투자분석사는 연 2시간 이상 윤리교육

매매제한 : 공표 후 24시간, 분석사는 본인 담당 업종 불가 미담당 24시간, 공표 시 재산적 이해관계 고지해야 함

의무표시사항, 표지금지사항 준수, 준법감시인의 사전 승인 후 심사청구 필요
