# 제1과목 자산관리 기본지식 (40)

## 1. 재무설계의 의의 및 재무설계 프로세스 (10)

### 01 개인재무설계

**개념**

① 목표달성을 위해 재무적/비재무적 자원을 관리

② 재무상담은 문제평가, 재무설계는 목표에서 시작

③ 단기적 뿐아니라 중장기적 목표달성(평생동안 지속)

**배경**

① 사회경제적 : 자산/부채의 증가, 금융시장 개방 및 국제화, 금융상품 다양화 및 금융 관련 법규 강화

② 인구통계적 : 1인가구증가, 저출산/고령화, 노동환경변화

③ 소비자의식 변화 : 개인주의적 사고방식과 개별성 추구, 비재무적요구 증가, 재무설계의 중요성 인식

- 노년부양비 = 노년인구 ÷ 생산가능인구
- 노령화지수 = 노년인구 ÷ 유년인구

**자산관리사**

종류 : FP / AFPK, CFP / 투자상담사

### 02 재무설계 절차

- 관계정립 → 정보수집 → 재무상태 분석/평가 → 재무설계 제안 → 실행 → 사후관리

**1단계 관계정립**

**유망고객**

① 지인, 기 계약자, 소개

② 재무목표가 있는 사람, 경제적 능력이 있는 사람, 만남이 가능한 사람, 실행력이 있는 사람

**고객면담 채널**

① DM (Direct mail) : 심리적 부담 적고 동시접촉 가능, 맞춤형으로 제작할 것

- 고객과 친밀한 관계 형성 가능, 면담을 매끄럽게 진행 가능

② TA (Telephone approach) : 심리적 부담 크고 상품판매 목적으로 활용금지, 약속잡는 목적으로

- 유망고객과 쉽게 만날 수 있음, 효율적 면담 가능

③ SMS : 심리적 부담 적고 비용 저렴, 발송시간 주의

**효과적인 질문 :** 고객 스스로 말할 수 있도록 하는 질문, 완고한 고객일수록 적극적 경청과 공감 필요

- 상황파악 질문 : ~는 어떤가?
- 문제인식 질문 : ~은 문제없는가?
- 시사 질문 : ~면 어떻게 할 것인가 (결과에 대한 심각성을 인지하게 함)
- 해결 질문 : ~가 있다면 살 것인가

**2단계 고객 정보수집 및 재무목표 설정**

**재무목표**

① 단기(1년 이내), 중기(1~10년), 장기(10년 이상)

② 생애주기별 재무목표

- 청년기 : 첫 직장잡기, 결혼자금
- 가족형성기 : 자녀출생자금, 자동차구매자금
- 자녀양육기 : 자녀교육자금, 주택구입자금
- 자녀성장기 : 자녀 교육/결혼 자금, 주택확장자금
- 가족축소기 : 노후자금, 기타 목적자금
- 은퇴/노후생활기 : 노후생활자금, 상속/증여 계획

**정보수집**

① 방법

- 직접면담(비재무적 정보파악 용이)
- 설문서(시간 절약, 생각 잘 반영) → 상담 전과 상담 중
- 인터넷(시간/비용 절약, 쌍방향 의사소통)
- 전화(질문 및 확인시 유용)

② 재무적정보 : 종업원복지, 개인사업 관련 자료

③ 비재무적정보 : 예상수명, 재무설계 경험, 위험수용성향

**3단계 재무상태 분석**

**자산부채상태표** : 특정시점, 재무상태 변동의 결과 / 자산 구성, 부채 규모, 유동성 파악

자산 : 현금성자산, 금융투자자산, 부동산자산, 개인사용자산

- 현금성 : 예금, CMA, MMF, MMDA, 6개월 미만 만기 RP
- 금융투자 : 6개월 이상 주식, 채권, 뮤추얼펀드, ELS, ELD, ELF

**현금흐름표** : 일정기간, 재무상태 변동의 원인 / 소득, 생활수준, 저축 및 투자능력 파악

현금유입 : 모든 소득과 모든 현금유입

현금유출 : 고정지출, 변동지출, 저축 및 투자

- 고정 : 대출상환금, 세금, 주택관리비, 공교육비
- 변동 : 외식비, 사교육비, 교양오락비, 교통통신비, 건강의료비, 피복비 등

**4단계 설계제안**

- 훌륭한 제안은 고객이 자산관리사의 도움을 필요하게 만드는 것

**제안서 구성**

- 표지 및 서문
- 목차 (고객 현황 요약 - 제안 목적 - 고객 재무상황 분석 - 대안 제시 및 제언)

**제안서 작성시 유의사항** : 표준화된 양식 없음

① 너무 많은 대안을 나열하지 말 것

② 고객 상황변화를 반영한 유연한 대안을 제시

③ 주관적 의견이며, 자산관리사에 따라 다름을 설명

④ 고객의 가치관이나 생활방식을 고려한 대안 제시

⑤ 고객의 재무목표에 적당한 최선의 대안부터 제시

- 목차 : 고객 현황 요약, 제안 목적, 고객 재무상황 분석, 대안 제시 및 제언

**연령별 고려사항**

- 20대 : 소비에 관심많고, 정보나 관심도 많음, 자기만의 라이프스타일 고수
- 30대 : 자녀에 대한 투자에 관심 많음, 부동산 투자는 적절히
- 40대 : 샌드위치세대, 투자여력없고 교육비지출 많음
- 50대 : 자녀결혼 등 거액의 지출 집중, 안정적 성향, 연금 납입기간은 짧게, 납입금액은 크게

**직업별 고려사항**

- 개인사업자 : 종업원 4대보험, 노란우산공제, 절세, 금융상품으로 효과적으로 자산이전
- 근로소득자 : 생애주기별 목표달성, 예산수립, 소득/지출의 통제, 소득공제(절세전략), 재취업·창업 방안
- 전문직 : 주소득원의 보장자산, 투자방안, 세무조사
- 임대사업자 : 실물 - 금융자산비율, 세금, 자산이전, 세무조사

- 노란우산공제 : 개인사업자를 위한 제도
- 3층 보장제도 : 은퇴 후 필요소득 보장을 위한 제도, 국민연금-퇴직연금-개인연금

**5단계 실행**

**효과적인 가입제안**

① 논리적으로 설명하되 감성적 스토리텔링 제공

② 전문가라는 신뢰 부여, 고객이익에 반하는 결정 금지

③ 미루거나 거절하는 고객에 대한 거절처리 숙지

④ 재무목표 달성시 구체적 이익과 금액을 제시

**계약체결 기법**

- 묵시적동의법 : 동의를 전제로 진행
- 양자택일법 : 두가지 중 하나 선택하게 함
- 예화법 : 타인의 계약체결 사례 제시
- 손해암시법 : 미가입시 손해일 것이라 설명

**6단계 사후관리**

정기점검

- 고객에 관한 사항 : 신상변화, 수입변화, 건강변화, 재무목표 변화
- 투자에 관한 사항 : 수익률, 세금, 수수료 문제, 경제상황 점검

## 2. 경제동향 분석 및 예측 (15)

### 01 거시경제

**거시경제의 시계**

① 단기 : 가격/임금 경직적, 불완전고용, 총수요 분석

② 장기 : 가격/임금 신축적, 완전고용, 기술변화 없고 생산요소의 총량 고정, 총공급 분석

③ 최장기 : 기술발전 가능, 생산요소의 총량 가변적, 경제 성장 분석

**거시경제의 4대시장과 5대 경제주체**

시장 : 생산물시장, 요소시장, 대부자금시장, 외환시장

| 생산물시장 | 균형 실질 GDP - 물가 |
| --- | --- |
| 요소시장가 | 균형 실질 임금 - 고용량 |
| 대부자금시장 | 균형 이자율 - 대부자금 거래량 |
| 외환시장 | 균형 환율 - 외환거래량 |

주체 : 가계, 기업, 정부, 해외, 중앙은행

**국민소득 항등식**

수출 - 수입 = [(조세 - 재정지출) + 저축] - 국내투자

→ 상품/서비스수지 = 국내총저축 - 국내투자 = 민간 순저축 + 재정수지

### 02 생산물 시장

- 물가 - 실질국민소득  평면

**AD 곡선**

- 물가와 실질소득 평면에서 우하향의 기울기
    - 우하향 이유 : 구매력 효과, 실질통화공급 효과, 부의 효과, 순수출 효과 **(구실부순)**
- 총수요 증가 : 가계 부 증가, 물가상승/소득증가/이자율상승 기대, 기술발전, 환율상승, 무역장벽 강화
- 총수요 감소 : 가계 부채 증가, 이자율상승/조세부담 증가 기대, 상대물가 상승

**AS 곡선**

- 물가와 실질소득 평면에서 단기 우상향 (아래로 볼록), 장기 수직
- 단기총공급 증가 : 인구/자본량 증가, 기술/생산성 향상, 요소가격 하락, 총수요 증가 예상, 긍정적 충격
- 단기총공급 감소 : 임금상승, 생산요소 가격 상승, 부정적 공급충격
- 장기총공급 증가 : 총노동시간 증가 (인구, 경제활동율 증가), 노동생산성 증가 (물적 인적 자본증가, 기술발전)

**AD-AS**

- 실제GDP > 잠재GDP : 경기과열, 인플레이션 가속화

### 03 노동시장

- 고용량 - 실질임금 평면

- 공급곡선은 우상향, 수요곡선은 우하향
- 노동고용량 증가시 총생산량은 체감적으로 증가
- 임금 변동시 수요/공급량, 이외 요인 변동시 곡선 이동
- 자연실업율 : 완전고용상태의 실업률

### 04 인플레이션/실업

**인플레이션**

- 수요견인 : 통화공급 증가, 조세부담 완화, 긍정적 미래 기대, 환율 상승, 실질이자율 하락
- 비용인상 : 생산성 하락, 생산요소 가격 상승, 자연재해에 따른 생산설비 망실(스태그플레이션)

GDP 성장속도 < 통화량 증가속도, 부의 분배

**실업**

- 참가율 = 활동인구 / 노동가능인구
- 고용률 = (취업자 ÷ 노동가능인구)×100
- 실업률 = (실업자 ÷ 경제활동인구)×100

종류 : 마찰적실업, 구조적실업, 계절적실업, 경기적실업

- GDP갭 > 0 이면 확장국면, < 0이면 수축국면
- 단기필립스곡선 : 인플레이션과 실업률, 우하향곡선 / 장기필립스곡선 : 자연실업율 수준에서 수직의 형태

### 05 재정/통화정책

**재정정책**

- 재원조달 : 조세징수(불완전 구축효과), 공개시장매각(구축효과), 중앙은행인수(구축효과없이 통화량 증가)
- 내부시차 길고 외부시차 짧음, 재량적정책 파급효과 큼
    - 재량적 재정정책 : 정부가 상황을 보고 조세 등을 조절 (수동)
    - 비재량적 재정정책 : 정부가 자동안정화장치를 통해 조절 (자동)

**통화정책**

- M1 (협의통화): 현금, 요구불예금, 수시입출금식 예금
- M2 (광의통화): M1, 만기 2년 미만 정기 예·적금, 만기 2년 미만 금융채, MMF, RP
- Lf (금융기관 유동성): M2, 만기 2년 이상 예·적금, 만기 2년 이상 금융채, 보험상품
- L (광의 유동성): Lf, 국채, 지방채, 회사채, 기업어음

- 화폐 지표
    - 본원통화 = 화폐 발행액 + 은행시재금 + 중앙은행지준예치금
    - 통화승수 = 통화량/본원통화
- 역(-) : 지급준비율, 현금보유비율, 초과지급준비율
- 정(+): 준통화비율
- 정책수단 : 지급준비제도, 공개시장조작, 중앙은행 대출, 외환시장개입
- 내부시차 짧고 외부시차 길고 변화도 큼 /  과열된 경기 진정시 유용, 경기부양효과는 미비
- 물가안정목표제 : 3년 중기목표 설정, 통화량 등 중간목표 없음, 운용목표 = RP금리

### 06. 이자율 (대부자금시장)

- 실질이자율 - 대부자금거래량

이자율변동 : 무위험이자율, 위험, 조세/정부보조, 만기

- 기간구조이론 : 기대이론, 시장분할이론, 유동성프리미엄 이론(우상향의 수익률곡선 설명)
- 대부자금  공급곡선 우상향, 수요곡선 우하향 (실질이자율 - 대부자금 거래량 평면)
- 이자율 변동시 수요·공급량, 기타 변동시 수요·공급 변동
- 외부충격
    - 재정흑자 : 대부자금 공급의 증가로 이자율 하락

### 09 경기변동과 경기예측

**경기변동**

- 회복 → 정점 → 수축 → 저점
- 장기추세선 기준 확장국면/수축국면 구분 (실질 GDP 사용)

특징 : 반복성, 불규칙성, 예측불가능, 비대칭성, 변동의 크기 다르고 시차를 두고 변동

GDP 대비 변동성 비교

- GDP < : 내구제 소비, 주거용 건설투자, 기업투자, 국제교역
- GDP = : 생산성
- GDP > : 실업률

**경기예측**

개별경제지표 : 수출(중립) 등

종합경기지표 : 발표빈도, 수정/변경, 일관성, 리드타임 고려

- 선행 : 재고순환지표, 경제심리지수, 기계류내수출하지수, 건설수주액, KOSPI, 장단기금리차, 수출입물가
- 동행 : 비농림어업취업자수, 광공업생산지수, 건설기성액, 서비스업생산지수, 소매판매액지수, 내수출하지수, 수입액
- 후행 : 취업자수, 생산자제품재고지수, 소비자물가지수변화율, 소비재수입액, CP 유통수익률

경기전환점 : 현재와 반대방향으로 3개월 연속 변동

문제점 : 거짓신호, 리드타임, 월간변동 분석의 한계, 상반된 신호, 빠른 경제구조 변동시 대응성 저하

**설문조사를 이용한 경기예측 :** BSI, CSI

**계량모형을 이용한 경기예측**

- 시계열모형 : 간단하고 시간/비용 절약, 경제환경 영향 설명 곤란, 이론적 근거 취약, 정교한 분석곤란, 단기
- 거시경제모형 : 일관성/동시성 유지, 오차관리 용이, 파급효과 구체적 측정가능, 변수간 관계의 안정성 약화시 예측력 저하, 오차발생 필연적, 시간/노력 과다

**물권**

![image.png](1%EB%B6%80/image.png)

의미 : 특정의 독립된 물건 자체를 객체로 하여 권리를 실현시키는 재산권

종류 : 기본물권 (점유권, 소유권), 제한물권 (용익물권, 담보물권)

- 용익물권 : 지상권, 지역권, 전세권 → 소멸시효 대상
- 담보물권 : 유치권, 질권, 저당권, 근저당권

효력 : 제한물권 > 기본물권 > 채권

공시 : 등기(부동산), 점유(동산), 명인(수목) / 부동산 등기는 추정력만 인정

변동 : 법률행위는 등기해야 발생, 법률규정은 등기하지 않아도 효력발생

소멸 : 목적물 멸실, 20년 (용익물권), 포기, 혼동

**채권**

의미 : 채권자가 채무자에 대해 일정한 행위를 요구할 수 있는 권리

양도 : 지명채권(양도인 양수인 합의), 지시채권(증권에 배서), 무기명채권(증서 교부)

- 중첩적 채권인수 : 중복하여 채무자가 되는 것

소멸 : 변제, 대물변제, 공탁, 상계, 경개, 면제, 혼동

- 변제, 대물변제 : 갚음
- 공탁 : 공탁소에 갚음
- 상계 : 상쇄시킴 (단독행위)
- 경개 : 채무를 소멸시키고 새로 만듬
- 면제 : 면제
- 혼동 : 동일인이 채권자이며 채무자인경우

**계약**

의미 : 둘 이상의 당사자가 청약과 승낙을 통해 법률적 효과를 발생시키는 행위

종류

- 낙성계약 : 합의만으로 성립
- 요물계약 : 합의 외에 특정 행위가 완료되어야 성립

- 소비대차 : 소유권 이전과 반환을 약정 (낙성계약)
- 소비임치 : 보관을 위탁하고 상대가 승낙

### **03. 은행거래 기본법률**

- 유가증권 : 은행이 증권을 교환에 돌려 부도반환시한이 지나고 결제를 확인한 때에 예금계약이 성립
- 전자자금이체 : 금액의 정보에 대하여 수취인의 계좌가 개설된 금융회사의 계좌원장에 입금기록이 끝난 때

대출계약 : 금전소비대차

대출 : 유상·낙성·쌍무계약

종류

- 증서대출 : 대출약정서 작성하고 대출
- 어음대출 : 은행에 약속어음을 발행해줌
- 어음할인 : 기업이 어음을 만기일 전에 은행에 판매 (유동화)
- 당좌대출 : 당좌예금 잔고가 마이너스여도 가능하게 함 (마통)

회수 : 변제, 상계

기한이익상실사유 : 만기 전에 갚게 하는 것

상계의 요건(상계적상)

- 채권이 서로 대립하고 있을 것
- 상계를 하는 자의 채권인 자동채권과 상계를 당하는 자의 채권인 수동채권 모두 변제기에 있을 것
- 채권의 성질상 상계가 허용될 것
- 서면에 의한 상계 통지를 할 것

행사 : 의사표시를 해야 상계의 효과 발생, 서면에 의한 상계의 의사표시 요함

효과 : 유효한 상계가 있으면 자동채권과 수동채권은 그 상계적상 시로 소급하여 대등액에 관하여 소멸

청약철회권

### **08. 가족관계 관련 법규**

**혼인**

- 민법상 계약이며 당사자 사이의 합의와 신고로 성립
- 부부재산의 귀속에 관하여 별산제를 취함
- 부부 중 누구에게 속하는 것인지 분명하지 않은 재산은 부부의 공유로 추정
- 일상 가사에 대하여는 부부는 서로 대리권이 있음

**이혼**

- 협의이혼 : 민법상 계약, 요식행위 (가정법원의 확인 필요)
- 재판상 이혼 - 당사자 일방의 청구로 가정법원의 판결로 혼인 해소
- 재산분할청구권 - 이혼한 날부터 2년, 유책배우자도 청구 가능
- 이혼 시 혼인으로 생긴 인척관계는 모두 소멸

**친권**

- 부모가 미성년의 자녀를 보호·교양하는 권리임과 동시에 의무
- 친권자는 자녀의 재산에 관한 법률행위에 대하여 그 자녀를 대리, 이해상반행위에는 특별대리인 선임해야함 (미선임은 무효)
- 친권의 행사는 부부공동으로 하고 일방이 행사하지 못할 경우 다른 일방이 행사

**후견**

- 제한능력자나 그 밖에 보호가 필요한 사람을 보호하는 것
- 법정후견 : 미성년후견, 성년후견, 한정후견 (부분), 특정후견 (특정사무)
    - 피한정후견인은 혼자 법률행위 가능
- 임의후견(후견계약) : 질병·장애·노령·그 밖의 사유로 인한 정신적 제약으로 사무를 처리할 능력이 부족한 상황에 있거나 부족하게 될 상황에 대비하여 자신의 재산관리 및 신상보호에 관한 사무의 전부 또는 일부를 다른 자에게 위탁하고, 그 위탁사무에 관하여 대리권을 수여하는 계약

**상속**

- 순위 : 직계비속과 배우자 → 직계존속과 배우자 → 피상속인의 형제자매 → 4촌 이내의 방계혈족
    - 사실혼의 배우자는 포함되지 않음
    - 자녀가 부모보다 먼저 사망 시 자녀의 배우자와 자녀가 상속 (대습상속)
    - 앞 순서가 없을때만 넘어감
- 효력 : 상속인은 상속이 개시된 때에 피상속인의 재산에 관한 모든 권리·의무를 포괄적으로 승계
- 비율 : 상속분은 균분, 배우자의 상속분은 5할 가산 / 유언분할 → 협의분할 → 심판분할
- 포기
    - 3개월 내 승인·포기 가능, 승인·포기 없으면 단순승인으로 간주
    - 채무초과 사실을 중대한 과실 없이 모른 경우 안 날로부터 3개월 내에 한정승인을 할 수 있음
    - 상속을 포기한 경우 상속분은 다른 상속인의 상속분의 비율로 귀속

**유언**

- 상대방 없는 단독행위

형식

- 자필증서 : 전문, 날짜, 주소, 이름 수기작성, 도장날인 (사인불가)
- 녹음 : 내용, 이름, 날짜 구술, 증인참여 (확인 녹음)
- 공정증서 : 공증인 앞에서 구술, 증인 2인 참여, 적고 낭독확인
- 비밀증서 : 유언장 작성 후 봉투 봉인 날인, 증인 2인과 함께 공증인 제출
- 구수증서 : 급박한 사유시 증인 2인, 유언자 필기, 사유 종료부터 7일 이내 법원 검인
    - 일정한 방식을 따르지 않으면 무효
- 유증 : 유언에 의하여 재산상의 이익을 타인에게 무상으로 주는 단독행위
- 유류분: 일정범위의 상속인에게는 최소한의 생활보장 내지 부양을 위해 인정하는 제도
    - 직계비속과 배우자는 1/2, 직계존속은 1/3
    - 반환청구권 : 유류분권자가 피상속인의 증여 또는 유증으로 인하여 그의 유류분에 부족이 생긴 때에는 부족한 한도에서 증여 또는 유증된 재산의 반환을 청구할 수 있음 (인지부터 1년, 상속개시부터 10년 이내)

### 02 종합소득 산출

종합소득금액 = 2천만원 초과 이자/배당소득 + 사업소득 + 근로소득 + 연금소득 + 기타소득

    - 열거된것만 기타소득으로 간주

종합소득금액 → (세액공제) → 결정세액 → (기납부세액) → 납부할 세액

**종합소득공제**

인적공제 : 본인, 배우자, 부양가족에 대해 1인당 연 150만원 공제

    - 장인 장모 가능, 며느리 사위 형수 제수 불가능
    - 연간소득금액에는 퇴직/양도소득도 포함
- 추가공제 : 70세 이상 경로우대 (100), 장애인 (200), 부녀자 (50), 한부모 (100)
    - 부녀자공제 한부모공제는 중복 불가능

연금보험료공제 : 납입보험료만큼 공제

    - 이중공제 방지 (보험료, 교육비, 기부금, 월세)

③ 예금 등 금융자산을 분산 증여

④ 연간 금융소득 평준화

⑤ 중도해지 이자소득감액분에 대한 처리는 세부담이 적은 방법 선택

⑥ 타익신탁을 활용

⑦ 주식형 펀드상품을 활용

⑧ 장기저축성보험을 활용

⑨ 소득이 적은 개인에게 채권을 양도

### 04 연금소득 퇴직소득

**연금소득**

분리과세 : 사적연금소득 중 (원천징수되지 않은 퇴직소득을 연금수령, 인출하는 연금소득)

소득금액 :  총연금액 - 연금소득공제액 (350 이하면 전액 소득공제)

세율

- 공적연금 : 종소세 기본세율
- 사적연금 : 나이별 (70,80) 5%/4%/3%, 종신연금 4%

산재보상연금은 비과세

**퇴직소득**

퇴직 : 퇴직금 중간정산, 퇴직연금제도 폐지로 퇴직급여를 미리 지급받는 경우 포함

소득금액 = 퇴직급여 합계액 - 퇴직소득공제 (근속년수 & 환산급여 고려)

## 2. 금융소득종합과세 (8)

### 01 금융소득종합과세 개요

**특징**

- 금융소득 = 이자소득 + 배당소득
- 2천만원 초과부터 종합과세
- 채권은 보유기간별로 이자소득을 계산
- 종합과세대상이 되면 원천징수 세액은 공제
- 부부는 별도과세

### 02 이자소득, 배당소득

**이자소득(유형별포괄주의, 필요경비 불인정)**

- 채권/증권/예금 이자
- 채권/증권 환매조건부 매매차익
- 저축성보험 보험차익
- 직장공제회 초과반환금
- 비영업대금의 이익
- 신종금융상품 이자

**배당소득(유형별포괄주의, 필요경비 불인정)**

- 이익/건설이자배당
- 법인/집합투자기구로부터의 이익
- 출자공동사업자에 대한 손익분배비율 금액
- 배당/의제배당/신종금융상품 배당

Gross up : (배당소득+배당소득 법인세)로 계산하고 배당소득 법인세를 추후 공제

조건 : 종합과세 대상 & 법인세 과세대상 & 내국법인

- 배당소득금액 = 배당소득 x 11%
- 배당세액공제액 = 배당소득 x 10%

### 03 비과세/분리과세 저축

**ISA**

- 대상 : 근로자, 사업자, 농어민
- 특례 : 400/200만원 비과세, 초과분 9% 분리과세
- 기간 : 의무 3년
- 납입 : 총 1억, 연간 2천만원

**분리과세저축**

- 임의단체의 금융소득 : 원천징수
- 비실명금융자산 금융소득
- 직장공제회 초과만환금
- 법원보증금 이자소득

### **04 원천징수**

**원천징수세율**

기본세율 : 직장공제회 초과반환금

14% : 일반 금융소득

25% : 비영업대금 이익, 출자공동사업자 배당소득

30% : 분리과세 신청한 3년 이상 장기채 이자 (2018 이전)

45% : 비실명 금융자산에 대한 소득으로 지급자가 비금융기관인 경우

90% : 비실명 금융자산에 대한 소득으로 지급자가 금융기관인 경우

**이자소득의 원천징수시기**

- 약정에 따른 이자지급일 (지급개시일이 아님)
- 상속개시일, 증여일

**배당소득의 원천징수시기**

- 잉여금처분 결의일
- 잠자결의일, 퇴사일, 탈퇴일, 잔여재산가액 확정일, 합병등기일, 분할등기일, 결산확정일

### 05 세액

종합과세와 분리과세 중 큰 금액

## 3. 양도소득세 (8)

**과세대상**

- 토지, 건물, 부동산 이용권(지상권, 전세권, 등기된 부동산 임차권), 부동산을 취득할 수 있는 권리
- 주식 (1%/2%/4% 이상 지분 or 50억 이상의 대주주 혹은 장외양도, 비상장법인 주식)
    - 소액주주 K-OTC 양도소득은 비과세
- 기타자산 (사업용고정자산과 함께 양도하는 영업권, 비상장주식, 부동산 과다보유법인 주식)
- 파생상품 거래이익, 신탁수익권 양도

**시기** : 양도대금 청산일

- 대금청산일 불분명 → 등록일
- 대금청산 전 등기 → 등기 접수일
- 장기할부조건 양도 → 접수일/인도일/사용수익일 중 빠른 날
- 본인이 건설할 건축물 → 사용승인서 교부일
- 상속 : 상속개시일
- 증여 : 증여받은 날 (부동산은 접수일)

**산출**

- 양도가액 - 취득가액 - 기타의 필요경비 = 양도차익
    - 양도가액 = 실제 거래액, 확인불가 시 매매사례가액 → 감정가액 → 환산취득가액 → 기준시가
        - 5년이 안 된 건물에 감정가액/환산취득가액 적용 시 5% 가산
    - 기타의 필요경비 = 취득 부대비용 (등록세, 수수료) + 자본적 지출액 + 양도비
        - 매매사례가액/감정가액/환산취득가액 적용 시 기준시가의 3%
    - 특례 : 배우자/직계존비속으로부터 증여받은지 10년 이내면 최초취득당시의 가액으로 계산
- 양도차익 - 장기보유특별공제 = 양도소득금액
    - 장기보유특별공제액 : 3년 이상 보유한 토지/건물/조합원 입주권 대상
    - 장기보유특별공제율
        - 1주택 2년이상 x 4% (보유/거주) (~80%)
        - 1주택 이외는 3년이상 x 2% (~30%)
        - 적용 배제 : 미등기자산, 비사업용 토지, 1세대 2주택 이상
- 양도소득금액 - 양도소득기본공제 = 양도소득과세표준
    - 양도소득 기본공제 = 부동산/주식/파생/신탁수익권 각각 연 250만원 공제
- 양도소득과세표준 x 세율 = 양도소득산출세액
    - 양도소득세율
        - 부동산 : 50%, 40%, 기본세율
        - 기타자산 : 기본세율 (비사업용 토지 50% 이상인 주식은 + 10%)
        - 신탁수익권 : 3억 이하 20%, 3억 초과 (6천만 + 3억초과금 x 25%)
        - 주식 : 소액주주 10%(20%), 대주주 20%(30%)
- 양도소득산출세액 - 감면세액 = 결정세액
    - 8년 이상 거주하는 자경농지는 5년간 2억원 감면, 근로소득 연 3700만 이상이면 제외
- 결정세액 + 가산세액 = 총 결정세액
- 총 결정세액 - 기납부세액 = 자진납부세액

**부당행위계산**

특수관계자와의 거래에서 시가와 차액이 3억 이상  or 5%이상

**1세대 1주택의 양도**

1세대 인정 : 배우자와 거주, 30세 이상, 배우자 사망 혹은 이혼, 독립된 생계 유지 가능자 ( > 중위소득 40%)

기간 제한없이 비과세

- 건설임대주택 5년 이상 거주 후 양도
- 해외 이주로 세대 전원 출국, 2년 이내 양도
- 1년 이상 국외거주, 2년 이내 양도
- 1년 이상 거주 후 부득이한 사유로 세대 전원이 다른 시군으로 주거이전

특례

- 일시적인 2주택 : 1년 이상 거주 후 2주택 취득, 취득 3년 내 종전주택 양도 시 비과세
- 상속으로 2주택 : 일반주택 양도 시 비과세
- 세대합가 2주택 : 봉양을 위해 합가 시 10년 이내 먼저 양도하는 주택은 비과세
- 혼인으로 2주택 : 혼인으로 합가 시 5년 이내 먼저 양도하는 주택은 비과세
- 농어촌주택 + 일반주택 : 일반주택 양도 시 비과세
- 다가구주택 : 하나의 매매단위로 양도 시 단독주택으로 간주
- 겸용주택 : 주택이 어쩌고

- 비과세대상이라도 12억 이상이면 (1-12억/양도가액) x 양도차익 과세

**신고시기**

부동산 : 달의 말일부터 2월 이내

주식 : 반기의 말일부터 2월 이내

**절세방안**

① 양도부동산의 보유기간을 잘 따진 후 양도시기를 잡는다.

② 세법 개정안에 관심을 가져 유리한 해에 양도한다.

③ 한 해에 두 개 이상의 양도는 가급적 피한다. 다만, 양도차손이 발생하면 같은 해의 다른 양도차익과 통산이 가능하므로 이를 활용한다.

④ 주택 양도 전에 1세대 1주택자임을 꼼꼼히 확인한다.

⑤ 1세대 1주택 비과세제도를 적극 활용한다.(양도 순서 고려 및 비과세 특례 규정의 활용)

⑥ 부담부증여를 고려해 본다.

⑦ 배우자 이월 과세를 회피한 배우자 우회양도를 고려한다. (증여받은 후 10년 이후에 양도)

## 4. 상속세 / 증여세 (16)

### **01 상속세**

유언 방식 : 자필증서, 녹음, 공정증서, 비밀증서, 구수증서

유류분 : 법정상속분 중에서 보장되는 비율, 배우자/직계비속 1/2, 직계존속  1/3

**상속세 산출**

- 상속재산 + 간주상속재산 + 사전증여재산 + 상속추정재산 - 비과세재산 - 불산입재산 = 상속재산가액
    - 간주상속재산 = 보험금, 신탁재산, 퇴직금
    - 사전증여재산 = 상속인 10년 이내, 상속인 이외의 자에게 5년 이내 증여한 재산
    - 추정상속재산 = 1년 이내 2억 이상, 2년 이내 5억 이상 피상속인 재산에서 용도불분명한 인출/부담채무
        - 현금,예금,유가증권 / 부동산 및 관련권리 / 무체재산권으로 묶어서 계산
        - 미소명금액 중 min[미소명금액 20%, 2억] 공제
    - 비과세재산 = 상속세 신고기간 내 국가, 지자체에 기증한 재산 / 제사 주제 족보와 제구
    - 과세가액 불산입액 : 공익법인/신탁 출연재산
- 상속재산가액 - 과세가액공제액 = 상속세 과세가액
    - 과세가액공제액 = 공과금 + 장례비용(5백 ~ 1천, 봉안시설사용비용 : 5백) + 채무
- 상속세 과세가액 - 기초공제 - 기타인적/배우자/동거주택/금융재산 공제 - 재해손실공제 = 상속세 과세표준
    - 기초공제 = 2억원 + 가업상속공제 + 영농상속재산
        - 가업상속공제 = 10/20/30년 경영 시 300/400/600억
        - 영농상속공제 = 농지 30억
    - 기타인적공제 = 자녀 (5천), 노인 (5천), 장애인 (기대여명x1천), 미성년자 (19세까지 연당 1천)
        - (기초 + 기타인적공제) vs 5억 일괄공제 중 선택가능 (배우자 상속 제외)
    - 배우자공제 = min(배우자 실제 상속받은 재산, 상속재산가액x상속지분비율, 30억)
        - 5억 미만이면 5억 적용, 배우자상속재산 분할기간까지 분할하지 않으면 5억 적용
    - 동거주택 상속공제 = 주택가액 100% (한도 6억)
        - 10년 이상 동거, 1주택 보유자 → 무주택자 상속
    - 금융재산 상속공제 = 순금융가액 2천만까지는 100%, 이후 20% 공제 (한도 2억)
    - 감정평가수수료 = 500만원 한도
- 상속세 과세표준 x 상속세율 = 상속세 산출세액
    - 상속세액 = 10% ~ 50%
    - 자녀가 아닌 직계비속은 30% 할증과세
        - 20억 초과 or 미성년자인 경우 40%할증과세 (단, 자녀가 사망한 경우는 30%로 동일)
- 상속세 산출세액 - 단기재상속세액공제 - 외국납부세액공제 - 증여세액공제 - 신고세액공제 + 가산세 = 신고납부 상당세액
    - 단기재산속세액공제 = 10년이내 사망 시 1년에 10%
    - 외국납부세액공제 = 국외재산을 국외에 납부시 공제
    - 증여세액공제 = 증여재산합산 시 당초납부한 세액공제
    - 신고세액공제 = 자진납부세액 상당액 3%
    - 가산세 = 신고불성실 + 납부불성실
- 신고납부 상당세액 - 분납신청세액 = 신고납부세액

**상속세 납부**

- 상속개시일 있는 달의 말일부터 6개월
- 1천만원 초과 시 2개월 내 분납 가능
- 2천만원 초과 시 담보로 연부연납 가능 (가업 20년 일반 10년), 분납 신청 시에는 불가

### **02 증여세**

증여 : 증여자가 대가 없이 재산을 상대방에게 준다는 의사표시를 하고 상대방이 승낙하여 성립하는 계액

- 세법상 현저히 낮은 대가를 받고 재산을 이전하는 경우 포함
- 세법상 상속세/증여세를 부당하게 감소한 것도 포함

증여세 납세의무 : 영리법인 x, 비영리법인, 비거주자 (국내재산)

증여자 연대납부 : 수증자가 거소불분명, 수증자가 납부능력 없음, 수증자가 비거주자

**증여세 산출**

- 증여재산가액 + 재차증여재산가액 + 증여의제금액 + 증여추정금액 - 비과세재산 - 과세가액불산입재산 = 과세가액
    - 증여가 아닌 경우 : 사인증여(상속세), 증여재산 반환(3월 내), 위자료, 재산분할, 권리말소, 협의분할
    - 증여한 채무에 대해 증여자가 양도소득세 납부
    - 재차증여재산가액 = 동일인에게서 10년간 증여받은 금액을 합산과세
    - 증여의제 : 보험금, 저가양수 고가양도
        - 보험 수취인과 납부자가 다르면 증여로 간주, 증여받아 납부 시 납부액 차감액을 증여재산가액으로
        - |시가-대가| - min (시가 30%, 3억)을 증여가액으로 간주
    - 증여추정 = 배우자, 직계존비속 양도, 특수관계자에게 양도 후 3년 내 양도한 경우, 출처분명 재산취득액
        - 출처불분명 금액 < min(재산취득액 20%, 2억) 이면 증여추정하지 않음
    - 비과세재산 = 국가나 지자체 증여, 장애인 수취인 보험 (4천), 정당이 증여받음, 사회통념상 인정되는 것
    - 과세가액불산입 = 공익신탁, 공익법인출연재산, 장애인이 증여받아 신탁한 경우 / 5억 한다
- 증여세 과세가액 - 증여공제 - 재해손실공제 = 증여세 과세표준
    - 증여공제 = 배우자 6억, 직계존속 5천, 직계비속 5천, 기타친족 1천
    - 세대생략 할증과세 30%
- 증여세 과세표준 x 세율 = 증여세 산출세액
- 증여세 산출세액 - 기납부 세액공제, 외국납부 세액공제 - 신고세액공제 + 가산세 = 신고납부세액

신고 : 증여일 달 말일부터 3월 이내

② 금융재산의 비율이 높으면 부동산 취득도 고려하고, 보유기간이 오래된 부동산은 처분하지 않고 상속을 고려

④ 재산을 처분하거나 채무를 부담하는 경우에는 소명 자료를 철저히 확보하고, 상속개시 직전에는 예금 인출을 하지 않는다. (금융재산상속공제에서 제외됨에 유의)

⑤ 배우자상속공제 및 동거주택상속공제의 활용

⑥ 손자나 손녀에게 직접 상속하는 방법도 고려하고, 협의 분할 후에도 재산재분할도 고려

⑦ 상속재산에 저당권 설정 시 유의

⑧ 전문가의 검증을 받을 것(상속공제 종합한도, 재산재 분할 검토, 상속개시 6개월 이내에 부동산 처분 시 주의)

# 제3과목 보험 및 은퇴설계 (20)

## 1. 보험설계 (10)

### 01. 보험 개론

**위험 종류**

- 치명적 위험: 파산으로 이끌 수 있는 잠재적 손실 노출
- 중요한 위험: 손실회복 위해 외부자금 차입해야 하는 위험
- 일반적 위험: 현재의 소득·자산으로 보전가능한 손실노출

**위험 관리기법**

- 고빈도ㆍ고강도 위험 : 위험회피
- 저빈도ㆍ고강도 위험 : 보험가입
- 고빈도ㆍ저강도 위험 : 손해빈도통제 & 자체조달
- 저빈도ㆍ저강도 위험 : 위험보유 & 경상비 활용

**보험의 원칙**

- 수지상등의 원칙: 순보험료 총액 = 지급보험금 총액 (전체적인 관점)
- 급부반대급부 균등의 원칙: 자신의 위험에 상응하는 보험료 납부 (개별 위험에 상승하게 보험료 산출)
- 대수의 법칙: 발생확률(경험생명표)
- 실손보상의 원칙: 손해보험은 입은 손해만큼 보상받는것이 합리적

**보험료 구성**

보험료 = 순보험료(위험보험료, 저축보험료) + 부가보험료(신계약비, 유지비, 수금비)

- 예정위험률 : 예정사망률↓ → 사망보험료↓, 생존보험료↑
- 예정이율 : 예정이율↓ ⇒ 보험료↑, 보험기간↑, 납입기간↓ ⇒ 보험료 변동폭 大, 순수보장형보다 만기환급형 보험료변동폭 大
- 예정사업비율 : 낮아지면 보험료 낮아짐

**보험계약법**

상개적 강행법성 : 불이익변경금지의 원칙, 적용배제

보험계약의 주요특성: 불요식 낙성계약, 유상 쌍무계약, 사행계약, 부합계약

보험약관의 해석원칙 : 신의성실의 원칙, 계약당사자 의사우선의 원칙, 작성자 불이익의 원칙

철회 : 보험증권을 받은 날로부터 15일(최대 청약일로부터 30일)

취소 : 계약성립일로부터 3개월 이내 취소

지급 : 적립금 반환청구권, 보험료 청구권 3년

미지급

- 피보험자가 고의로 자신을 해친 경우: 기납입보험료를 계약자에게 반환(피자기계)
- 수익자가 고의로 피보험자를 해친 경우: 미지급보험금에 해당하는 해약환급금을 계약자에게 지급(수피해계)
- 계약자가 고의로 피보험자를 해친 경우: 계약자에게 기납입보험료 반환×(계피본다)

**보험 종류**

주계약 : 주보험 + 의무 부가특약

특약 : 보장 추가 확대용(질병, 재해, 입원, 수술, 정기), 가입자 편의용(우량체 할인, 선지급서비스, 연금전환)

분류 : 일반계정/특별계정, 개인/단체, 보장성/저축성, 생존/사망, 배당/무배당, 단생/연생,

확정금리형/연동금리형/실적배당형/자산연계형

연금보험 : 종신연금형, 확정연금형, 상속연금형, 혼합연금형

유니버설보험 : 자유 납입, 중도 인출 가능

변액보험 : 보험 + 투자, 최저보증만 보호, 특별계정 운용, 실적배당률 적용

- 특별계정 투입보험료 : 이체사유 발생일 기준가격으로 일반계정 → 특별계정 이체
- 이체사유 발생일 : 청약철회기간 종료 다음날/승낙일, 납입기일 T+2

**간병보험**

### 04. 손해보험

- 객체는 피보험자의 재산
- 사고발생 여부/시간/규모 불확실

**주택화재보험**

- 대상 : 주택으로만 쓰이는 건물, 명기물건, 자동담도물건

- 예외 : 고의로 생긴 손해, 도난 및 분실, 화재와 무관한 파열, 전기적 사고의 손해
- 보상 : 보험가입금액이 보험가액의 80% 이상인 경우 보험가입 금액 한도 내에서 손해액 전액 지급
    - 잔존물 제거비용은 10%, 기타협력비용은 전액 지금

**화재보험**

- 대상 : 물건
- 손해 : 주택화재보험과 동일 (폭발, 파열에 의한 직접손해 제외)
- 의무 : 건물을 30일 이상 비워두거나 휴업, 15일 이상 수선
- 특약 : 도난위험담보, 구내폭발위험담보, 풍수재위험담보, 기업휴지손해담보, 신체손해배상책임담보

**특종보험**

종류 : 임의/의무배상책임보험, 손해사고기준/배상청구기준 배상책임보험

다중이용업소 화재배상보험 : 화재로 인해 타인의 신체나 재물에 손해를 입혔을 때 보상해주는 의무가입보험

**장기손해보험**

- 보험기간 15년 이내, 만기환급금 부분 운용, 1회 보험금이 보험가입금액의 80% 미만 시 자동복원
- 보험료 납입방법 다양, 보험료의 납입최고와 계약의 해지, 해지계약의 부활가능(2년, 3년 이내), 보험계약대출

**자동차보험**

- 종류 (대인I, 대물 의무가입)
    - 대인배상 I : 사망보험금, 부상보험금, 후유장해보험금
    - 대인배상 II : 대인배상 I을 초과하는 손해를 배상
    - 대물보상 : 1사고당 보험가입금액 한도로 보상 (2천만 의무)
    - 자기신체사고 or 자동차상해

특약

- 가족운전자 한정특약 : 형제·자매는 가족 범위 배제
- 운전자연령 한정운전특약: 사고 당시 만 나이 기준
- 다른 자동차 운전담보특약: 무보험차상해 가입 시 자동 가입

면책 : 고의로 인한 사고

## 2. 은퇴설계 (10)

### **01 은퇴설계 기본이해**

정의 : 근로소득이 없는 은퇴 이후의 삶을 행복하게 영위하기 위해 재무적인 요소와 비재무적인 요소를 균형 있게 설계하는 것

3대 축 : 경제(재무), 건강, 삶의 보람(비재무)

**은퇴환경 변화**

- 기대수명 증가, 고령화 전망
- 문제점 : 노후준비 부족, 노후빈곤 문제, 노후의료비 증가, 노인 간병 문제

**노후에 대한 인식**

- 에릭슨 : 자아통합감 대 절망감의 시기
- 액티브 에이징

위험요소 : 장수 리스크, 인플레이션 리스크, 노후 건강 리스크

은퇴자금 설계 주요 포인트

- 은퇴 후 연금받기까지의 소득공백기인 은퇴 크레바스
- 적립과 인출 시 주의점 (빨리 시작, 늦게 인출)
- 부동산을 활용한 은퇴설계 유의점 : 유동성 리스크, 가격하락 리스트 대비)
- 부부 중심의 은퇴설계 : 부부형 연금, 주택연금
- 노후 필수 자금 : 노후 생활비, 의료비, 장기 간병비용
    - 평균 평생 의료비 1억, 평균 국민연금 수령액 40만원

**비재무적 요소**

은퇴 이후 변화 : 생활리듬 변화, 자신의 입장 변화, 가정 내 역할 변화, 경제감각 변화, 건강상태 변화

은퇴설계 : 가족을 포함한 관계, 고령자 주거, 사회활동과 시간활용, 웰다잉

- 협동조합 : 5인 이상이면 금융업 보험업 제외 설립 가능, 무조건 1인 1표

**연령대별 은퇴설계**

20~30대 : 3층 연금제도 활용(국민연금, 퇴직연금, 개인연금 / IRP 등)

40대 : 소득이 가장 높은 시기, 선저축 후소비, 노후자금과 목돈 만들기 추구

50대 : 자녀지원과 노후준비, 3층 연금제도 활용

60대 : 기존 자산으로 노후자금 활용, 상속 및 증여 준비

### 02. **은퇴 관련 제도**

**우리나라 노후소득보장 체계(3층 연금제도)**

- 공적부조 : 국민기초생활보장제도, 기초연금제도
- 1층 : 국민연금, 특수직역연금
- 2층 : 퇴직연금
- 3층 : 개인연금

**국민연금제도**

- 1988년 10인 이상 사업장 도입
- 연금액은 물가병동률 반영
- 대상 : 18세~60세, 최소가입기간 10년
- 종류 : 사업장 가입자, 지역가입자, 임의가입자, 임의계속가입자
- 납입 : 기준월소득액 x 9%, 사업장가입자는 회사와 본인이 절반 부담
- 수령 : 65세 +/- 5년
- 납입액은 소득공제, 60세 이상은 대출가능 (한도 1000만)

**특수직역연금제도**

공무원연금 : 장기급여 (퇴직급여, 유족급여, 재해보상급여, 퇴직수당), 단기급여 (재해보상급여, 부조급여)

군인연금제도 : 퇴역연금 (20년 이상 복무), 퇴직일시금 (20년 미만 복무)

- 공적연금 연계제도 : 각 가입기간이 10년 이상이면 연계연금 수령 가능

**퇴직연금제도**

종류 : 확정급여형, 확정기여형, IRP

- DB, DC도 IRP에 연간 1800만원 납입가능

수급 : 55세 이상, 연금수급 5년 이상

세액공제

- 연간 소득 5,500만원 이하 : 전체 900만원 한도, 세액공제율 16.5%
- 연간 소득 5,500만원 초과 : 전체 900만원 한도, 세액공제율 13.2%

연금과세

- 만 70세 미만: 한도 내 5.5%, 한도 초과 16.5%
- 만 70세 이상~만 80세 미만: 한도 내 4.4%, 한도 초과 16.5%
- 만 80세 이상: 한도 내 3.3%, 한도 초과 16.5%

**개인연금제도**

- 개인연금저축, 연금저축, 연금저축계좌 (연금저축신탁, 연금저축펀드, 연금저축보험)
- ISA만기 시 연금저축계좌도 이체 가능
- 1500만원 초과 시 종소세 과세

- 연금저축신탁: 은행, 자유적립식, 실적배당, 확정기간형, 원금보장, 예금자보호
- 연금저축펀드: 증권사/은행/보험사, 자유적립식, 실적배당, 확정기간형, 원금비보장, 예금자비보호
    - 5년이상, 55세 이후, 연600만 공제
- 연금저축보험: 증권사/은행/보험사, 정기납입, 공시이율, 확정기간형/종신형, 원금보장, 예금자보호

**주택연금제도**

- 개요 : 주택금융공사가 보증서를 발급하면 은행이 주택연금을 지금
- 대상 : 부부 중 1명이 만 55세 이상, 공시가 12억 이하
- 지급 : 계약기간동안 거주 보장, 사망시에도 배우자에게 지급

**농지연금제도**

- 개요 : 60세 이상 농업인의 농지를 담보로 연금을 받음
- 대상 : 60세 이상 농업인, 과거 5년 이상 영농경력
- 지급 : 종신형, 기간형 / 사망 시 배우자에게 지급

**노랑우산공제제도**

- 개요 : 사업체
- 대상 : 소기업, 소상공인 범위의 개인사업자/법인대표
- 지급 : 60세 이상 10년 이상 납부시 청구 가능
- 공제 : 연간 최대 200만원

**노인장기요양보험제도**

- 대상 : 65세 이상
- 본인부담 시설 20%, 방문 15%

**성년후견제도**

- 종류 : 성년후견, 한정후견, 특정후견

### **03. 은퇴설계 프로세스**

1단계: 고객과 관계정립 및 정보수집

① 고객과 관계정립을 하고 전문가로서 신뢰 구축

② 정보수집: 면담 방법과 질문지를 활용한 방법 병행

③ 은퇴설계 목표의 구체화: 목표의 명확화 & 수치화

④ 라이프 이벤트 표 작성

2단계: 고객 현황 분석 및 은퇴설계 제안

① 현금흐름표 분석, 가계대차대조표 분석, 보장 분석, 금융자산의 포트폴리오 분석, 세금 분석

② 제안 내용 검토: 노후 준비자금 재검토, 목돈 지출에 대한 재검토, 가계수지 전체나 보유자산의 재검토, 수입을 늘리는 대책 검토, 라이프 플랜 변경에 따른 재검토, 고객에게 추천할 대안 검토

③ 대안을 여러 개 제시해서 고객이 최종적인 선택

3단계: 실행 지원 및 사후관리

① 금융상품 선택 시 고객 이익을 최우선해 상품 추천

② 사후관리 : 매년 1~2회 제안서 재검토(사회경제적 환경변화와 고객의 생각 변화에 대응)
