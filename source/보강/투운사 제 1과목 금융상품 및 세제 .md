# 제 1과목 : 금융상품 및 세제 (20)

소유자: 조동희 | 서울 의학과 ­

## 1. 세제관련 법규 및 세무전략 (7)

### 01 조세 총론

**종류**

국세 : 내국세 (직접세, 간접세, 목적세), 관세 // 지방세 : 보통세, 목적세

![image.png](%EC%A0%9C%201%EA%B3%BC%EB%AA%A9%20%EA%B8%88%EC%9C%B5%EC%83%81%ED%92%88%20%EB%B0%8F%20%EC%84%B8%EC%A0%9C%20(20)/image.png)

**서류 송달**

교부송달 : 송달장소에서 교부

우편송달 : 등기우편으로 교부

전자송달 : 신청에 한해 교부

공시송달 : 공고 14일 경과 시 송달로 간주

**납세의무 성립**

- 소득세, 법인세, 부가가치세 : 과세기간이 끝나는 때
- 상속세 : 상속 개시 시점
- 증여세 : 재산 취득 시점
- 인지세 : 과세문서 작성 시점
- 증권거래세 : 매매 확정 시점
- 종합부동산세 : 과세기준일
- 원천징수 : 지급 시점

**납세의무 확정**

- 신고확정 : 소득세, 법인세, 부가가치세 등 대부분
- 부과확정 : 상속세, 증여세
- 자동확정  : 인지세, 원천징수 소득/법인세 등 특수 소득/법인세

**납부의무 소멸**

사유 : 처분 취소, 국세부과 제척기간 만료, 국세징수권 소멸시효 완성

- 제척기간 : 일반조세 10년, 국제부정 15년 // 상속증여세 15년 (예외존재)
- 소멸시효 : 5년 (5억 이상 10년) / 단, 납부고지, 독촉, 교부청구, 압류 시 시효 정지

**납부의무 확장**

승계 : 합병법인, 상속인 (상속재산 한도)

이차 : 청산인, 출자자, 법인, 사업양수인

**납세관련 정정**

기한 내 : 덜 냈을 때 수정신고 (2년), 더 냈을 때 경정청구 (5년)

기한 후 : 기한후과세표준신고서 제출 (6개월)

**조세불복제도** : (이의신청), 심사청구, 심판청구 / 90일 이내 제기

**국세우선의 원칙**

변제순서 : 예외 > 국세채권 > 일반채권

예외 : 선집행 체납처분비, 절차에 사용된 비용, 법정기일 전 설정된 채권, 우선변제임차보증금/채권

### **02 소득세**

특징 : 종합과세제도, 열거주의, 신고납세, 개인단위, 누진과세, 주소지 과세

(종합과세 = 이자, 배당, 사업, 근로, 연금, 기타소득을 인별로 종합)

**납세의무자 분류**

- 거주자 : 국내 주소 or 183일 이상 거소를 둔 개인
- 비거주 : 거주자가 아닌 개인

**원천징수**

- 완납적 : 원천징수로 종결, 납세의무자는 확정신고의무가 없음 (분리과세)
- 예납적 : 원천징수만큼을 가납부세금으로 산정하여 추후 제해줌 (종합과세)

**방법**

- 거주자 : 종합과세, 분류과세 (양도소득, 퇴직소득), 분리과세
- 비거주 : 종합과세 (국내 사업장/부동산), 분리과세 (그 외)

**A. 이자소득**

특징 : 필요경비 인정되지 않음, 비과세/분리과세 이자소득은 제외

대상

- 채권증권 이자/환매조건부 매매차익, 예금적금 이자, 저축성 보험 보험차익, 직장공제회
- 이자관련 파생상품에서 얻은 이익

**B. 배당소득**

특징 : 필요경비 인정되지 않음, 비과세/분리과세 배당소득은 제외

❗집합투자기구로부터의 이익은 배당소득으로 분류됨

대상

- 이익의제외국법인 배당, 법인 배당, 집합투자기구 이익, 인정배당(법인세법)
- 수익분배의 성격이 있는 것(의제배당), 배당소득 관련 파생상품에서 얻은 이익
    - (의제배당 수입시기)
        - 감자 : 감자결의일
        - 해산 : 잔여재산가액 확정일
        - 합병 : 합병등기일
        - 분할 : 분할등기일
        - 잉여금 자본전입 : 자본전입결의일

배당세액공제제도 : 법인세만큼 배당소득에 가산 (간주배당), 이후 차감 / 이중과세 조정이 목표

제외 : 집합투자기구에서 상장주식 매매차익, 주식기반 파생 매매차익, 벤처기업 주식 매매차익

**C. 금융소득 (이자소득 + 배당소득)**

비과세 금융소득

- 공익신탁, 10년이상 1억 이하 장기저축성보험
- 농어거목돈마련저축 이자, 비과세종합저축, 조합 예탁금 (3천), ISA

무조건 분리과세 : 분리과세 (종결)

- 직장공제회 초과반환금 (기본세율), 비실명거래 이자배당소득 (45%, 90%)
- 법원 납부로 생기는 이자소득 (14%), 1거주자 단체 이자배당소득 (14%), ISA 한도초과 (9%)

조건부 종합과세 : 무조건 분리과세 대상 이외 이자/배당소득

- 2천만원 이하면 분리과세 (14%) 이상이면 종합과세

무조건 종합과세 : 위의 경우에도 국외 지급은 종합과세

원천징수제도 : 탈루방지 및 징수편리를 위해 소득지급자가 대신 납부

종합과세 산출세액 **:** 둘 중 큰 걸로

- 일반산출세액 : (종소과표 - 2천) x 기본세율 + 2천 x 14%
- 비교산출세액 : (종소과표 - 금융소득) x 기본세율 + 금융소득 x 14%

    (최저 세율이 14%미만이 되지 않게 하는 것)


⚠️세율 정리

09% : ISA 계좌 한도 초과 금액 이자배당소득

14% : 법원 납부 보증금의 이자소득, 기타 이자배당소득

25% : 비영업대금의 이익

30% : 분리과세를 신청한 장기채권의 이자, 할인액

45% or 90% : 비실명거래로 인한 이자배당소득

기본세율 : 직장공제회 초과반환금

**D. 양도소득 (분류과세!)**

특징 : 열거소득에 대해 과세 (채권 - 비열거)

대상

- 부동산 (토지, 건물, 부동산 관련 권리)
- 주식 (주식 및 출자지분, 비상장주식 / 소액주주 제외)
- 기타 : 시설물 이용권, 사업용 고정자산 + 영업권, 특정주식
- 특정주식A : 부동산이 자산 50% 이상인 법인 주식, 50%이상 양도 시
- 특정주식B : 부동산이 자산 80% 이상인 법인 주식, 1주라도 양도 시

계산

- 양도가액 = 실거래가 → 매매사례가 → 감정가 → 환산가 → 기준시가
- 양도차익 = 총수입금액 - 필요경비 (취득가액, 기타필요경비)
- 양도소득금액 = 양도차익 - 장기보유특별공제 (3년 이상)
- 양도소득과세표준 = 양도소득금액 - 양도소득기본공제 (연 250)

**E. 종합소득세**

요약 : 금소 2천 이상, 아니어도 사업소득 or 300만 이상 기타소득이 있으면 대상

(단, 비과세, 무조건 분리과세 금융소득은 제외)

대상

- 금융소득 > 2000만원 : 신고납부 의무
- 금융소득 < 2000만원 : 이외 소득에 따라
    - 근로소득, 소득이 없는 경우 : 무관
    - 사업, 기타소득 (연 300만원) : 신고납부 의무

기한 : 다음 해 5월 1일~31일

### **03 증권거래세**

**대상**

- 국내법에 따라 설립된 법인의 주권 양도 (상장과 무관)
- 외국법인의 주권이 한국거래소 상장된 경우

❗비과세 : 국가 or 지자체의 양도, 발행매출, 대차거래

**세율**

- 코스피 : 0.03%
- 코스닥 : 0.18%
- 코넥스 : 0.10%
- 비상장 : 0.35%

**기한 :** 금투업자 내달 10일, 그 외 반기 말일부터 2개월 내

단체의 납세의무 : 법인격 없는 단체 중 법인으로 간주하는 단체는 법인세

### 04 절세전략

상속세 : 신고세액공제, 금융자산공제

증여세 : 세대생략증여, 저평가활용, 10년단위 증여, 레버지리

성인자녀 10년 5천, 미성년자 10년 2천 비과세

## 2. 금융상품 (8)

### 01 금융 총론

**금융기관**

은행권

- 은행 : 한국은행, 일반은행, 특수은행 →  고유업무, 겸영업무, 부수업무
- 비은행 예금취급기관 : 상호저축은행, 신용협동기구(~조합), 우체국예금

제2금융권

- 보험사 : 생명보험 / 손해보험회사, 공제, 우체국보험
- 금융투자 : 투자매매업, 투자중개업, 집합투자업, 투자자문업, 투자일임업, 신탁업
- 여신전문 : 카드회사, 할부금융회사, 창업투자회사, 대부회사
- 증권금융 : 순수민간영리법인, 증권인수금융, 증권유통금융, 증권담보금융

**금융투자상품**

- 증권 : 원본초과손실 가능성이 없는 투자상품 (~증권)
- 파생 : 원본초과손실 가능성이 있는 투자상품 (선물, 옵션 등)

**개인종합자산관리계좌 (ISA)**

- 직전 3년간 한번도 금융소득종합과세 대상이 아닌 경우
- 연 2000만원, 5년 1억원 까지 납입 가능, 인출 제한 없음, 3년 의무가입

일반형 : 19세 or 15~19 근로 / 200만원 비과세

서민형 : 총급여 5천만 or 종합소득 3800만 / 400만원 비과세

농어민 : 종합소득 3800만 이하 농어민 / 400만원 비과세

중개형 : 펀드, ELS, 리츠, 회사채 등 + 국내상장주 / 투자자 선택

신탁형 : 펀드, ELS, 리츠, 회사채 등 + 예금 / 투자자 선택

일임형 : 모델포트폴리오 선택

비과세 상품 : 비과세종합저축, 조합출자금, 조합예탁금, 농어가목돈마련저축

**현금관리계좌 (CMA, cash management account)**

의의 : 입출금이 자유로우면서 금액을 안전한 단기 금융상품에 투자하는 계좌

종류 : MMF형, RP형, MMW형 / 전부 예금자 보호 대상이 아님

**연금저축상품**

- 연금저축보함 (생명보험, 손해보험)
- 연금저축신탁 (은행)
- 연금저축펀드 (금융투자)

### **02 예금성 금융상품**

**종류**

요구불예금 : 보통예금, 당좌예금, 별단예금

저축성예금 : 확정금리 (정기예금, MMDA, 정기적금), 실세금리

**정기예금** : 예치기간을 정하여 만료 전에는 지급청구 불가

**MMDA** : Marcet Deposit Account (시장금리부 수시 입출금식예금)

→ 매일 예치금액마다 다르게 이자 적용

**재형저축** : 3년 고정금리로 이자 소득세를 부과하지 않음

- 5천만 이하 근로소득자 or 종합소득금액 3500만 이하 개인사업자 대상
- 최소 7년, 최대 10년 / 만기해지 시 이자소득세 면제

**주택청약 관련**

주택청약저축 : 국민주택 청약권이 주어짐 / 예금자 보호

주택청약부금 : 저축 금액에 따라 대출해줌 / 예금자 보호

→ 주택청약 종합저축으로 통합, 기존 신규가입은 중단됨

**시장성상품**

- 양도성예금증서 (CD) : 할인식 정기예금증서, 중도해지 불가, 양도 가능
- 환매조건부채권 (RP) : 일정기간 후 고정수익율 환매수, 중도 환매 가능 (깎임) / 예금자보호 x
- 표지어음 (Cover bill) : 금융이관이 발행한 어음, 중도해지 불가, 양도 가능
- 기업어음 (Commercial Paper) : A123, BCD

### **03 신탁상품**

신탁 : 위탁자 - 수탁자 - 수익자 (자익/타익) / 3면관계

선택 : 운용대상, 운용방법, 운용조건 초기 지시, 운용자 선택 아님

종류

- 금전신탁 : 특정금전신탁, 분리과세특정금전신탁, 연금저축신탁
- 재산신탁 : 금전채권신탁, 유가증권신탁, 부동산신탁 (관리, 처분, 담보)

특징

- 강제집행, 경매 불가, 수탁자 상속재산 아님
- 수탁자가 사망하여도 존속됨
- 재산은 수탁자에게 귀속, 실질적으로는 수익자에게 귀속

**연금저축신탁**

- 납입 : 연 1800 + ISA 만기전환금액만큼 납입 / 의무 5년
- 공제 : 연 400만 + 연 200만 (50세) + ISA 만기전환금액 10%
- 수령 : 만 55세 이후부터 수령 (5.5% - 4.4% - 3.3%) / 의무 10년

**예금자보호제도**

기관 : 은행, 보험사, 증권사, 종금사, 상호저축은행

유사 : 우체국, 신용협동기구 (조합, 새마을, 농수협단위조합)

예금 : 1인당 5천만원까지 보장

상품 : 예금, 부금, 어음 / 예수금 / 일반 보험계약 / 발행어음, 표지어음, CMA

### 04 보험상품

**A. 생명보험**

원리 : 대수의 법칙, 수지상등의 원칙, 사망률과 생명표

특징 : 미래지향, 수혜자는 타인, 가격에는 목표이윤 없음

영업보험료 = 순보험료 (위험보험료 + 저축보험료) + 부가보험료 (신계약비, 유지비, 수금비)

분류 : 보장성험, 저축성 / 정액, 변액 / 단생, 연생, 단체취급, 단체 / 체증, 체감

종류 : 사망, 생존, 양로

**B. 손해보험**

특징 : 물보험, 실손보상, 피보험이익이 있어야 계약 성립, 보험가액 존재

보험자 대위 (잔존물 대위, 청구권 대위)

종류 : 화재보험, 해상보험, 특종보험, 장기손해보험

**B-1. 화재보험**

정의 : 우연한 화재사고로 보험목적물이 입은 손해를 보상

종류 : 일반화재보험, 주택화재보험

목적 : 건물, 동산, 시설, 공기구, 비품, 가재도구, 기계

보상 : 화재손해 (직접, 소방, 피난 - 5일), 폭발손해, 잔존물제거비용 (10%)

비보상 : 고의 또는 중대한 과실, 화재 후 도난 혹은 분실, 전기에 의한 피해

**B-2. 자동차보험**

종류 : 책임보험, 상해보험, 물보험 / 자동차종합보험

의무 : 모든 자동차, 9종 건설기계 / 구내 운행, 미군, US, 외국인, 피견인 제외

배상 : 대인 (1인당 사망 1.5억, 부상 3천, 장해 1.5억), 대물 (1사고당 2천)

**B-3. 장기손해보험**

특징

- 3년 이상, 만기환급금 o, 보험금액 자동복원, 다양한 납입주기
- 일반손해보험과 다르게 저축보험료 있음 (저축의 성격 가짐)
- 1회 사고 보험금이 가입금 80% 미만이면 자동 복원



### 05 집합투자 (펀드)

정의 : 2인 이상에게 판매한 자금으로 운용지시 없이 결과를 배분하는 것 (49인 아님)

특징 : 집단성, 간접성, 실적배당과 투자자평등원칙, 펀드자산 분리, 분산투자

종류 : 투자신탁, 투자 / 투자유한 / 투자합자 / 투자유한 회사, …

(투자신탁 - 수익증권, 그 외 - 지분증권)

구성 : 위탁자 - 수익자 - 수탁자

**펀드유형**

- 증권집합투자기구 : 50%이상 증권에 투자
- 부동산집합투자기구 : 50%이상 부동산에 투자
- 특별자산집합투자기구 : 50%이상 특별자산에 투자
- 혼합자산집합투자기구 : 비중제한 없음
- 단기금융집합투자기구 : 전부 단기금융상품 (단기 : 75일 이내) /  = MMF

* 파생은 분류에 없다

* **MMF (단기금융집합투자기구)**

투자대상

- 잔존만기 6개월 이내 CD
- 잔존만기 5년 이내 국채증권, 1년 이내 지방채 증권 / RP는 무관
- 잔존만기 1년 이내 어음
- 30일 이내 단기대출
- 다른 MMF 증권, 단기사채

운용방법

- 잔존 가중평균 만기는 개인 75일, 법인 120일, 그 외 60일 이내
- 잔존만기 1년 이상 국채증권은 5% 이내 운용
- 채무증권은 최상위/차상위등급에만 투자, 재산의 40% 이상
- 환매조건부매도는 펀드재산 5%이내

**특수한 형태의 집합투자기구**

환매금지형 : 시장성 없는 자산이 20% 초과 or 부동산, 특별자산, 혼합자산

종류형 : 증권을 여러 종류로 발행 (보수, 수수료 차이)

전환형 : 타 펀드로 전환가능

모자형 : 모 펀드에 유일하게 투자 가능한 자 펀드를 공모

ETF : 주식처럼 매매 가능하게 만듬, 30일 이내  상장

집합투자증권 매도 : 15일 이내 대금 지급, 폐쇄형은 설립 90일 내 상장

**집합투자기구 평가**

집합투자재산 평가 : 시가평가 (MMF는 장부평가 가능, 0.05%변동 시 고시), 없으면 공정가액

집합투자기구 회계 : 말일 2개월 내 감사

예외 : 총액 300억 이하, 총액 300~500억 & 직전 6개월간 발행 없음

집합투자기구 관리 : 소극적 감시, 0.3 이상 산정가격 차이 시 시정요구

자산운용보고서 3개월에 1회 이상 보고

사모투자기구 : 100인 이하 (일반투자자 49인 이하)

### **06 주가지수연동형 상품**

**A. ELD (Equity-Linked Deposit, 주가연계예금)**

의의 : ‘예금’처럼 예치, 제시수익률로 보장, 예금자 보호

주관 : 은행

**B. ELS (Equity-Linked Security, 주가지수연계증권)**

의의 : 주가 지수와 연계한 ‘증권’, 예금자 비보호,

채권 이자만큼을 옵션에 투자하는 방식 등 사용 (cash extraction)

주관 : 증권사 (공모, 사모 가능)

구조

- knock-out : 목표 도달 시 조기상환 (채권 + 콜/풋 매수)
- bull-spread : 만기시점에 비례 (채권 + 낮은 콜 매수 + 높은 콜 매도)
- digital : 만기시점에 기준점 이상/이하를 따져 지금 (채권 + 콜/풋 매수)
- reverse-convertible : 기준점 이상 시 정해진 수익 (채권 + 콜 매도)

**C. ELF (Equity-Linked Fund, 주가연계펀드)**

의의 : 다양한 ELS에 투자하는 ‘펀드’ → 간접적으로 지수를 따라가는 펀드

(ETF : 직접 지수를 따라가는 펀드)

주관 : 운용사

**D. ELW (주식워런트증권)**

정의 : 기초자산을 정해진 가격에 사거나 팔 수 있는 권리가 부여된 ‘파생’성격 증권

가격 : 행사가치 + 시간가치

### 07 자산유동화증권 (ABS)

정의 : 보유 자산을 집합하여 증권을 발행 후 현금흐름을 이용하는 것, 자산을 양도

효과 : 자산유동화, 현금조달, 투자선택 다양성 확보

신용보강 : 외부(신용공여, 지급보증), 내부(후순위증권, 초과스프레드, 예치금, 풋백옵션)

종류 : CBO(회사채), CLO(대출채권), MBS(주택저당채권), 부동산 PF, 카드

**부채담보부연계증권 (CDO)** : 반환될 채무를 기반으로 한 유동화증권

**모기지담보부증권 (MBS)** : 장기 발행, 높은 신용등급, 수익률 높음

**다계층증권 (CMO)** : (floater, inverse F, super F, PAC, TAC, VADM )

### 08 퇴직연금제도

의의 : 퇴직금을 적립한 뒤 퇴직 시 지급하는 제도

원리금 비보장자산 투자한도 70%, 파생은 헤지 목적인 경우만 허용

종류

- 확정급여형 (DB) : 중도인출 불가, 기업 운용책임 - 급여가 사전확정
- 확정기여형 (DC) : 중도인출 가능, 직원 운용책임 - 부담금이 사전확정

개인형퇴직연금제도 (IRP) : 이직해도 계속 적립 가능

## 3. 부동산관련 상품 (5)

### 01 부동산 개론

**정의**

- 협의 : 토지와 그 정착물 (민법)
- 의제 : 공사수단을 갖추어 거래 객체로 인정되는 것 (공장재단, 광업권, 선박)
- 복합 : 다양하게 정의

**특성**

- 자연적 : 부동성, 부증성, 개별성, 영속성
- 인문적 : 용도의 다양성, 합병분할의 가능성, 사회/경제/행정적 위치 가변성

**권리**

- 점유권 : 사실상의 지배를 인정
- 소유권 : 사용 / 수익 / 처분을 할 수 있는 권리
- 제한물권 : 어떤 목적을 위해 타인의 물건을 제한적으로 지배하는 물권
    - 용익물권 : 지상권, 지역권, 전세권 / 사용가치 지배
    - 담보물권 : 유치권, 질 권, 저당권 / 교환가치 지배

**부동산 등기**

의미 : 등기부에 부동산 및 관련 권리관계를 기재하는 것

가등기 : 실체법상 요건이 불비한 경우

특징 : 공시력(권리를 알림) 인정되나 공신력(공적으로 신뢰)은 인정되지 않음

* 동산 : 공신력 있음 (선의취득)

### 02 부동산 관련 규제

**토지이용계획 :** 국토계획, 광역도시계획, 도시계획, 기타계획

**지목에 따른 구분 :** 전, 답, 대, 과수원, 목장용지, 도로, 하천, 구거, 유지, 묘지

**목적에 따른 구분**

용도지역 : 모든 토지에 적용

- 종류 : 도시(주거, 상업, 공업, 녹지), 관리 농림 자연환경보전 지역
- 건축제한 : 아파트 (도시지역 - 1종 전용/일반 주거지역 등)

용도지구 : 특정 목적을 위해 지정

- 종류 : 경관 고도 방화 방재 보존 시설보호 취락 개발진흥 특별용도제한
- 건축제한 : 도시계획조례에 따름

용도구역 : 그냥 지정

- 종류 : 개발제한 도시지역공원 시가화조정구역 수산자원보호 구역
- 건축제한 : 지정목적에 위배되는 건축

* 건폐율(건축면적/대지면적), 용적률(지상층 연면적/대지면적) 제한

**건축**

신축 : 없는 곳에 축조

증축 : 있는 곳에 추가

개축 : 일부 철거하고 추가

재축 : 날라간 곳에 다시 축조

이전 : 해체 없이 위치만 옮김

### 03 부동산투자

**부동산 시장**

특징 **:** 국지성, 비표준화성, 비공개성, 높은 거래비용, 부증성, 장기적

변동 : 경기회복 → 경기호황 → 경기후퇴 → 경기불황 →

지표 : 건축의 양, 가격변동, 거래량

투자 프로세스

- 투자목표 및 제약조건 확인
- 시장환경분석 (부동산경기상황, 위험 등)
- 현금흐름예측 (운용 현금흐름, 매도 현금흐름)
- 투자의 타당성 분석 (간편법, 현금흐름할인법)
- 투자결정

* 현금흐름할인법 : 현재가치로 환산하여 계산

순현재가치 (NPV) = 현금유입 현재가치 - 투자액

내부수익률 (IRR) = NPV를 0으로 만드는 할인률

수익성지수 (PI) = 부동산 투자 수입 현재가치 / 부동산 투자액

(순이익이 얼마인가 / 수익률이 얼마인가 / 투자금 대비 수익이 몇 배인가)

CoC (Cash on Cash) 수익률 : 순현금흐름 / 자기자본 → 시간가치 반영 안함

대출비율 : 저당대출원금 / 부동산가격

부채상환비율 : 순운용소득 / 부채상환액

**부동산관련 서류**

- 토지이용계획 확인서 : 법적 제약 확인
- 지적공부 : 토지 등록정보 확인 - (토지대장, 임야대장, 지적도 등)
- 건축물대장 : 건축물 등록정보 확인
- 등기부등본 : 소유권 및 권리관계 확인

**부동산 가격**

- 시장가격 : 매도/매수호가
- 기준시가 : 국세 부과 기준
- 시가표준액 : 지방세 부과 기준
- 감정평가액 : 감정평가사가 조사, 평가한 가격
- 공시지가 : 단위면적당 가격

### 04 부동산관련 금융상품

**부동산 투자회사 (REITs)**

종류 : 자기관리REITs, 위탁관리REITs, 기업구조조정REITs

- 각각 6개월 이내 70억 / 50억 / 50억의 최저자본금 필요
- 자기관리REITs외에는 본점 외 지점을 두는 것, 직원 고용 및 상근임원 두는 것 금지

특징

- 현물출자 설립 불가, 발기설립만 가능
- 자산 70% 이상 부동산에 투자, 80% 이상 부동산 및 관련 증권, 현금에 투자
- 국내소재 1년 내 처분금지, 국외소재 정하는 기간 내 처분금지
- 이익배당한도 90% 이상 배당
- 2년 이내 발행주식 30% 이상 일반청약
- 자금차입은 자기자본 2배 초과 불가, 주총결의로 10배까지 가능
- 타 회사의 의결권 주식 10%, 동일인 증권 5% 초과 취득 금지

단, 기업구조조정 REITs는 다음이 적용되지 않음

- 주식 30% 일반청약 제공
- 부동산 처분기간 제한
- 부동산 및 관련 증권 80% 규정 (부동산 70%는 여전히 적용)
- 나대지 개발사업 전에 처분 금지



**대출형 부동산펀드 (Project Financing)**

구조 : 미래에 발생할 현금흐름을 담보로 개발자금을 조달

특징 : 담보의 한정, 위험 배분, 복잡한 절차, 높은 비용

위험 : 시장위험, 사업위험, 주체위험, 재무적위험, 채권보전위험 → 저당권/담보신탁으로 안정성 확보

담보확보 방법

- 저당권 : 법원경매로 채권실행, 후순위권리설정 가능
- 담보신탁 : 신탁회사에서 직접매각, 부담액 낮음, 환가액 높음

종류

- Bridge loan : 인허가 전 급하게 필요한 자금의 조달에 사용
- 본 PF : 허가 후 착공시점에 진행, 비교적 장기간

### 05 부동산 가치평가

**부동산 가치**

- 가치 발생요인 : 효용, 상대적 희소성, 유효수요
- 가치 형성요인 :  일반적 (사회적, 경제적, 행정적) / 지역적 / 개별적
- 가치 추계원칙 : 예측, 수요 공급, 최유효이용, 외부성의 원칙

**감정평가**

- 비교방식 : 시장성 / 거래사례비교법, 임대사례비교법 → 주된 평가법
- 원가방식 : 비용성 /  복성식 평가법 (재조달원가 - 감가수정액)
- 수익방식 : 수익성 / 수익환원법 (순수익 / 환원이율)

**가치분석**

A. 영업상의 현금흐름

- 가능총소득(PGI) = 면적 당 임대료 x 임대가능면적
- 유효총소득 (EGI) = PGI - 공실 + 기타소득 (주차장 등)
- 순영업소득 (NOI) = EGI - 영업경비
- 세전 영업현금흐름 (BTCF) = NOI - 부채서비스액
- 세후 영업현금흐름 (ATCF) = BTCF - 소득세/법인세

B. 매각현금흐름

- 순매도액 = 매도가격 - 매도경비
- 세전 매각현금흐름 = 순매도액 - 미상환 대출잔액
- 세후 매각현금흐름 = 세전 매각현금흐름- 자본이득세(양도소득세)
