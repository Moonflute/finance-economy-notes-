### 2. 금융투자상품 및 금융투자업

**2.1 금융투자상품 :** 원금손실 가능성(투자성)이 있는 모든 금융상품

**증권**

정의 : 원금 초과 손실 가능성은 없는 것

종류  (6)

- 채무증권 : 특수채, 기업어음, 이자연계 파생결합채권 (사적인 금전채권은 제외)
- 지분증권 : 주권, 신주인수권, 출자지분 (무한책임사원 지분 제외)
- 수익증권 : 금전식탁, 투자신탁의 수익권
- 투자계약증권 : 공동사업에 투자하고 손익을 받기로 한 권리가 표시된 것
- 파생결합증권 : 수익이 외생적 지표에 따라 결정되는 모든 증권 (ELW, ELS, DLS, CLN)
    - 제외 : 이자연계, 옵션파생상품의 권리,  사채 일부 등
- 증권예탁증권 : 위 5가지 증권을 예탁받은 자가 발행국 외의 국가에서 발행한 것

**파생상품**

정의 : 원금 초과 손실 가능성 있음

종류

- 장내/장외
- 선도, 옵션, 스왑

* 금융투자상품이 아닌 것 : 원화 CD, 관리형 신탁 수익권, 주식매수선택권 (스톡옵션, ↔ 청구권)

**2.2 금융투자업**

정의 : 이익을 얻을 목적으로 금융투자상품 거래 등을 영업으로 하는 것.

종류

- 투자매매업 : 자기의 계산으로 매매, 권유, 청약하는 직종 / 적용배제 : 본인이 발행
- 투자중개업 : 타인의 계산으로 매매, 권유, 청약하는 직종
- 집합투자업 : 펀드를 운용
- 투자자문업 : 자문에 응하는 직종
- 투자일임업 : 전부 또는 일부를 일임받아 투자자별로 운용하는 것
- 신탁업 : 저작권신탁 등

* 온라인소액투자중개업

정의 : 온라인상에서 진행되는 투자중개업

조건 : 금융위 등록, 5억원 이상의 자기자본

규제 : 투자한도

기타 : 자기의 계산으로 취득 불가

**2.3 투자자**

**전문투자자**

- 절대적 : 일반투자자 대우를 받을 수 없음 (국가, 한은, ~공사, 금투협, 금감원, 은행, 외국정부, 집합투자기구 등)
- 상대적 : 일반투자자 대우를 받겠다는 의사를 통지한 경우, 장외파생거래 시 일반투자자 대우

                 (주권상장법인, 기금관리법인, 지방자치단체, 공제사업법인)

- 자발적
    - 법인 : 100억원 이상 금융투자상품 잔고 보유
    - 개인 : 5000만원 이상 + (연소득 1억+ or 부채차감자산 5억+ or 전문분야 1년이상 종사)

**일반투자자**

전문투자자가 아닌 투자자, 적합성원칙, 적정성원칙, 설명의무가 적용됨

### 3. 금융투자업자에 대한 규제, 감독

- 상법에 따른 주식회사만 가능

**3.1 인가/등록**

등록 대상 : 투자자문업, 투자일임업, 온라인소액투자중개업, 일반사모집합투자업

인가 대상 : 투자매매업, 투자중개업, 집합투자업, 신탁업

- 과정 : 예비인가 (2개월) → 인가 (3개월)
- 요건 : 충분한 자기자본 (max(5억, 법률) → 이후 70% 이상 유지), 전문인력 5인 이상
    - 전문인력 : 관련 업무 2년 이상 종사한 자
    - 1명 : 투자자문업
    - 2명 : 투자일임업, 일반사모집합투자업, 신탁업
    - 3명 : 집합투자업
    - 5명 : 투자매매업, 투자중개업
    - 일반사모집합투자업 - 10억
- 유지 : 인가 요건 유지 못하는 경우 금융위가 취소 가능

**3.2 자산건전성**

분류 : 정상(0.5) - 요주의(2) - 고정(20) - 회수의문(75) - 추정손실(100) (대손충당금)

산정 : 매분기마다 자산과 고정 이하 분류 채권에 대해 산정

**3.3 순자본비율 규제**

의의 : 현금화 가능 자산 > 상환의무 부채 가 항상 유지되어야 함

원칙

- 연결재무제표의 장부가액 기준
- 시장위험과 신용위험 둘 다 고려
- 차감항목은 위험액 산정하지 않음
- 부외자산과 부외부채도 위험액 산정

산정

- 순자본비율 = (영업용순자본 - 총위험액) / 필요유지자기자본
    - 영업용순자본 : 자산총액 - 부채총액 - 차감항목 + 가산항목
    - 총위험액 : 시장위험액 + 신용위험액 + 운영위험액
    - 필요유지자기자본 : 업무별로 요구되는 자기자본의 합
    - 매일 산정, 매월말 기준 1개월 내 금감원장 제출

### 4. 투자매매업자 및 투자중개업자에 대한 영업행위 규제

**4.1 업무 관련 규제**

매매형태 명시 : 매매업자인지 중개업자인지 사전에 밝혀야 함

자기계약 금지 : 예외 - 시장을 통한 매매, 본인 판매 집합투자증권 매수, 금융위가 정하여 고시하는 경우

- 자기주식 취득 시 3개월 이내에 처분해야 함

최선집행 의무 : 상장주권에 해당 (채무증권, 비상장증권은 예외)

임의매매 금지 : 청약이나 주문을 받지 않고는 매매 금지

**4.2 불건전 영업행위 규제**

선행매매 금지 : 예외 - 정보를 이용하지 않음을 입증 or 객관적으로 명확한 경우

스캘핑 금지 : 예외 - 조사분석자료가 매매를 유도하지 않는 경우, 새로운 내용이 없는 경우

일임매매 금지 : 예외 - 투자일임업의 형태로 가는 경우, 일임받을 필요가 있는 경우

조사분석자료 공표 후 24시간 이내 매매금지 (일부 예외)

조사분석자료 작성자에게 기업금융 업무와 연동된 보수 지급금지

계약 채결 후 40일 이내 관련 조사분석자료 공표제공 금지

* 단, 고객의 요청이라면 24시간 이내라도 매매가능

기타 불건전영업행위 금지 : 65세 이상은 녹취해야 함

### 5. 증권 발행시장 공시제도

**5.1 증권신고서**

의의 : 50인 이상 불특정 다수를 대상으로 증권시장 밖에서 모집/매출하는 경우 관련 사항을 공시

주체 : 해당 증권 발행회사

종류 : 모집 (50인 이상 신규증권), 매출 (50인 이상 기발행 증권)

- 50인 기준 : 청약 권유 ~ 6개월 전 동안 해당 증권에 대해서 권유받은 사람 포함 (모집 매출 x)

대상 : 10억 이상

면제 : 국채, 지방채, 국가보증 증권, 주택저당증권, 학자금대출증권, 만기 3개월 내 전자단기사채

- 총액 10억 미만인 경우는 간소화된 서류로 대체

규정 : 의무는 없어도 공지는 해야 함

효력 : 주식(10일), 증권(15일), 채권(7일), 무보증사채 (5일), 보증사채/담보부사채(3일)

- 효력발생기간동안 별도의 조치가 없으면 효력이 발생된 것으로 봄

특수 신고서

- 일괄신고서 : 수리일 3일 뒤부터 효력발생
- 정정신고서 : 수리일 1일 뒤부터 효력발생 / 증권수 80%~120% 변경은 효력발생일 동일
    - 3개월 이내에 정정신고서를 제출하지 않으면 증권신고를 철회한 것으로 간주
- 철회신고서 : 청약일 전일까지 제출 가능

**5.2 간주모집**

의의 : 형식적으로는 사모 모집이지만 공모로 간주하여 규제하는 제도

정의 : 50인 미만, 1년 내 50인 이상 전매가능한 경우

* 전매가능성이 없다고 보는 경우 : 1년의 보호예수를 걸어놓는 경우

**전매기준**

의의 : 러프한 기준으로 사모로 발행된 증권이 불특정 다수에게 풀리는 것, 우회적으로 공모의 효과를 내는 것 방지

규정 : 의무보유기간 설정, 권면분할 금지 (일정 기간 50개 미만으로 유지)

**5.3 증권분석기관**

자격 : 신용평가업자, 회계법인, 채권평가회사

제한 : 임원이 출자 (1%), 기관이 출자 (3%), 공동지분자 (5%)

**5.4 투자설명서**

의의 : 증권의 모집/매출을 위해 투자자에게 교부되는 설명서

의무 : 취득하려는 자에게 반드시 사전에 교부해야 함

면제 : 전문투자자, 거부의사 밝힌 자, 기취득자

종류 : 투자설명서, 예비투자설명서 (수리 후 효력 전), 간이투자설명서 (수리 후 효력 전 광고용)

### 6. 증권 유통시장 공시제도

**6.1 정기공시**

대상 : 주권상장법인, 증권발행인, 외부감사대상법인 (증권소유자 > 500)

면제 : 파산, 500인 → 300인

기한 : 사업보고서 90일 내, 반기/분기보고서 45일 내 → 금융위, 거래소 제출

**6.2 주요사항보고**

의의 : 회사의 재무구조나 경영활동에 중대한 영향을 미치는 사실이 발생했을 때 지체 없이 보고

대상 : 어음 및 수표의 부도 (부실채권 10%), 영업활동의 전부 혹은 중요한 정지, 회생절차개시 신청

- 양수하려는 영업부분의 자산액이 최근 자산총액 10% 초과

기한 : 발생 다음날까지

* 법적 제제 가능

제외

- 일상적인 영업활동
- 법에 의한 자기주식 취득, 검사대상기관과의 거래, 자산유동화
- 공개매수, 공개매수청약권, 출자증권의 양수/양도
- 외국법인은 연결재무제표 제출 시 개별재무제표 생략 가능, 해당 국가 기준 따르기 가능

**6.3 수시공시**

의의 : 기업에 중요한 변화가 발생하는 경우 지체 없이 보고 (사유발생일 다음날까지)

종류 : 자율공시, 조회공시, 공정공시

특징 : 의무공시는 아니지만 법적 효과는 동일함 (법적 제제 불가능)

### 7. 기업의 인수합병 관련 제도

**7.1 공개매수**

의의 : 장외에서 특정 기업의 주식을 대량으로 매수하고자 할 때, 매수 기간, 가격, 수량 등을 사전에 공고

요건 : 6개월간 증권시장 밖 + 10인 이상으로부터 + 5% 이상 매수

- 특수관계인 : 법인의 임원, 계열사, 계열사 임원, 소유주식 1000주 이상

면제 : 소각 목적

절차 : 신문 등으로 공고 → 신고서 제출 → (발행인 의견표명) → 공개매수 실시 → 보고서 제출

- 기간 : 20일~60일, 공개매수는 철회금지 (예외 - 대항공개매수, 사전에 말한 사유), 응모주주는 철회 자유

**7.2 주식 등의 대량보유상황 보고(5% 보고)**

대상 : 7.1과 동일

요건 : 새로 5% 이상 보유, 5% 이상 보유자가 1% 이상 변동

면제 : 기주식에 따라 신주를 배정받은 경우, 주식 수는 그대로인 경우

기한 : 경영권 관련 시 5일 이내, 아니면 다음 달의 10일/20일 (일반투자자/전문투자자)

냉각기간 : 보유목적이 발행인의 경영권에 영향을 주기 위한 경우 5일까지 주식 추가취득/의결권 행사 불가

**7.3 의결권 대리행사 권유**

의의 : 피권유자의 합리적인 의사결정을 지원, 경영진의 안정적인 의결권 확보

대상 : 의결권 행사를 대리하게 권유, 의결권 위임 철회를 요구

방법 : 위임장용지와 참고서류를 교부 / 의사표시 시 전자우편으로도 가능

### 8. 장외거래 및 주식 소유제한

**8.1 장외거래**

의의 : 한국거래소 시장 밖에서 이루어지는 주식 거래를 의미

규정

- 투자매매업자는 환매조건부매도 업무만 가능
- 투자중개업자는 기업어음 중개 시 2명 이상의 신용평가업자로부터 신용평가 받아야 함
- 일반투자자는 장외파생상품은 위험회피목적으로만 거래가능
- 전문투자자 대상 채무증권 투자중개업은 체신관서에 대한 채무 증권 매매를 담당
- 영업용순자본 150% 미달 시 장외파생상품 매매는 중지해야 함

**8.2 공공적 법인의 주식 소유제한**

요건 : 경영기반이 정착되고 발전가능성이 있음, 높은 수익이 예상됨

제한 : 상장시 10% 이상 소유자는 그 이상 소유 금지, 나머지는 3% 이상 소유 금지

**8.3 외국인 증권 소유제한**

정의 : 외국인 - 국내에 6개월 이상 주소를 두지 않은 개인, 외국법인 - 외국정부, 지자체, 기업

제한 : 종목별 1인 한도제한, 외국인 전체 40% 제한

**8.4 채권의 장외거래매**

채권중개전문회사 : 당사자가 원하는 가격으로 체결시킬 것

환매조건부매매 : 금융위가 고시한 매매가격으로 매매, 환매수/매도 날짜를 정하여 매매

### 9. 불공정거래행위에 대한 규제

- 미공개중요정보 이용행위, 시세조종행위, 부정거래행위

**9.1 미공개정보 이용 규제**

대상 : 상장법인

대상 : 내부자 (임직원, 대리인, 주요주주), 준내부자 (감독자, 교섭자), 1차정보수령자

**공개매수 관련 정보 이용행위 금지**

**대량취득 및 처분 관련정보 이용행위 금지**

**내부자 단기매매차익 반환제도**

대상 : 내부자 (임원, 직원, 주요주주)

규정 : 매매 후 6개월 이내에 매매하여 발생한 이익에 대해 회사에 반환

내부자거래규제 적용대상 : 상장법인 or 6개월 내 상장예정

**9.2 시세조종행위 규제**

**시세조종행위 금지**

- 위장거래, 허위표시, 가격고정 또는 안정조작행위, 현선연계 시세조종행위

처벌 : 1년 이상 유기징역, 부당이익 3~5배 과징금 (부당이익 > 5억은 가중처벌)

**시장질서교란행위 금지**

종류 : 정보이용 교란행위, 시세관여 교란행위

처벌 : 5억 이하 과징금 or 부당이익의 1.5배

### 10. 금융기관 검사 및 제재에 관한 규정

의의 : 금감원장은 금융기관에 대한 검사를 실시

절차 : 심의회 설치 → 사전통지 → 의견진술 → 불복절차 → 이의신청 → 이사회 보고

- 심의회 설치 : 감독원장 판단으로 생략 가능
- 사전통지 : 사유에 따라 생략 가능
- 의견진술 : 상당한 기간을 정함
- 불복절차 : 이의신청, 행정심판, 행정소송
- 이의신청 : 이의신청 가능
- 이사회 보고

기타 : 현장조사는 1주일 전 통보

### 11. 자본시장 조사업무규정

**목적**: 불공정행위에 대한 조사

**대상** : 위법행위 혐의 발견됨, 조사를 요청받음, 관련 제보/민원을 받음

**면제** : 조사의 실익이 없는 경우, 풍문만 있는 경우, 검찰이 수사를 개시한 경우

**조치** : 형사제재, 시정명령/처분명령, 과태료/과징금, 증권발행 제한 등

- 제재심의위원회에서 결정, 금감원장이 생략 가능

**이의** : 1회에 한해 가능, 금감원장이 기각도 가능

### 12. 금융소비자보호법

**12.1 개요**

**상품 구분 (4)**

- **투자성 상품**: 금융투자상품, 투자일임계약, 신탁계약 등
- **예금성 상품**: 예금, 적금 등
- **보장성 상품**: 보험상품
- **대출성 상품**: 대출, 신용카드, 리스, 할부금융 등

**금융상품판매업자 구분**

- **직접판매업자**: 자신의 명의로 계약을 체결하는 자 (은행, 증권사, 보험사 등)
- **판매대리·중개업자**: 판매를 대리하거나 중개하는 자 (투자권유대행인, 카드모집인 등)
- **자문업자**: 투자자문업자 등

    → 인허가나 등록을 하지 않아도 금융상품판매업을 영위하면 금융상품판매업자로 분류


**12.2 6대 판매원칙**

**① 적합성의 원칙** : 대출성, 보호성 상품에도 적합성 원칙을 적용

**② 적정성의 원칙** : 일부 대출성, 보호성 상품에도 적합성 원칙을 적용

**③ 설명의무** : 각 법에 규정된 설명의무 통합

**④ 불공정영업행위 금지** :우월적 지위를 이용한 부당한 행위 금지.

**⑤ 부당권유행위 금지** : 오인할 우려가 있는 사실 등을 알리는 행위 금지

**⑥ 광고 규제** : 광고 시 필수 및 금지사항 규정

- 1,2 과태료, 3,4,5,6 과태료 과징금

**12.3 소비자 권리 강화**

**청약철회권**

- **내용**: 일반금융소비자는 일정 기간 내에 계약을 철회할 수 있음.
- **기간**: 투자성/대출성 상품은 7일, 보장성 상품은 15일.
- **예외**: 비금전신탁계약, 고난도 투자일임계약 등 일부 상품은 제외됨.

**위법계약해지권**

- **내용**: 판매원칙을 위반한 계약에 대해 소비자가 계약을 중도에 해지할 수 있는 권리
- **대상** : 5대 판매규제 위반 시 ↔ 광고위반은 제외
- **기간**: 위법 사실을 안 날로부터 1년, 계약일로부터 5년 이내 행사.
- **대상**: 투자성, 대출성, 보장성 상품에 해당
- **효력** : 원상회복의무는 없음

**사후구제제도 :** 금융분쟁조정제도, 소송중지제도, 조정이탕금지제도, 자료열람요구권 (8일), 손해배상책임

**12.4 분쟁조정 및 제재**

- **분쟁조정**
    - **효력**: 분쟁조정위원회의 조정안을 양 당사자가 수락하면 '재판상 화해'와 동일한 효력 발생.
    - **소송중지**: 조정이 진행 중일 때 소송이 제기되면 법원이 소송을 중지시킬 수 있음.
- **판매제한 명령**
    - **내용**: 소비자의 재산에 현저한 피해 발생 우려가 명백한 경우 금융위원회가 해당 상품의 판매를 제한하거나 금지할 수 있음.
- **징벌적 과징금 및 과태료**
    - **과징금**: 설명의무, 불공정영업행위, 부당권유행위 등 위반 시 관련 수입의 50%까지 부과.
    - **과태료**:
        - 6대 판매원칙(설명의무 제외) 위반 시 최대 1억원.
        - 청약철회권·위법계약해지권 관련 의무 위반 시 최대 3천만원.
        - 자료 미제출 또는 허위 제출 시 최대 1천만원.

## 02 회사법 (5)

> 회사법 (상법 제 3편)
>

* 합자회사 : 1명 이상의 무한책임 사원

### 1. 주식회사의 개념

**자본**

원칙

- 자본확정의 원칙
- 자본충실의 원칙 : 이익배당의 제한, 변태설립 감독, 법정준비금제도, 액면미달 발행 제한, 납입담보책임
- 자본불변의 원칙

### 2. 주식회사의 설립

절차 : 발기인 구성 → 정관 작성 → 주식발행사항 결정 → 주식 인수 및 납입 → 임원 선임 → 설립 등기

설립 종류

- 발기설립 : 설립 시 발행하는 주식 전부를 발기인이 인수
- 모집설립 : 발행 주식의 일부만 발기인이 인수하고 나머지는 주주를 모집하여 인수

정관 내용

- 절대적 : 목적, 상호, 발행 총수, 1주 금액, 주식총수, 본점 소재지, 공고법, 발기인 정보
- 상대적 : 변태설립사항 (발기인 혜택, 현물출자, 재산인수, 설립비용, 발기인 보수액)

    * 공증인의 인증이 있어야 효력이 발생함


발기인 : 정관에 발기인으로 서명한 자, 1주 이상 소유할 것

감독 : 이사, 감사가 담당

설립 등기

- 구성 종료 후 2주 이내 하여야 함
- 이후 주식인수 무효 주장 불가, 주식인수 취소 불가
- 2년 이내 설립무효소송만 가능 (대세효, 비소급효)
- 소송의 결과는 당사자 이외의 제3자에게도 미침

### 2. 주식과 주주

**2.1 주식**

의의 : 주식회사에서의 사원의 지위(주주권)를 의미하며, 자본금의 구성단위

종류

- **액면/무액면 주식** : 주권에 금액이 기재되었는지 여부에 따른 구분 → 액면주식은 100원 이상 균일해야
- **기명/무기명 주식** : 주주명부에 주주의 성명이 기재되는지에 따른 구분 → 무기명 폐지됨
- **종류주식** : 특수한 내용을 부여받은 주식 (이익배당, 의결권 행사, 상환 및 전환)
    - 복수의결권 및 거부권주식은 여전히 불가능

**2.2 주주**

의의 : 회사의 소유자로서 주식의 인수를 통해 회사에 대한 권리와 의무를 가짐

권리

- 자익권 : 이익배당청구권, 잔여재산분배청구권, 신주인수권
- 공익권
    - 단독주주권 : 의결권, 설립무효판결청구권, 신주발행유지청구권, 열람청구권
    - 소수주주권 : 10%(해산판결청구권) 3%(주주총회소집/재판상태검사/회계장부열람 청구권),  1%

의무 : 인수가액에 따른 납입의무만 짐

* **주주명부**

기재 시 주주로 추정됨, 성명과 주소를 기입

회사채권자도 언제가 주주명부의 열람 또는 등사 청구 가능

주주명부의 폐쇄

- 폐쇄기간은 3개월 이내의 기간으로 정함
- 폐쇄기간 2주 전에 공고하여야 함 (정관으로 규정 시 생략가능)
- 폐쇄기간동안 의결권 행사 불가능, 전환권 행가는 가능

**2.3 주식의 양도**

- 법률과 정관에 의해서만 주식양도를 제한할 수 있음

법률상 제한 : 권리주의 양도제한, 주권발행 전 주식양도 제한, 자기주식의 취득, 상호주식의 취득금지

- 권리주의 양도는 회사에 대해서는 효력 없음
- 회사 성립 6개월부터는 주권발행 전이어도 양도가 인정됨
- 자기수식의 취득은 주총 특별결의만으로 가능

정관상 제한 : 회사가 필요 시 이사회의 승인을 얻어 결과를 주주에게 서면 통지

**2.4 주식의 소각 병합 분할**

소각 : 이사회 결의 필요

병합 : 1개월 이상의 기간동안 주권을 제출하게 함

분할 : 주주총회 결의 필요, 액면가 > 100만 가능

**2.5 주식의 포괄적 교환과 이전**

의의 : 회사간에 주식을 전부 주고받는 경우 완전모회사/자회사를 성립하게 함

교환 : 완전모회사가 신주를 발행하여 완전자회사의 주식 전부와 교환

- 주식교환계획서 작성, 주주총회 특별결의 승인 필요

*** 지배주주의 매도청구**

의의 : 지분율 95% 이상의 지배주주가 필요시 나머지 주주에게 강제로 매도할 것을 청구하는 권리

목적 : 완전자회사화, 비상장화, 경영권 강화

절차 : 지분 보유 → 주총 승인 → 매매가액 결정 ( → 법원의 가격 결정 → ) → 2개월 내 매도

### 3. 회사의 기관

**종류**: 의사결정기관인 **주주총회**, 업무집행기관인 **이사회**와 **대표이사**, 감독기관인 **감사** 또는 **감사위원회**로 구성

**3.0 주주총회**

2주 전까지 통지

보통결의사항 : 출석 의결 1/2 & 총 발행주 1/4

- 지배주주 매도청구, 이사/감사/청산인 선임과 보수결정
- 재무제표 승인, 배당

특별결의사항 : 출석 의결 2/3 & 총 발행주 1/3

- 정관 변경, 회사 조직 변경 (다른 회사 영업 전부 또는 일부 양수), 이사/감사 해임
- 자본금 변동 (자본금 감소,  주식소각) 전환사채/신주 발행, 주식의 포괄적 교환이전
- 영업의 양도, 임대, 경영 위임

특수결의사항 : 총주주 동의 필요

- 주식회사 → 유한회사 조직변경, 이사의 회사에 대한 책임면제

**의결권**

1주 1의결권을 가짐

행사 : 대리행사 가능, 불통일 행사 가능 (회사는 거부 가능, 일부 제외)

제한 : 결의안에 특별한 이해관계가 있는 경우, 감사선임 시 3% 초과 보유, 타 회사 10%

투표 : 출석투표, 불출석투표 가능

**3.1 이사**

선임 : 집중투표로 선임 가능

특징 : 3년을 초과하지 못함, 3인 이상이어야 함 (자본금 < 10억은 1~2인 가능)

자격 : 자연인이어야 하며 법인은 불가능

의무 : 경업금지의무, 비밀유지의무

책임 : 손해배상책임, 자본충실책임, 결산에 대한 책임

* 사외이사 : 상장법인은 1/4, 2조원 이상 상장법인은 1/2, 회사의 상무에 종사하지 않음

**3.2 이사회**

소집 : 이사가 함

결의 : 과반수 출석, 출석의 과반으로 결의, 서면 불가능

권한 : 의사결정권 (대표이사 선임, 공동대표 결정, 신주발행, 사채발행)

의사록 : 주주에 한해서 열람 가능

**3.3 대표이사**

선임 : 이사회 선임

종임 : 이사 자격 상실 시

권한 : 회사에 영업에 관한 모든 행위 가능

* 집행임원

의의 : 대표이사를 대신하는 업무집행기관, 이사회의 업무집행과 감독기능을 분리

선임 : 이사회 선임/해임

임기 : 2년을 초과하지  못함 (정관으로 변경가능)

**3.4 감사**

선임 : 주주총회 선임 (주주는 최대 3%까지만 의결권 행사 가능)

자격 : 제한없음

임기 : 3년 내 최종 결산기 주주총회 종결시까지

권한 : 영업보고청구권, 업무상태조사권, 자회삭마사권, 이사회참석권, 이사 유지청구권, 주총/이사회 소집권

의무 : 이사에게서 받은 뒤 4주 이내 감사보고서 제출

**3.5 감사위원회**

구성 : 3인 이상의 이사, 사외이사 2/3 이상

선임/해임 : 이사회 결의

강제 : 자산총액 2조원 이상 상장회사는 상설로 두어야 함

### 4. 신주의 발행

**결정** : 이사회에서 결정

**절차** : 발행사항 결정 → 신주인수권자에 대한 통지/공고 → 주식 청약 → 납입 → 변경 등기

**신주인수권** : 주주는 보유 주식 수에 비례하여 신주를 배정받을 권리를 가짐, 증서 교부로 양도 가능

* 현물출자 가능

**4.1 액면미달발행 (할인발행)**

- 원칙적으로는 금지됨

허용

- 회사 설립 2년 후, 주주총회 특별결의, 법원 인가, 인가 1개월 내
- 상장회사는 법원 인가 생략 가능

### 5. 자본금의 감소

**종류**

실질적 감소 : 회사 재산의 환급(유상소각) 또는 주식의 무상소각.

명목상 감소 : 회사 재산의 변동 없이 자본금 액수만 감소(주식병합)

**절차** : 주주총회 특별결의 → 채권자 보호 절차(공고 및 최고) → 자본감소 실행 → 변경 등기

- 절차 완료 시 효력발생
- 결손 보전 목적의 자본금 감소는 주주총회 보통결의로 가능
- 6개월 내에 무효의 소 제기 가능, 소급효

### 6. 회사의 회계

**6.1 재무제표**

종류 : 재무상태표, 손익계산서, 현금흐름표, 자본변동표, 주석

작성 : 매 결산기마다 이사가 작성, 이사회 승인

제출 : 정기총회 6주 전 감사에게 제출, 4주 내 감사보고서 반환

**6.2 준비금**

법정준비금 : 이익준비금 (이익배당액 10% 이상을 자본금 1/2까지 적립), 자본준비금

- 용도 : 결손전보, 무상증자 / 자본금의 150% 초과 시 초과범위 내에서 감액 가능
- 주식배당 시 이익준비금 적립하지 않아도 됨

임의준비금 : 사업확장, 주식소각, 사채상환 등 용도

**6.3 이익배당**

종류 : 결산배당, 중간배당, 중기배당

- 정관으로 금전 외 배당 결정 가능
- 자본충실의 원칙 위반 시 위법배당으로 무효
- 배당은 결의일부터 1개월 내

**6.4 주식배당**

의의 : 이익배당을 신규발행 주식으로 하는 것

한도 : 이익배당 총액의 1/2을 초과하지 않는 범위 내에서 주식으로 배당

절차 : 주주총회 보통결의

6.5 중간배당

현물배당 가능

### 7. 사채

**7.1 사채**

의의 : 주식회사가 일반 대중으로부터 비교적 장기의 자금을 집단적, 반복적으로 조달하기 위해 발행하는 채무증서.

발행 : 이사회의 결의로 발행 → 모집에 응함 → 모집 후 납입

유통 : 납입 완료 후부터 가능

이자 : 이자지급권 소멸시효 5년, 상환청구권 시효 10년

집회 : 총액 10% 이상 보유자 소집청구 가능

관리 : 사채관리회사에 위탁 가능

기타 : 분할납입 가능

**7.2 특수사채**

**전환사채(CB)**: 발행 후 일정 기간 내에 소유자의 청구에 의해 주식으로 전환할 수 있는 권리가 부여된 사채

- 전환청구권 제출로 효력이 생김

**신주인수권부사채(BW)**: 사채권자에게 신주를 인수할 수 있는 권리가 부여된 사채

- 신주발행가액 전액 납부 시 효력이 생김

**교환사채(EB)**: 회사가 보유한 다른 회사의 유가증권과 교환을 청구할 수 있는 권리가 부여된 사채

이자지급권, 상환청구권 → 5년/10년 시효

**7.3 사채권자협회**

정의 : 회사의 사채를 가진 사람들의 모임

소집 : 총 사채의 10% 이상을 가진 사람들이 소집청구 가능

### 8. 회사의 합병, 분할

**8.1 회사의 합병**

의의 : 청산절차를 거치지 않고 회사가 합동함

절차 : 합병결의, 주주총회 특별결의, 합병등기 / 6개윌 후까지 대차대조표를 비치

규정 : 효력은 등기시점부터, 합병교부금 지급 가능, 권리의무는 포괄적으로 승계, 2주 전~6월 후 대차대조표 비치

무효 : 6개월 이내 소 제기만으로 가능

* 간이합병 :

**8.2 회사의 분할**

유형 : 인적분할 (주식을 주주에게 교부), 물적분할 (주식을 회사가 가져감)

절차 : 분할계획서 작성, 분할공시, 주주총회 특별결의, 분할등기

- 소규모 분할은 이사회 결의로도 가능

규정 : 효력은 등기시점부터

무효 : 6개월 이내 소 제기만으로 가능

### * 주식회사 특례

**자본금 10억 미만인 회사에 대한 특례**

- 감사 선임이 임의 (주주총회가 감사의 역할)
- 이사가 1인 혹은 2인 가능 (↔ 3인)

- 주주 전원 동의로 주주총회 소집 생략 가능

## 03 직무윤리 (5)

### 1. 직무윤리 일반

윤리 : 개인적이고 내면적인 규범, ‘있어야 할 법’

법 : 정당한 것을 지향하는 규범, ‘있는 그대로의 법’

정의 : 윤리가 사회적인 범위로 확장된 것, 법과 윤리를 포괄하는 개념

기업윤리 : 모든 구성원에게 요구, 추상적, 윤리강령

직무윤리 : 개개인이 지킬 행동, 구체적, 행동갈령

**강조의 이유 :** 윤리경쟁력의 시대 (시대의 변화, 거래비용, 생산성, 신종자본, 인프라, 사회적 비용감소)

**금융 투자업** : 산업의 고유속성(이해상충), 상품의 특성(원본손실), 소비자의 변화, 안전장치

**배경**

칼뱅 : 금욕적 생활주의, 자본주의의 정신적 원동력

베버 : 합리성, 체계성, 조직성, 합법성

국제 : OECD, 국가별 부패인식지수 (CPI), 김영란법, 윤리경영 평가척도, KoBEX

**적용대상**

- 투자권유자문인력 등 관련 전문자격증 보유자
- 자격증은 없으나 관련 업무에 실질적으로 종사하는 자
- 업무와 관련되어 있는 자
- 잠재적 고객을 대상으로 투자관련업무를 수행하는 자

### 2. 금융투자업 직무윤리

4**가지 의무**

- 기본원칙 (금융투자회사 표준윤리원칙)
- 이해상충방지 의무 (관련 법률)
- 금융소비자 보호의무 (관련 법률)
- 본인, 회사, 사회에 대한 윤리 (금융투자회사 표준윤리원칙)

**2.1 기본 원칙**

- **신의성실의 원칙**: 정직과 신뢰를 우선해야 함 → 위반 시 불법, 모든 단계에 적용됨
- **고객 우선의 원칙**: 자신의 이익보다 고객의 이익을 우선해야 한다.

**2.2 이해상충 방지 의무 (법률)**

개념 : 자본시장법 37조 2항 (투자자의 이익을 해하며 이익을 얻으면 안됨)

원인 : 사적업무에서 취한 정보를 공적업무에 사용 시 이해상충 발생

예시 : 과당매매

방지 : 이해상충관리의무, 공시 또는 회피, 정보교류차단, 조사분석자료 이해상충방지의무, 자기거래금지의무

- 정보교류차단벽 : 책임자, 담당 조직 설치 / 상시 정보교류 차단벽 설치
- 자기거래금지 예외 : 장내거래, 다자간매매채결회사 통한 거래

기타 : 인가등록시부터 구축 의무

**2.3 금융소비자보호 의무 (법률)**

**설명의무**: 금융상품의 내용, 투자에 따르는 위험 등 중요 사항을 소비자가 이해할 수 있도록 설명해야 한다.

개념 : 정보의 비대칭으로 금융소비자가 불이익을 얻지 않게 하는 것

의무 : ‘그때마다의 구체적인 상황에서 전문가로서의 주의를 기울여야’ 함

기관 : 이사회, 대표이사, 금융소비자보호총괄책임자 (CCO) - 위험관리 업무는 담당이 아님

* CCO : 민원접수 및 처리 관리감동, 민원발생부서 평가 기준 수립, 금융상품 단계별 체계 관리감독

단계별

- 상품개발단계 : 사전협의절차, 금융상품개발 관련점검, 외부의견 청취
- 상품판매이전 : 교육체계 마련, 판매자격 관리 (펀트, 증권, 파생)
- 상품판매단계 :  적합성 원칙 (KYC rule), 적정성 원칙, 설명의무 (일반금융소비자만) ***6대원칙**

    (손실보전 홍보 금지, 요청하지 않은 투자권유 금지 - 증권/장내파생은 허용)

    * 적합성의 원칙 : 일반금융소비자에게 권유할 때

    * 적정성의 원칙 : 일반금융소비자가 원하는 경우

- 상품판매이후 : 보고 및 기록의무 (자료열람요구권 6일), 누설 및 부당이용 금지, 모니터링제도

    ex) 해피콜 서비스 (7영업일), 고객의 소리, 위법계약해지권 (귀책이 있는 경우, 10일), 미스터리 쇼핑


* 철회가능기간 : 투자성(7일), 대출성(14일)

* 청약철회권 미적용 : 파생결합증권

* 불공정 영업행위 시 수입 50%까지 과징금, 1억까지 과태료 가능

*** 6대 판매원칙**

- 적합성 원칙 : 투자권유 희망 여부 확인, 전문투자자 여부 확인, 적절한 권유
- 적정성 원칙 : 고객이 원하는 금융상품이 고객에게 적정한지 확인
- 설명의무 : 설명서 제공, 기록해야 함, 청약철회권(고난도금융투자산품/투자일임계약/금전신탁계약) 설명
- 불공정영업행위 금지 : 불리한 행위 요구 금지
- 부당권유행위 금지 : 합리적 근거 제공, 적정한 표시, 요청하지 않은 투자권유 금지 (1개월), 기타 부당권유
- 허위·과장광고 금지 : 등록된 금융상품판매업자만 광고 가능

***** KYC rule (know your customer rule)

투자권유 여부 확인 → 일반/전문금융소비자 확인 → 고객 정보 파악 → 서명 → 투자 성향 파악 → 예금성 제외

**2.4 본인, 회사 및 사회에 대한 윤리 (금투사 표준윤리준칙)**

**본인에 대한 윤리**

- 법규준수 : 자본시장법 및 인접 법령 포함
- 자기혁신의무 : 임직원은 최소한의 지식을 갖추고 준칙을 준수
- 품위유지의무
- 공정성 및 독립성유지의무
- 사적이익추구금지의무 : 수령한 이익이 10억 이상시 공시, 관련 기록 5년간 보관, 동일거래는 이사회 사전승인
- 수수료 및 성과보수 제한

**회사에 대한 윤리**

- 상호존중의무
- 회사재산의 사적사용금지
- 경영진의 책임 : 법적, 민사적 책임
- 비밀정보 관리
- 위반행위의 보고 : 내부제보
- 고용종료 후의 의무 : 합리적인 기간 선관주의의무는 지속됨
- 대외활동 준수사항 준수 : 금전적 보상 수령 시 회사에 신고, 사견임을 명시

**사회에 대한 윤리**

- 시장질서 존중 : 교란행위 금지
- 주주가치극대화
- 사회적 책임

### 3. 직무윤리의 준수절차 및 위반 시 제재

**3.1 직무윤리위반 시 제재**

- 내부통제 : 내부통제기준(업무의 분장과 조직구조 관련사항) 설립, 준법감시인 활용

**내부통제위원회**

구성원 :  위원장(대표이사), 준법감시인

회의 : 반기에 1회 이상

면제  - 자산총액 기준 : 상호저축은행 7천억, 금융투자회사 - 보험회사 - 여신전문 5조

❗단 금융투자회사가 총합 20조 이상인 경우는 제외

**준법감시인**

임명 : 이사회 의결, 해임 : 이사회 2/3이상 의결

임면사실을 금융위원회에 보고

업무

- 이사회의 지휘를 받아 내부통제업무 수행,
- 재무경영과 무관한 보수 지급, 명확한 업무 위임 가능 (영업관리자), 임기 2년
- 지점장은 내부통제업무의 적정성을 정기적으로 점검하여 보고

**영업점 내부통제**

- 금융소비자용 전용공간은 직원과 분리되어야 한다
- 사이버룸은 반드시 명기하고 외부에서 보이도록 설치해야 함

* 영업점별 영업관리자

- 준법감시인이 지정, 연 1회 이상 교육 실시, 임기 1년
- 조건 : 1년 이상 근무, 상근근무자, 영업점장이 아닌 책임자급 (지점장 불가능)
- 1명이 2개 이상의 영업점을 묶어 업무 수행 가능

**외부통제** : 자율규제, 행정제제, 민형사책임, 시장통제

- 1억 이하 과태로 : 내부통제기준 x, 준법감시인x, 이사회 없이 준법감시인
- 3천 이하 과태료 : 준법감시인 보수지급 기준 x, 준법감시인이 겸직하게 함
- 2천 이하 과태료 : 준법감시인 임면 미보고

**준수 절차**

- **내부통제기준**: 회사는 직무윤리 준수를 위한 내부통제기준을 마련하고 준법감시인을 두어 운영해야 한다.
- **윤리강령**: 금융투자협회 등 자율규제기관은 표준윤리준칙을 제정하여 회원사들이 따르도록 권고한다.

**직무윤리 위반행위 제제**

- **자율 규제** : 회원 제명
- **행정 제재** : 금융위원회 - 금융투자업자 제재권, 임직원 조치권
- **민사 책임** : 손해배상 책임
- **형사 책임** : 양벌규정 등
