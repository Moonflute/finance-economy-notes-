**이산확률분포**

- 베르누이분포 : 취할 수 있는 값이 2개
- 이항분포 : n번의 베르누이 시행 중 k번 성공할 확률의 분포
- 기하분포 : 베르누이 시행에서 첫 성공까지 k번 실패할 확률의 분포
- 다항분포 : 3개 이상의 결과가 가능
- 포아송분포 : 단위 시간 / 공간 내에서 발생할 수 있는 사건의 횟수의 확률의 분포

**연속확률분포**

- Z 분포 : 표준정규분포, N(0,1)
- t 분포 : 스튜던트분포, 표본표준편차를 사용 T(t1)

- 지수 분포 : 어떤 사건이 발생하기까지 걸리는 시간
- 대수정규분포 : X = exp(Y), 단, Y ~ 정규분포
    - 로그를 취하면 정규분포를 따름, 왜도 > 0

**확률 과정**

### 4. 회귀분석과 예측

**4.1 회귀분석**

정의 : 변수간의 함수적 관계식을 통계적 방법으로 추정

가정

- 정규성 : 오차는 정규분포를 이룸
- 독립성 : 오차의 공분산은 0
- 등분산성 : 오차의 분산은 동일
- 선형성 : X-Y는 선형관계를 가짐

종류 : 단순회귀분석, 다중회귀분석 (공선성 확인, VIF)

- 단순회귀분석에서 beta = Cov(X,Y) / Var (X)

변수 선택 : 벌점화 방식 (AIC, BIC, 멜로우 Cp), 단계적 변수 선택법 (전진, 후진, 단계별)

유의성 검정 : R(결정계수), t검정 (회귀계수), F검정 (회귀직선), 잔차 분석 (정규성, 등분산성, 독립성-DW)

- R^2 = (SSR)/SST

SSE : 오차제곱합
SSR : 회귀제곱합
SST : 전체제곱합

**예측** : 회귀모형 사용, 시계열 자료로 미래 예측

### 3. IS-LM, AD-AS

**3.1 IS-LM 곡선**

의의 : 물가변동을 전제하지 않은 모형 (선진국)  **/** 재화시장 - 화폐시장 균형 / r-Y 평면

![image.png](I%20%EC%A6%9D%EA%B6%8C%EB%B6%84%EC%84%9D%20%EA%B8%B0%EC%B4%88/image%201.png)

국민소득 - 이자율 평면 (R : 이자율, Y : 실질국민소득, M : 통화량, P : 물가)

**IS 곡선 : 재화시장의 균형** (Y = C + I + G) - investment and saving

→ 이자율이 낮을수록 투자가 증가해 총수요가 증가함 (투자가 이득이기 때문)

→ 이자율이 낮으면 기대 이익의 현재가치가 높아져 이익이 높아지기 때문

→ I = I(r), Y = -ar + b 꼴

→ 이자율이 낮아질수록 투자가 증가해 국민소득은 증가

- 선 위가 아닐 경우 기업이 생산을 조절(국민소득 생산면)해서 지출수준에 맞춤

**LM 곡선 : 화폐시장의 균형** (M/P = L(Y,R) ) - liquidity and monetary

→ 소득이 증가하면 화폐수요가 증가, 이자율도 증가

→ 실질 화폐수요 = 거래적 수요 + 투기적 수요 = k(Y) + l(r) = 화폐공급량 / 물가

> IS : 이자율이 높은 상태에서는 주로 저축하여 소비가 줄어드므로 Y가 낮음
>

> LM : 이자율이 높은 상태에서는 그만큼 소득도 높다는 의미로 Y가 높음 (저축수요 + 공급 = 소득수요)
>

- 구축효과 : 확대재정정책 → 금리상승 → 투자위축 → 증가효과 후퇴
    - 고전학파 : 완전구축효과 주장
    - 케인즈학파 : 무구축효과 주장

통화정책

- 유동성함정 (케인즈학파)
    - 금리를 낮추어도 투자와 소비가 늘어나지 않음
    - 이자율이 이미 매우 낮으면 통화정책 (금리인하) 가 효과가 없음
    - 이자율이 낮아서 사람들이 돈을 그냥 가지고 있기 때문
    - LM 곡선은 수평이 됨 (Y가 변해도 r이 변하지 않음)
    - 화폐수요가 이자율에 대해 무한히 탄력적
    - 재정정책만 효과적
- 피구효과 (고전학파)
    - 물가의 하락이 실질구매력을 증가시켜 유동성함정을 탈출
- 리카르도의 불변정리 :  이자율을 낮춰도 민간에서는 결국 추후에 더 낼것이라 생각하여 지출을 늘리지 않음

**3.2 AD-AS 곡선**

의의 : P(물가) - Y(소득) 평면

![image.png](I%20%EC%A6%9D%EA%B6%8C%EB%B6%84%EC%84%9D%20%EA%B8%B0%EC%B4%88/image%202.png)

**AD 곡선 :** aggregate demand curve

- **IS-LM이 균형이 되는 Y와 P의 조합**
- 물가가 하락하면 실질 화폐공급(M/P)는 증가

    → LM 곡선 우측으로 이동 → 이자율 하락, 국민소득 증가

    → 즉 물가 - 국민소득은 (-)의 관계를 가짐


**AS 곡선 : 총공급곡선** - aggregate supply curve

- 한 국가의 생산능력을 보여줌

    → 고전학파는 수직이라고 주장 (임금이 유동적으로 변함을 가정)

    → 케인즈학파는 우상향이라고 주장 (명목임금이 고정되었기 때문) (* 화폐환상)


<설명>

1. 한계 생산성은 체감함
2. 기업은 실질임금이 한계생산성과 일치하게 노동을 필요로 함
3. 노동의 공급은 실질 기대임금으로 나타냄

고전학파는 가격이 완전 신축적이라고 주장함 → 임금 역시 완전 신축 → Y는 고정된 값을 가짐

케인즈학파는 명목임금은 경직적이라고 가정함 → Y는 P(물가)와 양의 상관관계를 가지게 됨

재정정책 : 투자를 조절 (G)

→ 확대재정정책은 초과수요를 발생시킴 → AD곡선이 우측으로 이동

통화정책 : 통화를 조절 (M)

→ 확대통화정책은 초과수요를 발생시킴 → AD곡선이 우측으로 이동

### 4. 화폐의 수요, 공급

**4.1 화폐의 수요**

- 중첩세대 모형 : 재화는 저장 불가능
- 거래잔액설 : 경제성장을 능가하는 화폐공급은 인플레이션 유발
- 유동성 선호설 : 실질화폐수요 = 거래적 수요 + 투기적 수요

### 5. 이자율의 결정과 변화

**이자율의 결정이론**

- 고전학파 : 이자율은 실물적 현상, 대부시장에서 결정됨
- 케인즈학파 : 이자율은 화폐적 현상, 수요-공급 균형점으로 결정됨
- 현대적 대부자금설 : 저축 + 화폐공급 변동 = 투자 + 정부예산적자 + 수요변동
- 피셔방정식 :  명목이자율 = 실질이자율 + 기대인플레이션율
    - 오쿤 : 기대인플레이션이 명목이자율에 전부 반영되지는 않음
    - 섬머스 : 자금 조달의 제약으로 기대인플레이션보다 명목이자율이 약간 낮음
    - 먼델-토빈 : 실질잔액효과로 완전한 피셔효과가 나타나지는 않음

**이자율의 기간구조**

- 불편기대이론 : 채권수익률은 미래이자율에 대한 예상을 그대로 반영
- 시장분할이론 : 채권 만기별 선호 대상이 다름
- 특정시장선호이론 : 기대이론 + 시장분할이론
- 유동성선호이론 : 장기채일수록 유동성 포기하는 대가로 추가 프리미엄을 요구함

**통화량 증가에 의한 이자율 변동**

통화량증가 → 유동성효과 (하락)→ 소득효과 (증가) → 피셔효과 (증가)

### 7. 환율의 결정과 변화

**환율변화**

- J curve 이론 : 평가절하시 일시적으로 수지악화 → 이후 수출증대
- S curve 이론 : 평가절하로 무역수지가 개선되어도 외환유입이 증가하여 인플레이션 발생

평가절상 : 통화의 가치가 상승

평가절하 : 통화의 가치가 하락

**환율결정**

- 구매력평가설 : 국가간의 물가지수의 비로 결정됨 / 절대적, 상대적
- 오버슈팅 : 정책의 효과보다 기대가 먼저 반영되어 단기간에 급격하게 변동 → 이후 회복
- 랜덤워크 : 사람들은 합리적 기대로 환율을 결정
- 통화론적 : 화폐수요가 감소하면 환율이 상승함

- 고정환율제 : 통화정책 효과 무력. 재정정책 효과 극대
- 변동환율제 : 통화정책 효과 증폭, 재정정책 효과 상쇄

**마샬-러너 조건**

- 한 국가의 통화 가치가 절하될 때, 그 나라의 무역수지가 개선되기 위한 조건
- (수출 수요의 가격탄력성) + (수입 수요의 가격탄력성) > 1

→ 통화 가치가 절하되는 경우 무역수지 (수출액 - 수입액) 이 개선됨

기업의 종류 : 개인기업, 합자회사, 주식회사 → 대리 문제 (성과급제, 제도적 장치, 매수합병)

**1.2 재무담당자**

역할 : 실물투자와 자금조달 의사결정

종류

- Treasurer : 자금제공자와 관계관리, 현금흐름 관리
- Controller : 재무제표 작성, 회계관리
- Chief Financial Officer (CFO) : 업무감독, 전체 재무 의사결정

### 2. 현금흐름의 분석

**자본자산 :** 양의 효용 (현금흐름)을 창출하는 경제적 요소

**현금흐름** : 모든 기간 동안의 현금흐름을 분석 (↔ 회계학)

- 감가상각, 지급이자, 배당금, 매몰원가 제외

**순현가** : 순현금흐름의 현재가치의 합

**할인율** : 미래가치를 현재가치로 비교할 때 필요한 정보 (k = 무위험 이자율 + 위험 보상률)

### 3. 자본예산

**자본예산** : 1년 이상의 현금흐름을 창출하는 자산, 현금 유입과 유출에 대한 계획을 세우는 것

**투자안 :** 상호배타적, 상호독립적, 상호의존적

**현금흐름 분석** : 세후기준, 부수 현금흐름 고려, 감가상각 등 제외, 매몰원가 제외, 이자비용 제외

**3.1 분석방법**

- 회수기간법 : 투자안의 회수기간이 기준치보다 짧은지 계산
- 회계적 이익률법 : 투자안의 이익률이 기준치보다 큰지 계산
- 순현가법 (NPV) : 순현가가 0보다 큰지 계산
- 내부수익률법 (IRR) : 투자산의 내부수익률이 적정 k 보다 큰지 계산

    → NPV가 IRR보다 더 중요한 기준

- 수익성 지수 (PI) : 미래 현금유입의 현재가치 / 초기 투자액, PI > 1일 때 투자안 채택

### 4. 재무구조이론

**4.1 재무구조이론**

의의 : 기업의 부채비율은 기업가치나 자본비용에 어떤 영향을 미치는가? / 최적의 타인자본-자기자본 비율을 결정

자금조달 결정

원천별 자본비용

가중평균 자본비용

**4.2 재무구조이론 종류**

**MM 이전의 재무구조이론 :** 적정 수준의 부채는 기업가치를 높일 수 있다

**MM (1958)**

주장 : 재무구조는 기업가치를 변화시키지 않는다

가정 : 완전자본시장 가정, 타인자본 지급불능 위험 없음

- 1명제 : 타인자본 사용과 무관하게 WACC는 일정하여 완전자본시장하에서 재정 이익은 불가능
- 2명제 : WACC에 대한 요구수익률 > 타인자본의 요구수익률 이면 자기자본은 증가 (재무레버리지)

**MM (1963)** : 법인세를 고려하면 타인자본의 증가가 기업가치의 증가를 가져온다 (이자가 비용처리)

**Miller (1977)** : 법인세와 개인소득세를 고려하면 타인자본의 증가와 기업가치는 무관하다

한계점

- 파산비용 : 무형자산 가치 없어짐, 유형가산 평가절하, 청산비용 발생
- 대리비용 : 감시비용, 확증비용, 잔여손실, 자기자본 대리비용, 타인자본 대리비용
- 신호효과 : 타인자본을 증가시켜 추가 이익을 낼 수 있다는 신호를 외부에 전달

### 5. 배당이론

친배당이론 : 확실한 현금흐름의 실현이 취지에 부합

반배당이론 : 세율을 고려하면 주가상승이 취지에 부합

## 04 포트폴리오 관리 (5)

### 1. 투자수익과 투자위험

**기대수익**

- 주식투자수익 = 배당소득 + 시세차익
- 추후 발생가능한 상황과 확률을 추정하여 평가

**1.1 수익률**

**내부수익률 (internal rate of return, IRR)**

- 미래에 받을 수익 = 현재 투자금이 되게 하는 수익률을 계산한 것
- aka 금액가중 평균수익률

**산술평균 수익률 (arithmetic average rate of return, ARR)**

- 단일 기간 각각의 수익률을 더해서 평균냄
- 미래 수익률 계산에 적합
- aka 시간가중평균 수익률

**기하평균 수익률 (geometric average rate of return, GRR)**

- 단일 기간 각각의 수익률을 곱하여 제곱근 씌움
- 과거 수익률 계산에 적합

**1.2 투자위험**

- 분산과 표준편차 사용

위험회피형 : 수익이 증가할 때 효욕은 체감, 가로축에 대해 오목

위험중립형 : 선형

위험선호형 : 수익이 증가할 때 효용은 체증, 가로축에 대해 볼록

**1.3 최적 증권의 선택**

- 지배원리, 효용 곡선

### 2. 효율적 분산 투자

- 포트폴리오의 기대 수익과 위험도 동일하게 계산

$$
\sigma_p^2 = w_X^2 \sigma_X^2 + w_Y^2 \sigma_Y^2 + 2 w_X w_Y Cov(XY)
$$

- 투자비율을 조정, 상관계수가 낮은 증권으로 선정

**마코위츠 모형**

**2.1 포트폴리오 결합선**

기대수익률 - 위험 곡선

$$
w_X = \frac{\sigma_Y^2 - \rho_{XY} \sigma_X \sigma_Y}{\sigma_X^2 + \sigma_Y^2 - 2 \rho_{XY} \sigma_X \sigma_Y}
$$

$$
w_X = \frac{\sigma_Y^2 - \text{Cov}(X, Y)}{\sigma_X^2 + \sigma_Y^2 - 2\,\text{Cov}(X, Y)}
$$

**N개 항목**

$$
\sigma_p^2 = \sum_{i=1}^N \sum_{j=1}^N w_i w_j \mathrm{Cov}(R_i, R_j)
$$

$$
\sigma_p^2 = \sum_{i=1}^N w_i^2 \sigma_i^2 + \sum_{i=1}^N \sum_{\substack{j=1 \\ j \neq i}}^N w_i w_j \mathrm{Cov}(R_i, R_j)
$$

→ 체계적 위험 존재

**2.2 무위험자산**

자산배분선 : 무위험자산이 포함된 경우의 투자기회선

$$
E(R_p) = R_f + w[ E(R_A)-R]
$$

 = **마코위츠 모형** (모든 종목 간의 공분산을 고려)

- 총 2n+n(n-1)/2

### 3. 단일 지표 모형

- 개별 주식과 단일시장지표의 공분산만 고려 (aka 단일 시장 모형)
- 가정 : 개별 증권 가격 변동 = 시장요인 변동 + 개별요인 변동
- 총 3n+2

**3.1 증권 특정선 (Characteristic Line, CL)**

- 의의 : 회귀분석의 결과
- 증권의 수익률은 시장수익률과 선형관계 가짐

$$
R_j = \hat{\alpha}_j + \hat{\beta}_j R_m + \epsilon_j
$$

$$
\hat{\beta}_j = \frac{\mathrm{Cov}(R_j, R_m)}{\mathrm{Var}(R_m)}
$$

(회귀분석의 특징)

$$
\hat{\alpha}_j = \bar{R}_j - \hat{\beta}_j \bar{R}_m
$$

$$
\sigma_p^2 = \beta_p^2 \sigma_m^2 + \sigma_e^2
$$

(1번 식에 분산을 취함)

$$
E(R_i) = R_f + \beta_i (E(R_m) - R_f)
$$

**3.2 단일 지표 모형 응용**

- 인덱스펀드 : 베타가 1이 되게 하고 잔차를 최소화함
- 다지표모형 : 지표 개수를 늘림
- 조정 베타 : 추정 베타 2/3 + 시장 베타 1/3

### 4. 자본자산 가격결정 모형

**4.1 자본자산 가격결정 모형 (Capital Asset Pricing Model, CAPM)**

의의 : 균형 상태에서 자본자산의 가격 - 위험 관계 예측 모형

가정

- 투자자는 기대수익과 분산기준에 기반해서 선택함
- 투자자들은 투자기간이 같으며 예측도 동일함
- 거래비용과 세금이 없음
- 무위험자산이 존재함
- 시장은 균형 상태에 있음 (효율적 시장은 아니어도 됨)

**4.2 자본시장선 (Capital Market Line, CML)**

의의 : 무위험 자산을 포함하여 분산투자하는 경우 효율적 ‘포트폴리오’의 기대수익 - 위험 관계

$$
E(R_p) = R_f + \frac{E(R_m) - R_f}{\sigma_m} \cdot \sigma_p
$$

![image.jpeg](I%20%EC%A6%9D%EA%B6%8C%EB%B6%84%EC%84%9D%20%EA%B8%B0%EC%B4%88/image%201.jpeg)

- 시장포트폴리오 : 위험있는 주식 포트폴리오 중 유일하게 효율적인 포트폴리오

**4.3 증권시장선 (Securities Market Line, SML)**

의의 : 비효율적 투자대상까지 포함한 모든 투자‘자산’의 기대수익 - 위험 관계

$$
E(R_i) = R_f + \beta_i \cdot (E(R_m) - R_f)
$$

**4.4 자본시장선(CML) vs 증권시장선(SML) 비교**

| 항목 | CML (자본시장선) | SML (증권시장선) |
| --- | --- | --- |
| 위험 지표 | **표준편차(σ)** → 총위험 | **베타(β)** → 체계적 위험 |
| 설명 대상 | **효율적 포트폴리오**
(시장포트폴리오 + 무위험자산) | **개별 자산, 비효율 포트폴리오, 효율 포트폴리오 모두 포함** |
| 선 위의 의미 | **완전한 효율적 포트폴리오만** 존재 가능 | **이론적 공정가치**에 있는 자산들 |
| 기반 이론 | 포트폴리오이론 (Markowitz) | 자본자산가격결정모형 (CAPM) |
| 의의 | 투자자는 무위험자산과 시장포트폴리오만 조합하면 최적이다 | 자산의 베타가 높으면 기대수익률도 높아야 한다는 기준선 역할 |
| 특징 | 개별 비효율 자산 포함 안 됨 | 어떤 자산이든 다 분석 가능 |

**4.5 차익 가격결정 모형 (APM)**

**CAPM vs APM 비교**

| 항목 | CAPM (Capital Asset Pricing Model) | APM (Arbitrage Pricing Theory) |
| --- | --- | --- |
| **모형 구조** | 단일 요인 모형 (시장수익률) | 다요인 모형 (거시경제 요인 등) |
| **요인 수** | 1개 (시장 위험) | 다수 (인플레이션, 금리, 산업성장 등) |
| **수익률 결정** | 시장수익률과 베타로 설명 | 여러 요인에 대한 민감도와 요인 수익률로 설명 |
| **베타 해석** | 시장 민감도 | 각 요인에 대한 민감도 |
| **가정** | 효율적 시장, 투자자는 평균-분산을 기준으로 행동 | 차익거래 기회는 존재하지 않음 (무차익 조건) |
| **적용 대상** | 주로 주식시장 | 주식뿐 아니라 채권, 파생 등 다양한 자산군에 적용 가능 |
| **측정 용이성** | 비교적 단순 (시장 수익률 기준) | 요인 선정과 계측이 복잡 |
| **수식 형태** | \( E(R_i) = R_f + \beta_i (E(R_m) - R_f) \) | \( E(R_i) = R_f + \beta_{i1}F_1 + \beta_{i2}F_2 + \cdots + \beta_{in}F_n \) |

### 5. 증권시장의 효율성

**5.1 효율적 시장가설 (EMH)**

의미 : 증권 가격은 모든 정보가 반영된 상태이다

종류

- 약형 : 과거의 정보가 반영되어 있다
- 준강형 : 공개된 정보가 반영되어 있다
- 강형 : 내부정보까지 반영되어 있다

특징

- 주가는 무작위적으로 변화
- 새로운 정보는 빠르게 반영됨
- 성공적인 투자전략은 없음
- 전문투자자는 평균 이상의 투자성과를 얻을 수 없음

검증 및 예외

- 약형 : 주말 효과, 1월 효과 ↔ 과거 주가 흐름에 패턴이 있는 반증사례
- 준강형 : 주식분할, 저 PER, 기업규모 효과, 저 PBR ↔ 공개된 정보인데도 효과적이므로 반증사례
- 강형 : 내부자 소유지분변동, 뮤추얼펀드 ↔ 내부정보/실력으로 추가이윤을 낼 수 있으므로 반증사례

### 6. 통합적 투자관리

과정 : 투자목표 설정 → 투자분석과 자본시장 가정 → 자산배분과 종목 선정 → 사후통제

**6.4  투자 성과 평가**

- 투자수익률 : IRR, ARR, GRR
- 샤프지수 : 초과수익 / 총위험 → 전부 위험자산일 때
- 트레이너지수 : 초과수익 / 베타
- 젠센지수 : 수익률 - SML
- 평가비율 : alpha / 잔차의 표준편차 → 섞인 자산일때

### 투자 성과 평가 지표 비교표
