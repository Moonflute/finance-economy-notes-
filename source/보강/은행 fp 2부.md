# 제1과목 금융자산 투자설계

## 1. 금융상품 (16)

### **01 목적별 금융상품**

요구불 예금 : 보통예금, 저축예금, MMDA, 당좌예금, CMA

- 보통예금 : 거의 거래되지 않음
- MMDA : 예치금액에 따라 차등금리
- 당좌예금 : 복수계좌 개설 가능
- CMA : 종합금융회사에서 취급하면 예금자보호 가능

적립식 저축 : 정기적금, 상호부금, 신용부금, 재형저축, 농어가목돈마련저축, 청년희망적금, 청년도약계좌

- 상호부금 : 일정 횟수 이상 납입시 담보대출이 가능한 상품, 정기적립식/자유적립식
- 재형저축 : 7년(서민형 3년) 유지 시 소득세 면제, 총급여 5000만 근로자 or 종소액 3500 이하 사업자, 연 1200만까지 (판매종료)

거치식 저축 : 정기예금, 정기예탁금, 양도성 예금증서 (CD), 환매조건부채권 (RP), 표지어음

- 정기예탁금 : 정기예금과 유사, 상호저축은행/상호금융에서 판매
- 표지어음 : 금융기관이 여러개의 어음을 묶어 발행하는 어음

주택청약 : 주택청약 종합저축, (청약예금, 청약부금, 청약저축)

- 청약예금 : 한번에 예치, 민영주택
- 청약부금 : 정기납입, 민영주택
- 청약저축 : 정기납입, 국민주택

채권상품 : 후순위채권, 산업금융채권, 중소기업금융채권

- 후순위채권 : 선순위 - 일반 - 후순위 - 주식
- 산업금융채권 : 국가 기간산업 자금조달용

기타상품 : 골드뱅킹, ISA

- 골드뱅킹: 금실물에 투자시 부가가치세, 금가격에 투자시 배당소득세, 금 1트로이온스당 미달러 표시가격
- ISA: 초과분 9.9% 분리과세, 연간 납입한도는 2천만원, 의무가입기간은 3년

예금자보호 x : 펀드, 특정금전신탁, 증권사CMA, CD, CP(기업어음), RP, 농어가목돈마련저축, 채권, 골드뱅킹, 정기예탁금

### **02 집합투자상품**

**집합투자기구**

자본시장법 : 포괄주의, 추상적, 기능별 규율

종류 : 신탁형, 회사형, 조합형 / 공모, 사모 / 신탁, 주식, 유한, 합자, 회사, 조합 / 증권, 파생, 부동, 실물, 특별

- 주식형 : 60% 주식관련 투자, 액티브/패시브
- 채권형 : 60% 채권관련 투자, 액티브/패시브, 주식 없어야함
- 부동산 : 50% 부동산 투자, PF/임대수익형/경매공매형/직접개발형/증권형
- 특별자산 : 50% 특별자산 (증권 부동산 외)
- MMF : 정해진 단기금융상품에만 투자 (avg 75/120/60일)

- 날짜 기준
    - 주식 50% 이상 : 기준가 T+1, 대금 지급일 T+3
    - 주식 50% 미만 : 기준가 T+1, 대금 지급일 T+2

- 환매금지 : 만기까지 환매금지
- 종류형 : 투자자별 차등 적용, class 구성
- 전환형 : 하위펀드를 모아 만듬
- 상장지수 : ETF

증권 : 채무, 지분, 수익, 투자계약, 파생결합, 증권예탁증권

- 수익증권은 한국예탁결제원 명의로 일괄예탁으로 발행

보수 : 판매보수 (1%), 판매수수료 (2%)

매입 : 미래가격 적용

- 단기금융집합투자기구는 장부가평가를 원칙으로 함
- 환매금지형 집합투자기구는 최초 발행일 90일 이내 상장해야 함

**제3장 기타금융상품**

구조화 상품 : ELS, ELT, ELF, ELB

- ETF는 30일 이내 상장해야 함

**신탁**

종류 : 금전신탁 (연금저축, 특정금전), 재산신탁 (유가증권, 금전채권, 부통산, 동산)

- 연금신탁 : 불특정금전신탁, 원본보존 및 예금자보호
    - 연금소득세 원천징수, 납입액 범위 내에서 세액공제
    - 최소 5년 이상, 연간 한도 1800만원
- 특정금전신탁 : 실적배당형 단독 운용, 중도해지는 가능

**대출**

대출금리 = 자금조달금리 + 리스크관리비용 등 원가 + 마진 + 가감조정

(* 프라임레이트 = 기본금리)

종류 : 특정채무담보/특정근담보/한정근담보/포괄근담보, 신용대출/주택담보대출/부동산담보대출/보증서담보대출

- 특정채무담보 : 지정 대출만 담보제공 (연장갱신 불가능)
- 특정근담보 : 지정 거래에서 발생하는 불특정 채무를 담보

- 주택담보대출: 일반적으로 금리가 낮음, 부동산시장이 과열되는 경우 : 낮은 LTV, DTI

**외환예금**

환율 : 시장평균환율 적용 (익일영업일 거래 거래량 가중평균)

- 최고 : 현찰매도율
- 최저 : 현찰매입율

외환예금 : 이자는 360로 환산하여 계산

대외계정 : 외국인, 비거주자가 사용하는 외화예금계정

외화수표: 은행수표는 우리나라의 자기앞수표에 해당, 개인수표는 우리나라의

가계수표, 당좌수표에 해당

**신용카드**

종류 : 체크카드, 직불카드, 선불카드

회원 : 본인회원, 가족회원

본인회원 : 만 18세 이상의 결제능력 있는 실명의 개인

한도 : 초기한도, 해외 기본한도, 가족회원 한도, 초과한도, 잔여한도

결제 : 회전결제(리볼빙)

사고 : 신고시점 이후 & 신고 전 60일 이내 대금 보상가능 / 위변조, 명의도용은 회사책임

- 본인 중대 과실인 경우는 보상하지 않음
- 타인 대여로 발생한 경우, 뒷면 서명하지 않은 경우 보상 불가

## 2. 주식투자 (15)

### 01 주식 개요

**주식투자**

- 투자요인 : 시세차익, 배당수익, 유무상수익, 비과세
- 결정요인: 기업 내적요인(양적요인, 질적요인), 기업 외적요인(시장 내적요인, 시장 외적요인)
- 기본적 분석, 기술적 분석

**주식시장**

- 발행시장: 1차시장, 자금조달 기능, 자본의 효율성 제고, 금융정책의 수단
- 유통시장: 2차시장, 환금성 제공, 공정가격 제공, 가격결정 지표, 유가증권 담보력 제공

**주가지수**

한국 : KOSPI, KOSPI200, KOSTAR

미국 : 다우존스(30종목), 나스닥(Non-US), S&P500

중국 : A주 중국인 대상, 위안화, B주 외국인 전용, 달러

- 주가평균 : 미국 다우존스지수, 일본 Nikkei225

**경기변동 : 침체기 → 회복기 → 활황기 → 후퇴기**

① 정(+): 경제성장률, 단기통화량, 완만한 물가상승, 환율

② 부(-): 장기통화량, 급격한 물가상승, 이자율, 원자재 가격

### 02 산업 분석

**산업분석**

① 경쟁구조 분석 : 진입장벽, 산업내 경쟁강도, 대체품위험, 구매자/공급자의 교섭력

② 수명주기 분석 : 도입기 → 성장기 → 성숙기 → 쇠퇴기

### **03 기업분석**

**재무비율**

- 수익성 : 총자본이익율(ROI), 자기자본이익률(ROE), 납입자본이익율, 매출액순이익율
- 안정성 : 유동비율, 부채비율, 고정비율, 이자보상비율(영업이익÷이자비용, 클수록 좋은 기업)
- 활동성 : 총자산회전율, 고정자산회전율, 재고자산회전율, 매출채권회수기간(매출채권÷1일 평균매출액)
- 성장성 : 매출액증가율, 총자산증가율, 영업이익증가율
- 유동성 : 유동비율, 당좌비율

**시장가치비율**

- PER : 주가÷주당순이익, 낮을수록 저평가
- PBR : 주가÷주당순자산, 낮을수록 저평가
- PCR : 주가÷주당현금흐름, 영업성과/자금조달능력 평가
- PSR : 주가÷주당매출액, 이익없는 기업 평가 가능
- 토빈의 q: 자산의 시장가치÷추정 대체비용 (q>1: 새로운 기업 진입, q<1 : M&A의 대상)
- 배당수익률 : 배당금÷주가, 투자금액 대비 수익률, 배당률(배당금÷액면가격), 배당지급율(배당금÷순이익)

**보통주평가모형**

- 배당평가모형 : 정률성장
- 이익평가모형 : 배당정책의 왜곡 보완, 재투자수익률이 요구수익률보다 높을때 주가 상승
- 자산가치평가모형 : 청산가치, 토빈의 q로 보완

**PER**

- PER= (1-f)(1+g) / k-fr
- 배당성향/이익성장율 클수록, 기대수익률 작을수록 커짐
- 장점 : 이익크기 비슷한 기업의 주가수준 비교, 무배당 주식도 평가, 구성요소 예측이 상대적으로 용이
- 단점 : 구성요소의 불일치, 회계처리방법 등 왜곡, 이론적 근거 불명확, 음(-)의 이익 기업 평가 곤란

**PBR**

- 기업이익, 활동성, 부채레버리지, 이익승수 종합적 반영
- 장점 : 수익가치에 대한 포괄적 정보 반영, 회계정보 활용 용이, 부(-)의 EPS 기업 평가 가능
- 단점 : 청산가치로 계속기업 전제 곤란, 기업의 수익력 평가 곤란, 인적자산/비지니스모델 등 미반영

**기타 주가배수**

- PSR : 왜곡가능성 가장 적고, 적자기업 평가 가능
- EV/EBITDA : EBITDA가 (-)인 기업없고, 회계처리 등의 영향 적으며, 자본집약적 산업평가에 유용, 운전자본 증가시 현금흐름 과대계상, 현금흐름 왜곡

### 04 포트폴리오 구성

**주식포트폴리오 구성 프로세스**

① 투자유니버스선정 → 모델포트폴리오 → 실제포트폴리오

② 하향식(Top-down): 섹터의 분류가 중요

③ 상향식(Bottom-up): 유망한 개별종목 선정 강조

④ 엑티브전략(시장 비효율적), 패시브전략(시장 효율적)

**포트폴리오 성과평정**

리밸런싱(투자비율 환원), 업그레이딩(재구성, 손실제거)

운용수익률 : 내부수익률(금액가중), 산술평균수익률(시간가중), 기하평균수익률(재투자 감안)

위험조정수익률 : 높을수록 우수

- 샤프 : 위험 1단위당 초과수익, 분산투자 잘된 펀드평가
- 트레이너 : T= (R₁-R₄) / β
- 알파 : a= (R-Rf) - [(Rm - Rf) × β], 펀드매니저 능력평가

**투자전략**

소극적 투자전략 : 단순매수보유전략, 인덱스투자전략, 평균투자법

적극적 투자전략 : 시장투자적기포착, 포뮬러플랜, 종목선정, 운용스타일

- 침체기 ⇒ 가치투자가 유리

## 4. 채권투자 (15)

### 01 금리

**금리 종류**

실질금리 = 명목이자율 / 물가상승률

실효금리 : 실제 평가되어 부담하는 금리

정책금리 : 중앙은행이 결정, 7일물 RP금리

채권수익률 : 만기수익률, 연복리수익률, 시장수익률

**금리 영향**

경기상승 : 금리상승

물가 : 상승기대시 금리상승, 상승시 금리상승

재정정책 : 확장정책시 금리상승, 긴축정책시 하락

통화정책 : 확장정책시 금리하락, 긴축정책시 상승

해외금리 : 공조화현상으로 상승시 금리상승

### 02 채권 개요

**채권 종류**

발행 : 국채, 지방채(무위험채권), 특수채, 금융채, 회사채(통안채는 금융채로 분류, 무위험채권)

이자 : 이표채, 할인채(1년 이하), 복리채

만기 : 단기(2년 이하), 중기, 장기(7년 초과)

콜(풋)옵션부사채 : 금리 하락(상승)시 권한 행사

전환사채 : 채권금리와 주가상승 동시 향유, 일반채권에 비해 낮은 금리, 전환권행사시 주가상승 곤란

이익참가부사채 : 이자지급 & 이익분배 참여 가능

**채권 가격**

말킬의 채권가격정리

① 채권수익률과 채권가격은 반대로 움직인다

② 만기가 긴 채권일수록 금리변동에 대한 가격변동폭이 크다.

③ 금리변동에 따른 채권가격의 변동폭은 만기가 길수록 증가하나 그 증가율은 체감한다

④ 같은 폭의 금리 움직임이라도 금리하락시 가격상승폭이 금리상승시 가격하락폭보다 크다

⑤ 표면이자율이 낮은 채권이 표면이자율이 높은 채권보다 금리변동에 따른 가격변동폭이 크다

- 수정듀레이션=듀레이션 / (1 + 채권수익률)

**수익률 곡선**

① 잔존만기와 금리와의 관계를 표시(기간구조)

② 우상향곡선이 일반적, 유동성선호이론이 잘 설명

③ 장단기스프레드가 확대(축소)되면 일드커브스티프닝(플래트닝): 일드커브 하락은 채권시장의 강세

- steepening = 금리차가 확대됨 (곡선 기울기 가팔라짐) = 약세 (Bearish)
- flattening = 금리차가 축소됨 (곡선 기울기 완만해짐) = 강세 (Bullish)

**채권의 신용등급**

① 무위험채권 : 국채, 통안채, 지방채 ↔ 크레딧물

② 총 18개 등급(AAA~D) : BBB-이상이 투자등급

③ 신용등급 높을수록 높은 가격, 낮은 금리로 발행/거래

④ 무위험채권과 크레딧물의 금리차 : 신용스프레드

⑤ 크레딧물이 국채에 비해 강세(약세)인 경우 신용스프레드 축소(확대)

채권의 투자수익

① 이자수익 : 이표채는 이자지급 후 다시 0에서 시작

② 자본손익 : 매입금리〉매도금리 일 때 이익

③ 롤링효과 : 롤링수익(5년이상 중기), 숄더효과(단기)

④ 투자수익률 = 이자수익률 + 자본수익률

⑤ 기대수익률 = 채권이자수익률+[(매입금리-1년후 평가금리)×잔존듀레이션]

채권의 위험

① 위험프리미엄=크레디트프리미엄+듀레이션프리미엄

② 듀레이션위험(시장위험): 금리상승, 채권가격하락

③ 신용위험 : 부도위험, 신용등급하락위험(채권가격 하락), 신용스프레드위험(크레딧물 약세)

④ 유동성위험 : 신용등급 낮을수록, 장기물일수록 증가

⑤ 중도상환위험 : 콜옵션부채권의 발행자의 권한 행사

    - **답:** 물을 상시적으로 직접 이용하여 벼, 연 등을 재배하는 토지 (논).
    - **과수원:** 과수류를 집단적으로 재배하는 토지와 이에 접속된 부속시설물 부지.
    - **목장용지:** 축산업 및 낙농업을 위한 초지, 축사, 부속시설물의 부지.
    - **임야:** 산림 및 원야(들판)를 이루고 있는 수림지, 암석지, 황무지 등의 토지.
    - **광천지:** 지하에서 온수, 약수, 석유류 등이 용출되는 용출구와 그 유지 부지.
    - **염전:** 바닷물을 끌어들여 소금을 채취하기 위하여 조성된 토지와 부속시설물 부지.
    - **대:** 영구적 건축물(주거, 사무실, 점포 등)의 부지와 이에 접속된 정원 및 부속시설물 부지.
    - **공장용지:** 제조업을 하는 공장시설물의 부지와 산업단지 내 공원 및 녹지 부지.
    - **학교용지:** 학교의 교사(校舍)와 이에 접속된 체육장 등 부속시설물의 부지.
    - **주차장:** 자동차 등의 주차에 필요한 독립적인 시설을 갖춘 부지와 부속시설물의 부지.
    - **주유소용지:** 석유, 액화가스 등을 판매하기 위한 시설물의 부지와 이에 접속된 부속시설물 부지.
    - **창고용지:** 물건 등을 보관하거나 저장하기 위하여 독립적으로 설치된 보관시설물의 부지.
    - **도로:** 일반의 교통을 위해 보행이나 차량 운행에 사용되는 토지.
    - **철도용지:** 교통 운수를 위하여 궤도 등의 설비를 갖춘 토지와 부속시설물의 부지.
    - **제방:** 자연유수, 조수, 모래, 바람 등을 막기 위해 설치된 방조제 등의 부지.
    - **하천:** 자연의 유수(流水)가 있거나 있을 것으로 예상되는 토지.
    - **구거:** 용수 또는 배수를 위하여 일정한 형태를 갖춘 인공적인 수로와 그 부속시설물의 부지.
    - **유지:** 물이 고이거나 상시적으로 물을 저장하고 있는 댐, 저수지, 연못 등의 토지.
    - **양어장:** 육상에 인공적으로 조성된 어류, 해조류 등의 양식 시설 부지.
    - **수도용지:** 물을 정수하여 공급하기 위한 취수, 정수, 송수 및 배수 시설의 부지.
    - **공원:** 일반 공중의 보건·휴양 및 정서생활에 이용하기 위한 시설을 갖춘 토지.
    - **체육용지:** 국민의 건강증진 등을 위한 체육활동에 적합한 시설을 갖춘 토지.
    - **유원지:** 일반 공중의 위락·휴양 등에 이용되는 대규모 시설(수영장, 낚시터 등) 부지.
    - **종교용지:** 일반 공중의 종교의식을 위한 예배당, 교회, 성당 등의 건축물 부지.
    - **사적지:** 문화재로 지정된 역사적인 유적, 고적, 기념물 등을 보존하기 위해 구획된 토지.
    - **묘지:** 사람의 시체나 유골이 매장된 토지와 이에 접속된 부속시설물의 부지.
    - **잡종지:** 위에 열거된 다른 지목에 속하지 않는 모든 토지 (갈대밭, 야외시장, 공항시설 등)

필지 : 토지를 구분하는 법률적 단위 (하나의 지번이 부여됨)

획지 : 토지를 구분하는 경제적 단위 (감정평가의 기본 단위)

신축, 증축, 개축, 재축, 이전

- **신축:** **건축물이 없는 대지**에 새로 건축물을 축조하는 행위 (부속건축물만 있는 대지에 새로 주된 건축물을 축조하는 것도 포함).
- **증축:** 기존 건축물이 있는 대지에서 **건축물의 면적, 높이 또는 층수를 늘리는** 행위.
- **개축:** 기존 건축물의 **전부 또는 일부(내력벽, 기둥, 보, 지붕틀 중 3개 이상)**를 철거하고 **종전과 같은 규모의 범위**에서 다시 축조하는 행위.
- **재축:** 재해로 멸실된 건축물의 **전부 또는 일부**를 멸실 당시와 **같은 규모 이하**로 다시 축조하는 행위 (법령의 기준에 모두 적합하면 종전 규모를 초과할 수 있음).
- **이전:** 건축물의 **주요 구조부(내력벽, 기둥, 보, 지붕틀)**를 해체하지 않고 **같은 대지의 다른 위치**로 옮기는 행위.

**전원주택** : 농가형/주말농장형/단지형, 투자보다 실수요 중심으로 접근, 보안, 교통 고려 필요

### 05 상가

정의 : 공간적으로 개발하여 건축한 부동산 상품으로 근린생활시설을 의미

종류 : 단지내상가, 근린상가, 일반상가, 테마상가, 복합상가, 오피스상가, 기타상가

매매가 = 대지면적 x 평당가격

전략

- 경기침체 시 수익성 감소
- 임차인 맞춰두고 분약
- 큰 업체를 세입자로 둬야 유리

**오피스텔**

정의 : 업무를 위주로 하되 숙식이 가능한 시설

특성 : 전세가 상승으로 이주 확대, 도시형생활주택과 경쟁관계

전략 : 수요가 안정적인 지역으로 선택, 2층 비는거까지 감당가능

- 영화관은 상권 활성화에 크게 도움되지 않음

### 06 토지

**도로** : 보행 및 자동차 통행이 가능한 너비 4m 이상의 도로

**농지** : 농업을 경영하기 위해 사용하는 토지

**농업인**

- 1,000㎡ 이상의 농지에서 농작물 또는 다년생생물을 재배
- 1년에 90일 이상을 농업에 종사
- 농업경영을 통한 농산물의 연간 판매액이 120만원 이상
- 고용되지 않고 고용하지 않는 생산자

**농업법인** : 업무집행권을 가진 자 중 농업인이 1/3 이상

**토지거래허가제도**

의의 : 토지거래허가구역 내에 기준면적을 초과하는 토지계약을 하고자 하면 허가를 받도록 함

대상 : 대물변제, 담보, 유저당계약 / 주거지역 60m^2 이상

제외 : 건물소유권 이전, 전세권/임차권/저당권, 증여/상속/유증

**토지 투자**

- 부동산 중 가장 환금성이 떨어짐에 유의
- 주변 지역의 개발 가능성을 점검해야 함, 도로에 접하지 않았으면 투자하긴 어려움
- 개발비용까지 추산해야 함
- 지속적으로 개선해야 함

- 주말농장은 100

### 07 경매

**종류**

강제경매 :  채권자가 채무자 소유의 부동산을 압류한 후 매각시켜 금전채권의 회수하는 강제집행

임의경매 : 전세권, 질권, 유치권, 담보가등기 등 담보권실행을 위한 경매

법원경매 :

- 낙찰자 미정시 신경매 진행
- 낙찰자 미납시 재경매 진행, 기존 낙찰자 입찰불가
- 매수신청보증금은 10%

인도명령 : 채무자가 낙찰부동산을 인도하지 않는 경우 대금완납 후 6개월 이내에 강제로 부동산을 인도하게 함

명도소송 : 인도명령의 요건을 갖추지 못한 경우 활용 (공매 등)

**유의사항**

- 입찰가격 수정 시 새 용지에 기입
- 경매는 1회 유찰 시마다 다음 회차 진행 시 감정평가액이 20%씩 낮게 조정, 1달 후 기일 결정
- 말소기준 권리는 저당권, 근저당권, 압류, 가압류 중 최선순위권리가 됨
- 경매물건의 밀린 관리비의 경우 대법원 판례에 따르면 매수인은 공용부분의 원금만 납부하면 됨
- 유치권에 주의

### 08 NPL

정의 : non performing loan, 금융권의 미회수채권, 부실채권 → 자산유동화회사 매각

투자 : 개인이 직접적인 참여방법은 차단됨, 개인은 NPL 채권 매입 불가능

## 4. 부동산 관리 전략 (3)

### 01 부동산 자산관리

**트렌드 변화**

- 리츠 시장 확대
- 양도차익보다는 운용수익이 목적
- 부동산 공급자 중심 → 수요자
- 소유투자 목적 → 거주이용 목적
- 자산관리, 전문위탁관리, 전문자산관리회사 등장
- 패키지, 인터넷 통합관리시스템
- 고층화 대형화, 투자기회 증대, 운영비용 증가
- 개인 자산의 70%가 부동산
- 단순 부동산중개업은 한계에 다다름
- 시설관리(FM) → 재산관리(PM)

**부동산 자산관리의 분야**

부동산자산관리(REAM): 자산운용업무(건물자산의 매입매각, 투자관리업무), 재무관리업무(파이낸싱, 회계, 수익관리 등), 최적의 포트폴리오업무 / 포괄적 관리

- 직접관리방식 / 위탁관리방식 / 혼합관리방식

부동산재산관리(PM): 건물자산의 임대관리업무를 수행하며 수익관리 공실관리, 시설물관리 등 수행

부동산시설관리(FM): 시설물 유지관리업무 수행

**부동산금융의 특징**

① 장기대출: 주택금융은 보통 20년의 장기(상업금융은 1년 내지 2년)

② 저리대출: 가계의 소득수준을 감안한 상환능력에 의하여 결정

③ 개인대상 : 신용의 제한, 간접금융에 의존

④ 채무불이행 위험: 주택이 직접적으로 수익을 발생시키지 않으므로 가계의 소득에 의존

### 02 부동산 금융상품

종류 : 직접투자, 간접투자

**리츠 (REITs)**

정의 : 부동산투자전문회사

종류 : 자기관리형, 위탁관리형, 구조조정형

- 위탁관리는 지점 설치 불가 (paper company)
- 국내 리츠는 개발사업, 단기매매 불가능

특징 : 단기매매 제한, 법인세 면제 안됨,

**부동산 개발금융 (PF)**

정의 : 개발 프로젝트의 자금을 조달하는 금융기법

종류 : 토지신탁, PF브리지론, PF loan형 ABS, 매출채권형 PF

**PF 브리지론**

정의 : 단기적이고 긴급한 자금 대출용
