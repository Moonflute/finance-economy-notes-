## 01 주식평가, 분석 (10)

### 1. 주식과 주식시장

**1.1 주식**

**보통주 - 우선주**

보통주 : 위험부담이 크지만 기대 미래이익이 큼

- 액면 무액면, 기명 무기명, 의결권 무의결권, 신주 구주

우선주 : 의결권이 제한되지만 배당과 재산권 행사가 우선

- 누적적 비누적적, 참가적 비참가적, 의결권부 무의결권, 전환우선 비전환우선

**1.2 금융시장**

**종류**

성격별 : 자본시장, 단기금융시장, 예금대출시장, 외환시장, 파생상품시장

기관별 : 직접금융시장, 간접금융시장

* 증권시장 : 발행시장 / 유통시장

**1.3 증권의 발행**

**발행형태**

직접발행 : 자기모집, 위탁모집 → 발행 위험을 본인이 부담

간접발행 : 총액인수, 잔액인수, 모집주선

사모발행 : 주로 직접발행 (주주배정 유상증자)

공모발행 : 모집 (신규발행), 매출 (기발행) → 50인 이상 대상 (전문가, 연고자 제외)

**기업공개 자격**

규모요건 : 자기자본, 상장주식수

분산요건 : 주식수, 주주수, 양도제한 없음

경영성과요건 : 매출액, 기준시가총액

안정성요건 : 영업활동기간, 감사의견, 매각제한 (보호예수)

**유상증자 - 무상증자**

유상증자 : 주식출자금을 받아 신주를 발행, 기업 총가치 증가

- 사모 (주주배정, 제3자 배정), 공모 (주주우선, 일반)
- 발행가 : 기준주가 x (1 - 할인율)
- 권리락 주가 = (기준주가 + 주당납입금) / (1+증자비율)

무상증자 : 자본잉여금 → 자본금 편입, 총가치는 동일, 주식수만 증가

- 주식배당 : 현금배당금의 1/2 까지 주식으로 배당 가능
- 권리락 주가 = 권리부 주가 / (1+증자비율)

**증권매매**

시간 : 9:00~15:30

주문  : 지정가주문(일반/조건부/최유리), 시장가주문, 목표가주문, 경쟁대량매매주문

규제 : 가격 제한폭 설정, 불공정거래행위 규제, 투자경고/위험종목 지정

        ### 03 재무분석

        **지표 종류**

        - 유동성지표 : 유동비율, 당좌비율, 현금비율
        - 안정성지표 : 부채비율, 부채-자기자본비율
        - 수익성지표 : 매출액영업이익률, ROA, ROE
        - 활동성지표 : 비유동자산회전율, 재고자산회전율, 매출채권회전율, 총자산회전율
        - 보상비율 : 배당성향, 이자보상비율, 고정비용보상비율

        **A. 유동성비율 (CAR, CR, QR) - 단기부채 상환력**

        - 현금비율 = (현금 + 시장성 유가증권) / 유동부채 → 현금보유 상태
        - 유동비율 = 유동자산 / 유동부채 → 단기부채 상환능력
        - 당좌비율 = 당좌자산 / 유동부채 → 일시적 재무위기 시 현금동원능력

        **B. 안정성지표 (DR, DER) - 중장기부채 상환력**

        - 부채비율 = 총부채 / 총자산 → 재무레버리지의 크기
        - 부채-자기자본비율 = 총부채 / 자기자본 → 자본에 대한 레버지리의 크기
        - 이자보상비율 = 영업이익 / 이자비용

        **C. 수익성지표 (OPM, ROA, ROE)**

        - 매출액영업이익률 = 영업이익 / 순매출액 → 영업효율성
        - 총자산이익률 = 순이익 / 총자산 → 자산에서 이익을 뽑아내는 효율
        - 자기자본이익률 = 순이익 / 자기자본 → 자기자본에서 이익을 뽑아내는 효율

        **D. 활동성지표**

        - 비유동자산회전율 = 순매출 / 비유동자산
        - 재고자산회전율 = 순매출 / 재고자산
        - 매출채권회전율 = 순매출 / 순매출채권
        - 총자산회전율 = 순매출 / 총자산

        **E. 보상비율 - 재무 이행능력**

        - 배당성향 = 배당금 / 순이익
        - 고정이자보상비율 = 영업이익 / 고정이자비율

        **이익지표**

        - 주당순이익 (EPS) = (당기순이익-우선주배당금) / 보통주 발행총수

            → 1주당 분배될 수 있는 순이익

    - ROE (자기자본이익률) = 당기순이익/자기자본
- EVA분석 (경제적 부가가치) : 세후 영업이익(NOPAT) - 투자자본 x 가중평균자본비용(WACC)
    - WACC = (부채비용 x 부채비중) + (자기자본비용 x 자기자본비중)
    - 주가변동의 설명요인

### 3. 주식가치평가모형

- 현금흐름으로 평가 : 배당평가모형, 이익평가모형, 잉여현금흐름평가모형
- 주가배수로 평가 : PER, PBR, PSR, PCR, EV/EBITDA
- 자산가치로 평가 : 주당 순자산가치

**3.1 현금흐름 평가**

**배당평가모형**

- 정률성장모형 : P = D1 / (k-g) = D0 (1+g) / (k-g)
- 제로성장모형 : P = D / k ↔ 배당 = 주가  x 요구수익률
- 고속성장모형
    - P (주가), k (요구수익률), g (성장률)

**이익평가모형**

주식의 가치 = (예상이익/k) + (예상이익/k) x (r-k)/(k-g) x 사내유보율

**잉여현금흐름(FCF) 평가모형**

의의 : 영업현금흐름 → 기업 전체 가치(V) 추정 → 주식가치(S) 산출

- 잉여현금흐름 : 세후 영업이익 - 투하자본 증가액

기업가치 = (기간동안 잉여현금흐름의 현재가치 합) + (이후 잔존가치의 현재가치) + 영업외가치

주식가치 = (기업가치 - 부채가치) / 발행주식수

FCF

$$
\begin{align*}\text{기업가치} &= \sum_{t=1}^{n} \frac{FCF_t}{(1 + WACC)^t} + \frac{TV}{(1 + WACC)^n} \\\\FCF &= NOPLAT - \Delta IC \\\\TV &= \frac{FCF_n \cdot (1 + g)}{WACC - g}\\\\\text{기업가치} &= \sum_{t=1}^{n} \frac{FCF_t}{(1 + WACC)^t} + \frac{FCF_n \cdot (1 + g)}{(WACC - g)(1 + WACC)^n} \\\\\text{주식가치} &= \text{기업가치} - \text{순부채} \\\\\text{주당가치} &= \frac{\text{주식가치}}{\text{발행주식수}}\end{align*}
$$

**가중평균자본비용**

WACC = (자기자본 x 자기자본비중) + (부채비용 x 부채비중 x 법인세반영)

자기자본비용 = 배당평가모형, 이익평가모형, CAPM모형 이용

**3.2 주가배수 평가**

**PER 평가모형**

PER (price earning ratio, 주가수익비율)

정의 : 주가 / 주당순이익 EPS = 기대배당성향 / (k-g)

의의 : 이익의 몇 배를 지불하는가

**PBR 평가모형**

PBR (price bookvalue ratio, 주가순자산비율)

정의 : P / BPS = (P/EPS) (EPS/BPS) = PER x ROE

의의 : 자산의 몇 배를 지불하는가

ROE = 매출액순이익률 x 총자산회전율 x (1+부채비율)

= 비용통제의 효율성 x 자산이용의 효율성 x 자본조달의 안정성

**기타 주가배수 평가모형**

PEG : PER / g(성장률)

PSR : 주가 / 주당매출액 - 벤처기업 평가에 사용

PCR : 주가 / 주당현금흐름 - 감가상각비가 많은 산업에 사용

EV/EBITDA : (시총 + 순차입금) / (영업이익 + 감가상각비) - 국제간 기업 비교에 사용

- 기업의 전체가치 / 기업의 수익가치 (내제가치)
- EV : 주주가치 + 채권자가치
- EBITDA : EBIT(세전 영업이익) + 감가상각비 = 세전 영업현금흐름

### 5. 수익률 곡선

**채권 가격**

$$
\text{채권 가격} = \sum \left( \frac{\text{이자}}{(1 + r)^t} \right) + \frac{\text{액면가}}{(1 + r)^n}
$$

**5.1 말킬의 정리**

의의 : 채권가격변동의 5가지 특징

1. 채권가격과 수익률은 역의 관계이다
    - 채권 가격 하락 → 가격 당 수익은 증가
    - 채권 가격 상승 → 가격 당 수익은 감소
2. 채권만기가 길면 채권수익률 변동에 대한 채권가격 변동폭이 커진다
3. 채권수익률 변동으로 인한 채권가격 변동은 만기가 길수록 커지지만, 증감률은 체감한다
4. 채권수익률 하락이 상승보다 가격변동폭이 크다 (볼록성)
5. 표면이자율이 낮으면 수익률 변동에 따른 가격변동률이 크다

**5.2 맥컬레이 듀레이션**

정의 : 채권금리 변화에 따른 채권가격변동성 (표면이자, 잔존만기 고려)

듀레이션 : 채권에 투자한 원금을 회수하는 데 걸리는 시간

특징

- 만기에 일시상환하면 잔존기간 = 듀레이션
- 이표채는 이율이 낮을수록, 만기수익률이 낮을수록, 잔존기간이 길수록 듀레이션이 커짐 (단, <잔존만기)

$$
P = \frac{S}{(1+r)^n (1+r\times \frac{d}{365})}
$$

$$
듀레이션= \sum_{t=1}^{T} t \cdot w_t = \sum_{t=1}^{T} t \cdot \frac{\frac{C_t}{(1+r)^t}}{P}
$$

**5.3 수정듀레이션**

의의 : 금리변동에 따른 채권가격의 변동 정도

$$
\text{수정 듀레이션} = \frac{\text{맥콜레이 듀레이션}}{1 + \text{만기수익률 (YTM)}}
$$

- 채권가격 변동률 : -수정듀레이션 x delta r
- 채권가격 변동폭 : -수정듀레이션 x delta r x P

**5.4 볼록성**

![image.png](II%20%EA%B0%80%EC%B9%98%ED%8F%89%EA%B0%80%EB%A1%A0/image%201.png)

정의 : 만기수익률에 대한 채권가격의 2차 도함수 / 채권가격

→ 금리 변동에 따른 듀레이션의 변화율

→ 양의 볼록성 : 금리 하락 시 채권 가격이 크게 상승

의의 : 실제로는 채권 가격과 수익률이 선형관계가 아님

- 볼록성이 클수록 유리
- 실제 채권가격은 듀레이션측정치보다 위에 있음
- 수익률이 하락할수록 볼록성이 증가
- 실효 연수익률 : 1년에 여러 번 이자를 지급하는 경우의 실제 수익률

$$
실효 연수익률 = (1 + \frac{r}{n})^n - 1
$$

계산법

$$
\frac{\Delta P}{P} = f(0) + \frac{f'(0)}{1!}X + \frac{f''(0)}{2!}X^2  = 듀레이션공식 + 볼록성공식 \\ . \\ =(-) 수정듀레이션 \cdot \Delta r + \frac{1}{2} \cdot convexity \cdot \Delta r^2
$$

- 뒤의 1/2 C r^2만큼이 보정되는 것

**실효듀레이션**

의의 : 금리 변동에 따른 채권 가격의 민감도

$$
\text{실효 듀레이션} = \frac{P_{-} - P_{+}}{2 \cdot P_0 \cdot \Delta y}
$$

$$
\text{실효 볼록성} = \frac{P_{-} + P_{+} - 2P_0}{P_0 \cdot (\Delta y)^2}
$$

**수익률곡선**

정의 : 다른 조건은 동일하고 만기만 다른 채권들의  잔존기간에 따른 수익률의 변동 (x: 잔존기간, y:수익률)

종류 : 우상향, 하강형, 굴곡형, 수평형 (대부분 장기채 금리가 높들이 우상향을 보임)

SPOT curve : 시장에서 관찰된 무이표채의 수익률

**선도이자율**

$$
(1 + S_2)^2 = (1 + S_1) \times (1 + f_{1,1})
$$

$$
\frac{t_2 \cdot S_2 - t_1 \cdot S_1}{t_2 - t_1}
$$

**기간구조이론**

의의 : 채권 만기 - 이자율 관계를 설명

의의 : 정해진 만기시점에 기초자산을 정한 가격에 매도/매수하기로 한 거래

* 선도거래 : 장외에서 이루어짐, 당사자 협의에 따라 자유롭게 이루어짐

특징 : 청산거래 가능, 증거금제도, 일일정산, 미결제약정, 실물인수도방식, 현금결제

$$
베이시스 = F^* - S_t=(r_d-r_f)\frac{T-t}{365}
$$

- 즉, 이자율에서 배당률을 차감한 만큼의 비율로 현재 가격을 미래시험 가치로 변환함

**2.3 선물가격결정**

베이시스 : 현물가격과 선물가격의 차이 (보유비용)

캐리모델 : 수익 - 비용 (선물의 이론가격) → 현물에 보유가격을 더해 선물가를 산정

$$
F = S + S \cdot (r_b - c) \cdot T
$$

- r_b : 현물을 보유하기 위해 필요한 자금을 조달하는 비용
- c : 현물을 보유함으로서 받을 수 있는 수익

→ S(r-c) = 순보유비용

* 내재선도금리 (단리)

$$
f_{t,T} = \frac{R_T \cdot T - R_t \cdot t}{T - t}
$$

* 내재선도금리 (복리)

$$
f_{t,T} = \left( \frac{(1 + R_T)^T}{(1 + R_t)^t} \right)^{\frac{1}{T - t}} - 1
$$

ex) 선물가격이 고평가 : 선물매도 현물매수 포지션

### 3. 옵션

정의 : 주어진 자산을 일정 시점에서 특정가로 매수/매도 할 수 있는 권리

**3.1 종류**

콜옵션 : 매수 권리 (강한 상승 예상 시 매수, 미 상승 예상 시 매도)

풋옵션 : 매도 권리 (강한 하락 예상 시 매수, 미 하락 예상 시 매도)

유럽식 : 만기에만 행사가능

미국식 : 중도에도 행사가능

매수 : 프리미엄만큼 손실 ~ 무한대의 이득

매도 : 무한대의 손실 ~ 프리미엄만큼 이득

**3.2 옵션의 가격**

프리미엄(옵션가격) = 내재가치 + 시간가치

기초가격의 위치별 상태

- 외가격 (비싸게 매수해야) < 콜옵션 행사가 < 내가격 (저렴하게 매수가능)
- 내가격 (비싸게 매도가능) < 풀옵션 행사가 < 외가격 (저렴하게 매도해야)

    → 외가격은 내재가치가 없는 상태, 대부분 거래되지 않고 소멸


**풋-콜 패리티**

만기와 행사가격이 동일한 풋옵션과 콜옵션의 관계 (이자추출 = 보호적 풋)

의의 : 콜옵션 매수 + 채권매수 = 풋옵션 매수 + 기초자산 매수

- 채권은 만기시점에 행사가격만큼을 지급하는 채권

이유

- 만기시점에서 K + Call vs 기초자산 + Put 비교
- S > K : Call 소멸 Put 행사 → K = K
- S = K : Call Put 미행사 → K = S
- S < K : Call 행사 Put 소멸 → S = S

→ 만기시점 K + Call = 기초자산 + Put

→ 현 시점 가치도 동일

$$
C + K e^{-rT}  = S_0 + P
$$

만기에 X의 가치가 되는 채권과 콜옵션 1개를 보유했다면

$$
c+\frac{X}{(1+r)^(T+t)} = p+S_t
$$

**옵션의 가격결정**

이항모형 : 주식이 만기에 2개의 가격만을 가진다고 가정 (a로 오르거나 b로 내리거나)

**블랙-숄즈모형**

의의 : 옵션의 이론적 공정가격 산출

$$
Call = S_tN(d_1,_t) - \frac{X}{(1+r)^T-^t}N(d_2,_t)
$$

$$
Put = Call +  \frac{X}{(1+r)^T-^t} - S
$$

→ 식 2 :  풋 콜 패리티

**기초자산의 변동성**

변동성 스마일 : 블랙먼데이 이전의 형태, 등가격 중심 좌우동형

변동성 스머크 : 블랙먼데이 이후의 형태, 내가격에서 외가격으로 갈수록 증가

**3.3 옵션 전략**

**헤지 전략**

- 보호적풋 : 주식 + 풋매수, 변동성 클 것으로 예상, 손실제한 이익무한 → 풋으로 저점 보장 (내려도 안전)
- 커버드콜 : 주식 + 콜매도, 변동성 작을 것으로 예상, 손실무한 이익제한 → 콜로 추가이익 (어차피 안오름)
- 펜스 : 주식 + 풋매수 + 콜매도

- 델타 헤지 : 주식 포트폴리오를 지속적으로 변화시켜 손익을 상쇄시킴
- 포트폴리오 보험 : 강세장에 계약 일부 환매, 약세장에 추가 계약

**스프레드 전략**

- 불 스프레드 (수직 스프레드)
    - 강세 : 낮은 행사가 매수 + 높은 행사가 매도 → 상승장 예상
    - 약세 : 낮은 행사가 매도 + 높은 행사가 매수 → 하락장 예상
- 시간 스프레드 (수평 스프레드) → 변동이 작으면 이익
    - 매수 : 동일 행사가의 근월물 매도 + 원월물 매수
    - 매도 : 동일 행사가의 근월물 매수 + 원월물 매도

**합성 전략**

- 스트래들 (V)
    - 롱 : 만기와 행사가격이 같은 (콜옵션 + 풋옵션) 매수

        → 시장이 크게 변동할 것으로 예상 시, 크게 오르거나 내리면 해당하는 걸 행사하면 됨

        → 최악의 상황 : 옵션 둘 다 소멸시키고 프리미엄만큼만 손실

    - 숏 : 만기와 행사가격이 같은 (콜옵션 + 풋옵션) 매도

        → 시장이 안정적일 것으로 예상 시, 프리미엄을 얻기 위해 매도

        → 최악의 상황 : 크게 시장이 움직여 옵션 행사만큼을 손실

- 스트랭글 (U)
    - 롱 : (행사가격 높은 콜옵션 + 행사가격 낮은 풋옵션) 매수

        → 스트래들보다 저렴한 프리미엄 지급

    - 숏 : (행사가격 높은 콜옵션 + 행사가격 낮은 풋옵션) 매도

        → 스트래들보다 안전지대를 구축

- 스트립 : 2 put 매수 + 1 call 매수
- 스트랩 : 2 call 매수 + 1 put 매수
- 거트 : ITM 옵션 이용

**차익거래 전략**

- 컨버젼 (매수차익) : p + S < c + B → 콜옵션 프리미엄이 고평가
- 리버설 (매도차익) : p + S > c + B → 풋옵션 프리미엄이 고평가
- 포트폴리오보험 : 방어적 풋, 이자추출전략, 동적자산배분, 동적헤징전략

**3.4 옵션 지표**

**민감도 지표**

- 델타 : 옵션가격 변화 / 기초자산 변화
- 감마 : 옵션가격 변화 / 기초자산 변화 / 기초자산 변화 → ATM에서 최대 (만기일)
- 쎄타 : 옵션가격 변화 / 시간 변화
- 베가 : 옵션가격 변화 / 변동성의 변화
- 로우 : 옵션가격 변화 / 금리 변화

|  | 델타 | 감마 | 쎄타(시간) | 베가(변동성) | 로우 |
| --- | --- | --- | --- | --- | --- |
| 콜매수 | +  | +(매수) | - (매수) | +(옵션특) | + |
| 콜매도 | -  | - (매도) | + (매도) | - | + |
| 풋매수 | - | + (매수) | - (매수) | + (옵션특) | - |
| 풋매도 | + | - (매도) | + (매도) | -  | - |

* 매수와 매도는 반대 부호를 가진다 (로우 제외)

* 시간이 흐르고 변동성이 적어야 매도 포지션이 이익이다 (+)

## 04 파생결합증권평가, 분석 (5)

### 1. 파생결합증권의 개요

**정의 :** 기초자산을 기초로하는 지수와 연계되어 정해진 방법에 따라 가격이 결정되는 것

**종류** : ELS, DLS, ELW, ETN

**발행 :** 투자매매업 인가 후 공모/사모발행

일괄신고서

- 사전에 발행물량을 신고하는 것
- 금지대상 : 공모 고난도 파생결합증권
- 허용대상 : ELW, ETN, ELS 중 주식/지수 기반 손실배수 1 이하
- 규정 : 일괄신고시 3회 이상 발행해야 함 / 모든 종류 발행 시 자기자본 1400억 이상 필요

### 2. 주식워런트증권 (ELW)

**ELW (Equity-Linked warrant, 주식워런트증권)**

정의 : 기초자산을 정해진 가격에 사거나 팔 수 있는 권리가 부여된 파생상품적 권리가 증권처럼 거래되는 것

거래단위 : 10주

종류

- 콜 ELW : 만기에 기초자산을 행사가격으로 인수하거나 만기가-행사가 차액 수령 가능 → 비싸면 이익
- 풋 ELW : 만기에 기초자산을 행사가격으로 인도하거나 행사가-만기가 차액 수령 가능 → 낮으면 이익

특징 : 매수만 가능 (↔ 옵션)

가격 : 내재가치 + 시간가치

- 콜 : (기초자산가격 - 행사가격) x 전환비율
- 풋 : (행사가격 - 기초자산가격) x 전환비율
- 시간가치 : 만기일에 0으로 수렴

    지표

- 전환비율 : 만기에 ELW 1단위당 얻을 수 있는 기초자산 수
- 프리미엄 : 시간가치 / 기초자산가격
- 손익분기점 : 행사가격 +/- (ELW가격 / 전환비율)
- 자본지지점 : 기초자산과 ELW의 만기실현가치가 같아지는 기초자산의 연간 기대수익률

    $$
    CFP_{call} = \left( \left( \frac{X}{S_0 - ELW_0} \right)^{\frac{1}{T}} - 1 \right) \times 100
    $$

    $$
    CFP_{put} = \left( \left( \frac{X}{S_0 + ELW_0} \right)^{\frac{1}{T}} - 1 \right) \times 100
    $$

- 패리티 : 행사가격 vs 기초자산가격 → 1보다 크면 내가격

대상 : 국내 개별주 (코스피 50개 코스닥 5개), 국내 주가지수 (코스피, 코스닥), 해외 주가지수 (니케이, 홍콩)

(미국 주가지수 x)

규정 : 1500만원 기본예탁금 필요, 증권회사가 LP 공급

상폐 : 개별주 ELW의 주식이 상폐, 발생사가 ELW 전부 보유하고 상폐신청, 상습 규정위반

행사 : 만기일에 이익이 발생한다면 자동으로 권리행사, 현금결제방식만 적용됨

### 3. 주가연계증권 (ELS/DLS)

**ELS (Equity-Linked Security, 주가지수연계증권)**

의의 : 주가 지수와 연계된 조건부 수익형 채권

종류 : ELS는 주식/주가 기반, DLS는 이자율/원자재/귀금속/신용위험 등 기반

특징 : 다양한 지급구조와 기초자산을 가짐

**구조**

유러피안 구조

- bull-spread : 채권 + 낮은 콜 매수 + 높은 콜 매도 → 높은 행사가 초과 시 원금 + 고정금액
- reverse-convertible : 채권 + 풋 매도 → 일정 가격 이상이면 주식으로 돌아옴 (전환사채의 반대)

배리어 구조

- 만기까지 범위 내 - 원금 + 옵션, 한번이라도 범위 초과 - 원금만
- up and out cut with rebate : 배리어 초과 기준
- down and out put : 배리어 미만 기준
- 양방향 구조 : 양 배리어 기준

조기상환 구조

- 기본구조 : 만기를 2~3년으로 하고 6개월 단위로 평가, 만기를 늘리고 원금보장성을 약화
- step down 구조 : 조기상환시점마다 조기상환 기준을 일정 비율식 완화

기타

- knock-out : 목표 도달 시 조기상환 (채권 + 콜/풋 매수)
- digital : 만기시점에 기준점 이상/이하를 따져 지금 (채권 + 콜/풋 매수)

### 4. 상장지수증권 (ETN)

**ETN (Exchange Traded Note)**

의의 : 기초지수와 연동된 상장채권

대상 : 한국거래소 대표지수, 인정되는 외국거래소 지수, 주식기반 (국내 5개 해외 3개 이상), 채권기반 (3개 이상)

특징

- 추적오차의 위험이 적음, 신용위험 존재

제도

- 규모 : 발행액 70억 이상, 증권수 10만 이상, 자기자본 50%까지 가능
- 발행 : 공모발행, 간주모집 / LP에게 일괄배정 → 거래소로 매출
- 구성 : 국내주식이면 5종목, 해외주식이면 3종목 이상으로 구성해야 함
- 상장 : 15일간 심사
- 유동성 공급 : 발행회사 or 제3의 증권회사

진입기준

- 자기자본 5000억 이상, AA- 이상, 순자본비율 150% 이상, 감사의견 3사업연도 적정

퇴출기준

- 자기자본 2500억 미만, BBB- 미만, 순자본비율 (100% 3개월 or 50% 미만), 감사의견 부적정/의견거절

지표

- 일일 지표가치 : 1증권당 지표가치 매일 산출
- 실시간 지표가치 : 15초마다 갱신
- 괴리율 : 1 - (시장가/지표가치)

위험

- 발행회사 신용위험, 기초자산 하락위험, 유동성 부족위험, 단기거래 비용증가위험 (수수료), 상장폐지 위험
- 일별 복리화 효과 위험 (레버리지), 롤 오버 위험 (선물)
