- 장기차입금의 상환일이 1년 내로 다가온 경우, 장기차입금 → 유동성 장기부채

# 1편 회계의 첫걸음

### 1장 회계의 기본개념

**재무제표의 종류**

- 재무상태표 : 자산, 부채, 자본
- 손익계산서 : 수익, 비용
- 자본변동표 : 자본의 크기와 변동내역
- 현금흐름표 : 현금의 유입 유출
- 주석 : 추가적 설명

**01 재무상태표**

의의 : 현 시점 기업의 재무상태

등식 : 자산 = 부채 + 자본

원칙 : 구분표시원칙, 총액주의, 유동 기준은 1년, 유동성배열법, 잉여금구분 (자본,이익), 미결산 금지

- 자산 : 현금, 예금, 매출채권, 미수금, 대여금, 상품, 건물, 토지, 비품
- 부채 : 매입채무, 차입금, 선수금, 미지급금 (채권자 지분)
- 자본 : 자본금, 이익잉여금 (소유주 지분)

**02 손익계산서**

의의 : 일정 기간동안의 경영성과

등식 : 수익 - 비용 = 순이익 or 순손실 = 기말자본 - 기초자본 = 기말자산 - 기말부채

원칙 : 발생주의, 실현주의, 수익비용 대응의 원칙, 총액주의, 구분표시의 원칙

- 수익 : 매출액, 임대료수익, 이자수익
- 비용 : 매출원가, 급여, 광고선전비, 지급임차료, 감가상각비, 이자비용

**03 현금흐름표**

의의 : 일정 기간동안의 현금의 유입과 유출

구성 : 영업활동, 투자활동, 재무활동

작성

- 직접법 : 각각의 항목들을 나열
- 간접법 : 영업활동, 투자활동, 재무활동으로 나누어 표시

**04 자본변동표**

구성 : 자본금, 자본잉여금, 자본조정, 기타포괄손익누계액, 이익잉여금

### 2장 회계의 흐름

자산 | 부채 자본

비용 | 수익

![image.png](%ED%9A%8C%EA%B3%84%EA%B4%80%EB%A6%AC%202%EA%B8%89%20(v)/image%202.png)

**거래** : 금액으로 측정할 수 있는 재무상태에 영향을 미치는 행위

**계정** : 거래 발생 시 자/부/자의 증감 변화를 구체적으로 기록하는 행위

**분개** : 거래를 대변 항복으로 분류하고 기록하는 행위

**전기** : 분개 후 계정별로 총계정원장에 옮기는 절차

회계순환과정

(회계기간) 거래의 인식 → 분개장에 분개 → 총계정원장 전기 → 수정 전 시산표

(결산종료) 결산 → 수정 후 시산표 → 계정의 마감 → 재무제표

총계정원장 : 거래를 하나로 모아둔 장부

→ 현금 건물 급여 등 각 항목 각각 T표로 나타낸 것

합계잔액시산표 : 차변 대변의 합계액과 잔액을 모은 것 (자산+비용=부채+자본+수익)

→ 계정과목을 가운데에, 나머지 대변 차변의 값을 나열한 것

손익계산서 작성 → 이익잉여금 계산 → 재무상태표 작성

# 2편 계정과목 이해하기

### 1장 자산계정(1) - 유동자산

- 재무상태표 - 유동자산

**분류**

1. 당좌자산
    - 현금 및 현금성 자산 (통화, 통화대용증권 / 요구불예금, 현금성자산) - 현금출납장, 현금과부족, 소액현금
    - 단기투자자산 (단기금융상품 - CD, RP, 특수목적 예금, 단기대여금, 유가증권)
    - 매출채권 (외상매출금, 받을어음), 미수금 (미수채권)
    - 미수수익 / 선급금, 선급비용 / 이연법인세자산
2. 재고자산

**유가증권**

종류 : 지분증권, 채무증권 (국채, 공채, 사채) / 단기매매증권, 만기보유증권, 매도가능증권

회계 : 보유기간 중 받는 배당과 이자는 영업외수익 처리

| 단기매매증권 | 매도가능증권 | 만기보유증권 |
| --- | --- | --- |
| 당좌자산 | 투자자산 (비유동) | 투자자산 (비유동) |
| 공정가치 (시장 공시가) | 공정가치 or 취득원가 (시장성 x) | 상각후원가 |
| 단기매매증권처분손익 (영업외 손익) | 매도가능증권처분손익 | 보유중에는 이자비용만 인식 |
- 매도가능증권의 처분은 ‘평가이익’ ‘처분이익’을 언제 계상하느냐에 따라 다르다

**채권**

종류 : 매출채권 (상거래), 미수금 (상거래 외, 비품 파는 등)

대손상각 : 회수 불가능 수취채권

**재고자산**

종류 : 상품, 제품, 반제품, 재공품, 원재료, 저장품 / 미착상품, 시송품, 적송품

금액 : max(매입원가, 시가)

매입원가 = 매입가 + 매입 부대비용 - 매입할인/매입에누리/매입환출

회계 : 분할법 (이월상품 / 매입 / 매출)

수량 : 기초재고액 + 당기매입액 - 매출원가 = 기말재고액 ← 있던거 + 산거 - 판거 = 남은거

- 계속기록법 : 기초재고수량 + 당기매입수량 - 당기판매수량 = 기말재고수량 ’팔때마다 기록‘
- 실지재고조사법 : 기초재고수량 + 당기매입수량 - 기말실지재고수량 = 당기매출원가 ’끝나고 재고를 카운트‘
- 재고자산감모손실 : 장부상보다 실재재고가 적은 경우

단가 (변하는 경우 결정법)

- 개별법 : 각각의 자산별로 개별취득원가를 사용
- 선입선출법 : 선입 가격을 선출 가격으로 계산
- 후입선출법 : 최근 가격을 선출 가격으로 계산
- 평균법 : 평균가로 사용 (실지재고조사법 - 총평균법, 계속기록법 - 이동평균법)

재고자산의 손실 : 재고자산평가충당금 / 정상감모손실, 비정상감모손실

### 2장 자산계정(2) - 비유동자산

**투자자산**

- 1년 이상 보유할 투자 수익을 위해 보유중인 자산, 영업활동과 무관

종류 : 장기금융상품, 장기투자증권, 장기대여금, 투자부동산, 지분법적용투자주식

**유형자산**

- 1년 이상 사용할 물리적 형태가 있는 자산, 영업활동에 사용

종류 : 토지, 건물, 구축물, 기계장치, 선박, 차량운반구, 건설중인자산, 기타의 유형자산

회계 : 취득원가 + 취득부대비용 - 매입할인

→ 감가상각 : 감가상각대상금액(날라가는 비용)을 내용연수(쓰는 기간)동안 분배함

- 정액법  : n빵 / 정률법 : 비율로 / 연수합계법 / 생산량비례법

취득 후 : 수선비 (자본적지출, 수익적지출)

처분 시 : 유형자산처분손익 = 처분가액 - 장부금액(취득원가 - 감가상각누계액)

**무형자산**

- 물리적 형태는 없지만 식별가능 + 기업이 통제권 가짐 + 미래 경제적 효익 있음

종류 : 영업권, 산업재산권, 개발비

감가상각 : 정률법 사용, 최대 20년, 잔존가치 0으로 처리

**기타비유동자산**

- 분류가 모호한 것들

종류 : 임차보증금, 이연법인세자산, 장기매출채권, 장기미수금

### 3장 부채계정

**1. 유동부채**

- 기업의 영업주기 내에 상환하여 소멸이 예상되는 매입채무, 미지급비용
- 보고기간 종료일부터 상환일이 1년 이내인 부채
- 보고기간 후 1년 이상 결제를 연기할 수 없는 부채

종류 : 단기차입금, 매입채무, 미지급금, 미지급비용, 이연법인세부채, 기타

**매입채무 :** 일반적 상거래에서 발생한 외상매입금, 지급어음 (외상매입금 - 약속, 지급어음 - 어음을 줌)

**미지급금** : 일반적 상거래 외에 발생한 채무 (유형자산 취득 등)

**차입금** : 1년 이내의 상환 - 단기차입금

**선수금** : 상거래에서 대금을 미리 받은 것 → 상품 인도의 의무 발생 (부채)

**예수금** : 상거래 외에서 일시적으로 현금을 수취한 것

미지급비용 / 선수수익 / 가수금

**2. 비유동부채**

- 보고기간종료일부터 1년 이내에 상환되지 않는 부채

종류 : 장기차입금, 사채, 퇴직급여충당부채

**장기차입금** : ↔ 단기차입급

**사채** : 발행한 회사채, 조기상환 시 사채상환손익 처리

**퇴직급여충당부채** : 충당부채 = 현재의무 + 자원필요 + 소요금액 추정가능한 대상

퇴급충부 → 퇴직급여로 나갈 돈을 나누어 회계반영

- 확정급여형 : 납입액 → 자산으로 회계처리
- 확정기여형 : 납입액 → 비용으로 회계처리

### 4장 자본계정

- 자산에서 부채가 빠진 것, 소유주 지분

단, 자산 처분 시 채권자가 우선적 청구권을 가진다

**분류**

- 자본금
- 자본잉여금
- 자본조정
- 기타포괄손익누계액
- 이익잉여금

**자본금**

- 주주의 지분
- 자본금 = 발행주식수 x 1주당 액면가액
- 할증/할인 발행 시 주식발행초과금 (**자본잉여금**), 주식할인발행자금(**자본조정**)으로 처리

**자본잉여금**

- 자본거래에서 발생한 이익

종류 : 주식발행초과금, 감자차익, 자기주식처분이익

**자본조정**

- 임시적인 자본 항목

종류 : 주식할인발행자금, 감자차손, 자기주식처분손실, 자기주식취득

**기타포괄손익누계액**

- 당기손익은 아니지만 자본에 들어가는 평가손익

종류 : 매도가능증권평가손익 (미실현보유손익)

**이익잉여금**

- 손익거래에서 발생한 이익 (=유보이익)

이익잉여금처분계산서

- 미처분이익잉여금 : 이월된 잉여금 + 당기순이익
- 임의적립금 등의 이입액

→ 합산 →

- 이익준비금 : 법에 의해 강제로 적립해야 하는 금액
- 임의적립금 : 목적을 가지고 회사가 적립하는 금액
- 배당 (현금배당, 주식배당)
- 차기이월미처분이익잉여금

### 5장 수익과 비용

수익 : 매출액, 영업외수익 - 판매기준, 진행기준

비용 : 매출원가, 판매비와 관리비, 영업 외 비용, 법인세 비용 - 직접대응, 기간배분, 기간대응

매출액 = 총매출 - 매출에누리 - 매출환입 - 매출할인

- 매출에누리 (판매대금 감면), 매출환입 (반품), 매출할인 (약정비율로 감면)

매출원가 = 기초상품재고액 + 당기매입액 - 기말상품재고액

매출총이익 = 매출액 - 매출원가

영업이익 = 매출총이익 - 판매관리비

- 판매비와관리비 : **급여, 퇴직급여, 복리후생비, 여비교통비, 통신비, 세금과공과금, 감가상각비, 임차료**

                              **수선비, 차량유지비, 경상연구개발비, 대손상각비, 수수료비용**


법인세비용차감전순손익 = 영업이익 + 영업외수익 - 영업외비용

- 영업외수익 : 이자수익, 배당금수익, 임대료, 유가증권/투자자산/유형자산처분이익, 단기매매증권평가이익

                       외환차익, 외환환산이익, 사채상환이익, 자산수증이익, 채무면제이익, 잡이익

- 영업외비용 : 이자비용, 기타의대손상각비, 유가증권처분손실, 단기매매증권평가손실, 재고자산감모손실,

                       외환차손, 외화환산손실, 기부금, 투자자산/유형자산처분손실, 사채상환손실

# 3편 결산과 재무재표

### 1장 결산

**결산수정사항 :** 손익 귀속시기 결정, 자산/부채 평가

**손익 귀속시기 결정**

- 발생과 기록 시기가 다른 것들을 수정

수익의 발생 (미수수익) : 받을 예정 - 자산

수익의 이연 (선수수익) : 미리 받음 - 부채

비용의 발생 (미지급비용) : 지불 예정 - 부채

비용의 이연 (선급비용) : 미리 지불 - 자산

**자산/부채 평가**

- 매출의 귀속시기 = 판매시점

매출채권 : 외상매출액, 받을어음, 대손충당금 계상

재고자산 : 재고 실사평가 (진부화), 재고자산감모손익, 기간귀속

외회자산 : 결산일 환율 반영, 외화환산손익 평가

유가증권 : 위탁된 증권의 잔고증명서 확인하여 대조

기타

- 감가상각
- 퇴직급여충당부채 : 퇴직금 계산
- 현금및현금성자산 : 시재액 점검, 잡손이익으로 처리
- 예금 : 기록과 대조, 계정분류

**결산절차**

예비절차 : 총계정원장에 전기, 계정별 잔액을 수정전 시산표에 정리

수정분개 : 시산표로 결산수정분개 시행, 수정후 시산표 작성

손익계산서 계정 마감 : 집합손익 계정 대체 → 이익잉여금 계정 대체

(비용계정 + 수익계정의 net만 자본계정의 이익잉여금으로)

재무상태표 계정 마감 : 총계정원장에서 각 잔액을 차기이월 항목으로 기입 → 마감분개 → 전기이월 (다음 년도)

### 2장 회계변경과 오류수정

**회계변경**

- 적절한 회계처리를 위해 회계처리방법을 바꾸는 것

종류 : 회계정책 변경, 회계추정 변경

방법 : 소급법, 당기일괄 처리법, 전진법

정책 - 소급법, 추정 - 전진법

**오류수정**

- 작성 후 발견된 오류를 수정하는 것

종류

- 순이익 동일 : 재무상태표 영향, 손익계산서 영향 ← 계정항목 분류 오류
- 순이익 변경 : 자동조정적 오류, 비자동조정적 오류

### 3장 재무제표 보는 법

**재무상태표**

유동성분석 : 유동비율, 현금비율, 당좌비율

안정성분석 : 부채비율, 자기자본비율

매출채권 분석 : 매출채권 회수율, 매출채권 회수기간

재고자산 분석 : 재고자산 회전율, 재고자산 평균회전기간

유형자산 분석 : 총자산회전율, 비유동자산회전율, 유형자산회전율

**손익계산서**

비용분석 : 판매비와 관리비 증감

이익분석 : 성장성 분석 (매출액 증가율, 순이익 증가율)

수익성 분석 : 매출총/영업/당기순이익률, 주당순이익, 자기자본이익률

**회계감사**

적정의견 - 한정의견 - 부적정의견 - 의견거절
