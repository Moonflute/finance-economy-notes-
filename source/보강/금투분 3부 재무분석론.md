# III. 재무분석론

소유자: 조동희 | 서울 의학과 ­

## 01 재무제표론 (10)

### 1. 재무회계의 개요

**회계의 종류**

재무회계 : 외부보고 목적

관리회계 : 내부보고 목적

세무회계 : 과세소득 목적

**회계 기준**

- 한국채택국제회계기준 (K-IFRS) : 상장기업 대상
- 일반기업회계기준 : 비상장기업 대상

**재무제표의 종류**

- 재무상태표 : 자산, 부채, 자본
- 손익계산서 : 수익, 비용
- 자본변동표 : 자본의 크기와 변동내역
- 현금흐름표 : 현금의 유입 유출
- 주석 : 추가적 설명

### 2. 재무제표에 대한 이해

**01 재무상태표**

의의 : 현 시점 기업의 재무상태

등식 : 자산 = 부채 + 자본

원칙 : 구분표시원칙, 총액주의, 유동 기준은 1년, 유동성배열법, 잉여금구분 (자본,이익), 미결산 금지

- 자산 : 현금, 예금, 매출채권, 미수금, 대여금, 상품, 건물, 토지, 비품
- 부채 : 매입채무, 차입금, 선수금, 미지급금 (채권자 지분)
- 자본 : 자본금, 이익잉여금 (소유주 지분)

**02 손익계산서**

의의 : 일정 기간동안의 경영성과

등식 : 수익 - 비용 = 순이익 or 순손실 = 기말자본 - 기초자본 = 기말자산 - 기말부채

원칙 : 발생주의, 실현주의, 수익비용 대응의 원칙, 총액주의, 구분표시의 원칙

- 수익 : 매출액, 임대료수익, 이자수익
- 비용 : 매출원가, 급여, 광고선전비, 지급임차료, 감가상각비, 이자비용

양식 : 성격별 분류, 기능별 분류

**회계 흐름**

자산 | 부채 자본

비용 | 수익

![image.png](III%20%EC%9E%AC%EB%AC%B4%EB%B6%84%EC%84%9D%EB%A1%A0/image%201.png)

**거래** : 금액으로 측정할 수 있는 재무상태에 영향을 미치는 행위

**계정** : 거래 발생 시 자/부/자의 증감 변화를 구체적으로 기록하는 행위

**분개** : 거래를 대변 항복으로 분류하고 기록하는 행위

**전기** : 분개 후 계정별로 총계정원장에 옮기는 절차

회계순환과정

(회계기간) 거래의 인식 → 분개장에 분개 → 총계정원장 전기 → 수정 전 시산표

(결산종료) 결산 → 수정 후 시산표 → 계정의 마감 → 재무제표

총계정원장 : 거래를 하나로 모아둔 장부

→ 현금 건물 급여 등 각 항목 각각 T표로 나타낸 것

합계잔액시산표 : 차변 대변의 합계액과 잔액을 모은 것 (자산+비용=부채+자본+수익)

→ 계정과목을 가운데에, 나머지 대변 차변의 값을 나열한 것

손익계산서 작성 → 이익잉여금 계산 → 재무상태표 작성

### 3. 재무상태표론

**3.1 자산계정(1) - 유동자산**

- 재무상태표 - 유동자산

**분류**

1. 당좌자산
    - 현금 및 현금성 자산 (통화, 통화대용증권 / 요구불예금, 현금성자산) - 현금출납장, 현금과부족, 소액현금
    - 단기투자자산 (단기금융상품 - CD, RP, 특수목적 예금, 단기대여금, 유가증권)
    - 매출채권 (외상매출금, 받을어음), 미수금 (미수채권)
    - 미수수익 / 선급금, 선급비용 / 이연법인세자산
2. 재고자산

**유가증권**

종류 : 지분증권, 채무증권 (국채, 공채, 사채) / 단기매매증권, 만기보유증권, 매도가능증권

회계 : 보유기간 중 받는 배당과 이자는 영업외수익 처리

| 단기매매증권 | 매도가능증권 | 만기보유증권 |
| --- | --- | --- |
| 당좌자산 | 투자자산 (비유동) | 투자자산 (비유동) |
| 공정가치 (시장 공시가) | 공정가치 or 취득원가 (시장성 x) | 상각후원가 |
| 단기매매증권처분손익 (영업외 손익) | 매도가능증권처분손익 | 보유중에는 이자비용만 인식 |
- 매도가능증권의 처분은 ‘평가이익’ ‘처분이익’을 언제 계상하느냐에 따라 다르다

**채권**

종류 : 매출채권 (상거래), 미수금 (상거래 외, 비품 파는 등)

대손상각 : 회수 불가능 수취채권

**재고자산**

종류 : 상품, 제품, 반제품, 재공품, 원재료, 저장품 / 미착상품, 시송품, 적송품

금액 : max(매입원가, 시가)

매입원가 = 매입가 + 매입 부대비용 - 매입할인/매입에누리/매입환출

회계 : 분할법 (이월상품 / 매입 / 매출)

수량 : 기초재고액 + 당기매입액 - 매출원가 = 기말재고액 ← 있던거 + 산거 - 판거 = 남은거

- 계속기록법 : 기초재고수량 + 당기매입수량 - 당기판매수량 = 기말재고수량 ’팔때마다 기록‘
- 실지재고조사법 : 기초재고수량 + 당기매입수량 - 기말실지재고수량 = 당기매출원가 ’끝나고 재고를 카운트‘
- 재고자산감모손실 : 장부상보다 실재재고가 적은 경우

단가 (변하는 경우 결정법)

- 개별법 : 각각의 자산별로 개별취득원가를 사용
- 선입선출법 : 선입 가격을 선출 가격으로 계산
- 후입선출법 : 최근 가격을 선출 가격으로 계산 (K-IFRS에서는 인정 안됨)
- 평균법 : 평균가로 사용 (실지재고조사법 - 총평균법, 계속기록법 - 이동평균법)

재고자산의 손실 : 재고자산평가충당금 / 정상감모손실, 비정상감모손실

**3.2 자산계정(2) - 비유동자산**

**투자자산**

- 1년 이상 보유할 투자 수익을 위해 보유중인 자산, 영업활동과 무관

종류 : 장기금융상품, 장기투자증권, 장기대여금, 투자부동산, 지분법적용투자주식

**유형자산**

- 1년 이상 사용할 물리적 형태가 있는 자산, 영업활동에 사용

종류 : 토지, 건물, 구축물, 기계장치, 선박, 차량운반구, 건설중인자산, 기타의 유형자산

회계 : 취득원가 + 취득부대비용 - 매입할인

→ 감가상각 : 감가상각대상금액(날라가는 비용)을 내용연수(쓰는 기간)동안 분배함

- 정액법  : n빵
- 정률법 : 비율로
- 생산량비례법 : (기초가-잔존가) / (실생산/추정총생산)

취득 후 : 수선비 (자본적지출, 수익적지출)

처분 시 : 유형자산처분손익 = 처분가액 - 장부금액(취득원가 - 감가상각누계액)

**무형자산**

- 물리적 형태는 없지만 식별가능 + 기업이 통제권 가짐 + 미래 경제적 효익 있음

종류 : 영업권, 산업재산권, 개발비

원가 : 구입가격 + 부대비용

감가상각 : 정률법 사용, 최대 20년, 잔존가치 0으로 처리

* 연구단계는 비용처리 개발단계는 무형자산의 취득원가로 처리

**기타비유동자산**

- 분류가 모호한 것들

종류 : 임차보증금, 이연법인세자산, 장기매출채권, 장기미수금

* 자산손상

자산이 회수가능액이 크게 미달되는 경우

징후 : 장부금액 >> 시가총액

차손 : 장부금액 - 회수가능액

환입 : 인식 전 장부금액 감가상각 후 잔액보다 작음, 영업권은 환입 불가

* 차입원가 자본화

적격자산의 취득을 위해 자금을 차입한 경우 (특정차입금과 일반차입금 구분)

**3.3 부채계정**

**1. 유동부채**

- 기업의 영업주기 내에 상환하여 소멸이 예상되는 매입채무, 미지급비용
- 보고기간 종료일부터 상환일이 1년 이내인 부채
- 보고기간 후 1년 이상 결제를 연기할 수 없는 부채

종류 : 단기차입금, 매입채무, 미지급금, 미지급비용, 이연법인세부채, 기타

**매입채무 :** 일반적 상거래에서 발생한 외상매입금, 지급어음 (외상매입금 - 약속, 지급어음 - 어음을 줌)

**미지급금** : 일반적 상거래 외에 발생한 채무 (유형자산 취득 등)

**차입금** : 1년 이내의 상환 - 단기차입금

**선수금** : 상거래에서 대금을 미리 받은 것 → 상품 인도의 의무 발생 (부채)

**예수금** : 상거래 외에서 일시적으로 현금을 수취한 것

미지급비용 / 선수수익 / 가수금

**2. 비유동부채**

- 보고기간종료일부터 1년 이내에 상환되지 않는 부채

종류 : 장기차입금, 사채, 퇴직급여충당부채

**장기차입금** : ↔ 단기차입급

**사채** : 발행한 회사채, 조기상환 시 사채상환손익 처리

**퇴직급여충당부채** : 충당부채 = 현재의무 + 자원필요 + 소요금액 추정가능한 대상

퇴급충부 → 퇴직급여로 나갈 돈을 나누어 회계반영

- 확정급여형 (DB) : 납입액 → 자산으로 회계처리
- 확정기여형 (DC) : 납입액 → 비용으로 회계처리

**3.4 자본계정**

- 자산에서 부채가 빠진 것, 소유주 지분

단, 자산 처분 시 채권자가 우선적 청구권을 가진다

**분류**

- 자본금
- 자본잉여금
- 자본조정
- 기타포괄손익누계액
- 이익잉여금

**자본금**

- 주주의 지분
- 자본금 = 발행주식수 x 1주당 액면가액
- 할증/할인 발행 시 주식발행초과금 (**자본잉여금**), 주식할인발행자금(**자본조정**)으로 처리

**자본잉여금**

- 자본거래에서 발생한 이익

종류 : 주식발행초과금, 감자차익, 자기주식처분이익

**자본조정**

- 임시적인 자본 항목

종류 : 주식할인발행자금, 감자차손, 자기주식처분손실, 자기주식취득

**기타포괄손익누계액**

- 당기손익은 아니지만 자본에 들어가는 평가손익

종류 : 매도가능증권평가손익 (미실현보유손익)

**이익잉여금**

- 손익거래에서 발생한 이익 (=유보이익)

이익잉여금처분계산서

- 미처분이익잉여금 : 이월된 잉여금 + 당기순이익
- 임의적립금 등의 이입액

→ 합산 →

- 이익준비금 : 법에 의해 강제로 적립해야 하는 금액
- 임의적립금 : 목적을 가지고 회사가 적립하는 금액
- 배당 (현금배당, 주식배당)
- 차기이월미처분이익잉여금

### 4 금융회계

**4.1 금융상품**

**금융상품 : 금융자산 & 금융부채**

금융자산 : 현금, 지분상품, 계약상 권리, 계약

- 권리 : 금융자산을 수취할 권리, 유리한 조건으로 교환하기로 한 권리
- 계약 : 수취할 자기지분상품 수량이 변동 가능한 비파생상품, 수량은 확정이나 결제금액이 확정이 아닌 파생상품 (풋 제외)

금융부채 : 계약상 의무, 계약

- 의무 : 금융자산을 인도할 의무, 불리한 조건으로 교환하기로 한 의무
- 계약 : 인도할 자기지분상품 수량이 변동 가능한 비파생상품, 수량은 확정이나 결제금액이 확정이 아닌 파생상품 (풋 제외)

복합자산 : 자본요소 + 부채요소인 비파생상품

- ex) 전환사채, 신주인수권부사채

**금융자산의 분류 (K-IFRS 1109)**

1. 상각후원가 측정 금융자산 (AC)
2. 기타포괄손익-공정가치 측정 금융자산 (FVOCI)
3. 당기손익-공정가치 측정 금융자산 (FVPL)

### 7. 연결재무제표

의의 : 지배기업의 종속기업, 관계기업, 공동지배기업을 포함하여 작성함

면제 : 소유주들이 동의한 경우, 공개되지 않은 시장에서 거래되는 경우

**7.1 종속기업**

**‘지배’ 기준**

- 피투자자에 대한 힘 (활동을 지시할 수 있음)
- 피투자자에 대해 관여하여 변동이익에 대한 권리를 갖는 경우 (양과 부 전부)
- 투자자의 이익을 위해 피투자자에게 힘을 사용할 수 있는 경우

**작성 절차**

- 재무제표 합산 : 동일 보고기간, 동일 회계정책 사용, 화폐 보정 (지배기업 통화로)
- 투자자본 상계 : 중복항목은 지배기업의 지분을 제거함 (종속기업에 대한 투자금 항목)
- 내부거래 제거 : 서로 계상한 채권, 채무, 수익, 비용은 상계하여 제거함 (과대계상 방지)
- 비지배지분 표시 : 종속기업의 지분 중 지배기업 귀속이 아닌 지분

**지배력 변동**

지분 처분 : 자본거래 회계처리

지배력 상실 : 모든 차이는 당기손익으로 처리

**사업결합 회계처리 (지배력 획득)**

영업권 : 비싸게 삼 (순자산 < 인수금액)

염가매수차익 : 싸게 삼 (순자산 > 인수금액)

**7.2 관계기업**

정의 : 기업이 피투자자에 대한 의결권의 20% 이상을 소유한 경우

회계처리

최초 취득 시 : 취득원가로 처리

취득 시점 후

- 자본 : 기타포괄손익누계액
- 배당 : 투자자산의 장부가액 차감
- 미실현 손익 : 투자주식을 차감

- 공정가치 : 측정일 환율
- 자산과 부채 : 보고기간 마감환율
- 수익과 비용 : 거래일 환산율

외화거래 : 화폐성 항목은 마감 환율 반영, 비화폐성 항목은 거래일 환율 환산, 공정가치는 측정일로 환산

환산손익 : 회계기간 손익으로 인식, 매도가능증권은 기타포괄손익으로 인식

재무재표 환산 : 재무상태표의 자산과 부채는 마감환율, 포괄손익계산서의 수익과 비용은 거래일 환율

→ 발생하는 외환차이는 기타포괄손익으로 인식

## 02 기업가치평가, 분석 (10)

### 1. 기업가치평가 개요

**1.1 기업가치평가**

의의 : 기업의 미래수익 창출력, 채권자가치 + 주주지분가치

목적 : 투자의사결정, IPO, 과세목적, 회계결산

방법

- 자산가치 : 청산가치, 장부가치, 시장평가가치가싲시ㅅ
- 수익가치 : DCF, EVA 할인모형, 배당할인모형ㅁ
- 상대가치 : PER, EV/EBITDA, PSR, PBR
- 법률적 : 상속세 및 증여세법, 증권 발행 및 공시규정저식발ㅈ미ㅁ

### 2. 상대가치평가법

**상대가치평가법**

| PER | EV/EBITDA | PBR | PSR |
| --- | --- | --- | --- |
| 주당 시총 | 시가총액 + 순차입금 | 주당 시총 | 주당 시총 |
| 주당 순이익 | 영업이익 + 감가상각비 | 주당 순자산 | 주당 매출액 |
| 순이익 대비 시가총액 | 기업의 수익창출력 | 순자산 대비 시가총액 | 매출액 대비 시가총액 |
| 적용이 간단 | 왜곡을 방지 | 적자에도 사용가능 | 적자인 경우가 없음 |
| 적자에 사용불가 | EBITDA 적자에 사용불가 | 자본잠식, 업종차 미반영 | 비용구조, 재무구조 미반영 |

* EV : 시총+총차입금-현금자산, 회사를 사려면 필요한 돈,

→ 시총은 지분가치만 반영하므로 부채를 포함, 회사가 가진 현금으로 부채를 상쇄하므로 현금자산은 빼줌

* EBITDA : 영업이익 + 감가상각비 → 실제 현금흐름에 가까운 분석을 위해 보정

→ 회사를 산다면, 기업에 버는 돈으로 얼마만에 회수가 가능한가

**유사회사**

의의 : 상대가치평가법으로 정적주가를 산출할 때 사용되는 기준값 회사

선정 : 사업의 유사, 경영성과, 재무사항 유사, 일반기기준 유사, 이해관계적 적정성

상대가치평가법의 장단점

### 3. 현금흐름할인법

**현금흐름할인법**

의의 : 기업이 창출할 미래의 현금흐름의 현재가치를 계산

기업가치 = 주주가치 + 채권자가치 = 영업가치 + 비영업가치

**계산식**

$$
\text{DCF} = \sum_{t=1}^{n} \frac{FCF_t}{(1 + WACC)^t} + \frac{TV}{(1 + WACC)^n}
$$

**FCF (free cash flow, 자유현금흐름)**

FCFF (free cash flow to the firm, 기업자유현금흐름, 잉여현금흐름)

= 기업의 영업활동으로부터 창출되는 현금흐름, (+)은 투자자에게 환원됨을 의미

- (영업이익 - 법인세) + 감가상각비 - 자본적 지출 - 운전자본 변동

FCFE (free cash flow to equity, 자기자본자유현금흐름)

- FCFF에서 부채가 상환된 후의 현금흐름
- FCFF - 이자 x (1 - 세율) + 순부채 증가

K : 자본비용, R : 비중

$$
\text{WACC} = K_{\text{타인}} \times R_{\text{타인}} \times t^* + K_{\text{자기}} \times R_{\text{자기}}
$$

$$
K_{\text{자기}} = R_f + \beta \times \left[ E(R_m) - R_f \right]
$$

$$
K_{타인}=평균차입현행이자율  (현행이자율)
$$

WACC : 요구되는 수익률

> FCFF 기간별로 산출 → 특정 시점까지의 FCFF의 현재가치 + 특정 시점의 잔존가치의 현재가치 = 영업가치
>

**하마다 모델**

의의 : 재무 레버리지가 beta에 미치는 영향 설명

위험 = 영업위험 + 재무위험

계산 : 부채포함 베타 = 무부채 베타 x (1 + (1-법인세율) 부채/자기자본 )

대상 : 비상장 기업

**초과이익모형 (RIM)**

- 주주관점에서의 기업가치평가 ( ↔ EVA : 주주 및 채권자 관점 )

기업가치 = 현재 자기자본 장부가치 + sum(현재가치)

초과이익 = 당기순이익 - (기초자기자본 x 자기자본비용)

### 4. EVA에 의한 가치평가

**EVA**

의의 : 자본비용을 초과하여 경제적 이익을 얼마나 창출했는가  → 세후 영업이익 - 들어간 자본비용

계산 : 세후영업이익 - WACC x 자본비용(투하자본)

* 자본비용 = 총자산 - 비영업자산 - 비이자부 유동부채 = 운전자본 + 비유동영업자산

**MVA**

의의 : EVA의 총합

계산 : EVA / WACC

기업가치 : 영업투하자본 + 비영업자산 + MVA

단점 : 감가상각의 영향이 큼 (과대평가), 기업규모의 영향을 받음

### 5. 공모주식 가치평가

공모가 산정법 : 자율

### 6. 규정에 의한 가치평가

합병가액 산정

**본질가치 산정**

의의 : 유사상장회사가 없는 경우의 기업가치 평가법 (있으면 PER등으로 비교)

계산 : 자산가치 + 수익가치 ← 2:3 비율로 합함

- 자산가치 = 조정 후 순자산가액 / 총 발행 주식수
- 수익가치 = 가중평균 주당추정이익 / 자본환원율 (=Max(차입금 가중평균 이자율 x 1.5, 세법상 할인율)

    단점 : 이론적 근거 없음, 자산공정가치 적용안됨, 성장성 반영안됨, 자본환원율 이론적 근거없음


**세법 가치평가**

상장주 : 평가기준일 전 후 2개월 종가평균

비상장 : 순손익가치 순자산가치 3:2로 가중평균

영업투하자본(IC) = 총자산 - 영업부채 - 비영업자산

세후 영업이익 (NOPLAT) = 영업이익 x (1 - 세율)

투하자본수익률 (ROIC) = 세후 영업이익 / 영업투하자본

EVA = NOPLAT - IC x WACC = IC x (ROIC - WACC)

MVA = EVA / WACC = 시가 - 투자자본

- EVA : 한 해의 경제적 부가가치
- MVA : 앞으로 영구히 발생할 EVA의 합

초과이익모형

- 초과이익 = 당기순이익 - 자기자본비용
