### 02 가치평가

가치평가 : 미래현금흐름 (S)를 적당한 할인률 (r)로 기간 (n)만큼 할인

미래가치

$$
 P = {S}{(1+r)^n}
$$

현재가치

$$
 P = \frac{S}{(1+r)^n}
$$

일반주 가치평가 (성장모형)

$$
P=\frac{D}{k}, \frac{D_1}{k-g}=\frac{1-b}{k-b \cdot ROC}
$$

- D = 배당금, k = 요구수익률, g = 배당성장률

요구수익률 = 무위험 + (시장수익-무위험) x beta

배당 수익률  = 주당배당 / 주가

배당 성향 = 주당배당 / 주당순이익 = 1- b (유보율)

배당성장률 = 사내유보율 x 자기자본이익률

### 03 재무분석

**지표 종류**

- 유동성지표 : 유동비율, 당좌비율, 현금비율
- 안정성지표 : 부채비율, 부채-자기자본비율
- 수익성지표 : 매출액영업이익률, ROA, ROE
- 활동성지표 : 비유동자산회전율, 재고자산회전율, 매출채권회전율, 총자산회전율
- 보상비율 : 배당성향, 이자보상비율, 고정비용보상비율

**A. 유동성비율 (CAR, CR, QR) - 단기부채 상환력**

- 현금비율 = (현금 + 시장성 유가증권) / 유동부채 → 현금보유 상태
- 유동비율 = 유동자산 / 유동부채 → 단기부채 상환능력
- 당좌비율 = 당좌자산 / 유동부채 → 일시적 재무위기 시 현금동원능력

**B. 안정성지표 (DR, DER) - 중장기부채 상환력**

- 부채비율 = 총부채 / 총자산 → 재무레버리지의 크기
- 부채-자기자본비율 = 총부채 / 자기자본 → 자본에 대한 레버지리의 크기
- 이자보상비율 = 영업이익 / 이자비용

**C. 수익성지표 (OPM, ROA, ROE)**

- 매출액영업이익률 = 영업이익 / 순매출액 → 영업효율성
- 총자산이익률 = 순이익 / 총자산 → 자산에서 이익을 뽑아내는 효율
- 자기자본이익률 = 순이익 / 자기자본 → 자기자본에서 이익을 뽑아내는 효율

**D. 활동성지표**

- 비유동자산회전율 = 순매출 / 비유동자산
- 재고자산회전율 = 순매출 / 재고자산
- 매출채권회전율 = 순매출 / 순매출채권
- 총자산회전율 = 순매출 / 총자산

**E. 보상비율 - 재무 이행능력**

- 배당성향 = 배당금 / 순이익
- 고정이자보상비율 = 영업이익 / 고정이자비율

**이익지표**

- 주당순이익 (EPS) = (당기순이익-우선주배당금) / 보통주 발행총수

    → 1주당 분배될 수 있는 순이익


### 04 레버리지 분석

고정비로 인해 손익이 확대되어 나타나는 효과 (DOL, DFL, DCL)

매출액 : 기업의 총 수익

영업이익 : 매출액 - 원가

순이익 : 영업이익 - 이자

- 영업레버리지 = 영업이익변화율 / 매출액변화율 = 매출 - 변동 / 매출 - (변동 + 고정)
- 재무레버리지 = 주당순이익변화율 / 영업이익변화율 = 매출 - (변동 + 고정) / 매출 - (변동 + 고정 + 이자)
- 결합레버리지 = 주당순이익변화율 / 매출액변화율 = 매출 - 변동 / 매출 - (변동 + 고정 + 이자)

    → DOL x DFL = DCL


* 이자비용 = 재무고정비

의의

영업레버리지 : 고정비로 증가된 영업이익을 봄

고정레버리지 : 차입금으로 증가된 순이익을 봄

### **05 현금흐름분석**

현금 = 보유현금 + 요구불예금

A. 영업활동으로 인한 현금흐름

당기순이익에서

- 현금유출이 없는 비용을 더하고 유입이 없는 수익을 뺌 (감가상각비, 유가증권평가이익)
- 투자활동으로 인한 현금흐름을 보정 (설비자산, 유가증권, 대여금)
- 재무활동으로 인한 현금흐름을 보정 (차입금, 자기주식, 유상증자)
- 영업활동으로 인한 현금흐름을 보정 (매출채권, 재고자산, 매입채무)

* 매출채권 : 아직 받지 못한 대금,  매입채무 : 아직 지불하지 않은 대금

B. 투자활동으로 인한 현금흐름

(+) 설비자산의 처분, 유가증권의 처분, 대여금 회수

(-) 설비자산의 취득, 유가증권의 매입, 대여금 대여

C. 재무활동으로 인한 현금흐름

(+) 차입금 차입, 자기주식 처분, 유상증자

(-) 차입금 상환, 자기주식 취득, 사채 발행비용

**주가에 따른 상대가치평가 (PER, PBR, Q)**

의의 : 주가가 높은지 낮은지를 판단하는 지표

- PER (주가이익비율) = 주가 / 주당순이익 → 이익 1단위당 얼마인가
- PBR (주가순자산비율) = 주가 / 주당순자산 → 자본 1단위당 얼마인가
- 토빈의 Q = 시가총액 / 자본의 대체원가 → 자본 1단위당 얼마인가
- EV / EBITDA 비율 → PER을 보완 (시가총액 + 순차입금 / 영업이익 + 감가상각비)

### 06 경영성과지표

**EVA (경제적 부가가치)**

- 세후 영업이익 - 투하자본 x 가중평균자본비용(WACC)

**MVA (시장부가가치)**

- 미래의 경제적 이익을 자본비용으로 할인한 현가의 합

**ROIC (투하자본수익률) :  세**후영업이익 / 투하자본

**자기자본이익률(ROE) :**  순이익 / 자기자본

**매출액영업이익률 :** 영업이익 / 매출액

## 4. 기술적 분석 (4~5)

### 00 기술적 분석

- 수요와 공급에 의해서만 결정된다는 가정
- 종류 : 추세분석, 패턴분석, 지표분석, 시장 구조 이론
- 다우이론 : 강세 (매집 → 마크업 → 과열) → 약세 (분산 → 공황 → 침체)

    강제장 약세장 3국면에서 대중과 전문가가 투자전략을 바꿈


### 01 추세 분석

**추세선**

- 상승추세선 : 저점을 연결
- 하락추세선 : 고점을 연결
- 추세곡선 : 주가의 움직임이 급격해짐 (부채형 추세곡선 - 약화)

**이동평균선 (5일, 20일, 60일, 120일)**

- 뱡향성 분석 : 자체의 방향성을 추세로 봄
- 배열도 분석 : 주가 > 단기 > 중기 > 장기 배열이 상승에너지가 강함
- 지지선/저항선 분석 : 저점을 연결 / 고점을 연결
- 크로스 분석 : 단기 MA가 중장기 MA 돌파 시 골든/데드크로스로 분석
- 이격도 분석 : 주가와 이평선의 괴리로 과열/침체 판단
- 밀집도 분석 : 이동평균선이 밀집되면 큰 변동성이 예상됨

**갭 (주가 사이 공간)**

- 보통 갭 : 횡보국면에서 나타남
- 돌파 갭 : 장기 조정국면에서 나타남, 추세변화 의미
- 급진 갭 : 거의 일직선으로 급상승, 급하락 시 나타남
- 소멸 갭 : 상승을 멈출 때 나타남
- 섬꼴 반전 : 상승이 끝나고 하락이 시작함을 의미

### 02 패턴 분석

- 반전형 패턴 : 헤드앤숄더, 역헤드앤숄더, 이중형 (M, W), 선형, 원형 바닥형, 천장형, 확대형
- 지속형 패턴 : 삼각형, 직사각형, 깃발형, 쐐기형, 다이아몬드형

**캔들 차트 분석**

- 한 개 : 우산형 (망치, 교수), 샅바형 (상승, 하락), 유성형, 해머형
- 두 개 : 장악형 (상승, 하락), 먹구름형, 관통형, 잉태형 (상승, 하락)
- 세 개 : 별 형 (샛별형, 석별형), 까마귀형

**사께다 전법 :** 삼공, 삼병, 삼산, 삼천, 삼법

### 03 지표 분석

**거래량 지표**

- OBV : 전일 대비 (상승한 날 거래량 합 - 하락한 날 거래량 합) - 횡보국면에서 주로 사용
- VR : (상승 or 보합일 거래량 합) / (하락 or 보합일 거래량 합), (70% - 150% - 450%)
- 역시계 : 주가 - 거래량의 관계는 8국면으로

**추세추종형 지표**

- MAO : 단기 이동평균 - 장기 이동평균
- MACD : 단기지수 이동평균 - 장기지수 이동평균, 시그널과 비교
- Envelope : 이평선 중심 플마 퍼센트
- 볼린저 밴드 : 이평선 중심 플마 표준편차

**추세반전형 지표**

- 스토캐스틱 : 일정 기간 편동폭 범위 중 현재의 위치 (%K), %K의 이평선 (%D)
- RSI : 개별주와 업종의 상대강도, 75% 이상 과매수 25% 이하 과매도

    failure swing : 고/저점을 돌파하지 못하는 경우

- ROC : 종가와 n일 전 주가의 차이

## 5. 산업분석 (3~4)

### 01 산업분석

구분 : 경기민감산업, 경기방어산업 / 경기선행산업, 경기동행산업, 경기후행산업

지수 : 경기선행지수, 경기동행지수, 경기후행지수

**산업구조 변화**

- 차이 발생 원인 : 수요변화 차이, 공급능력 차이, 외부적 요인
- 절대적 이론 : 리카도의 비교우위론 (노동), 헥셔-올린 모형 (노동, 자본)
- 상대적 이론 : 제품수명주기이론(기술혁신), 신무역이론(정부개입), 내생적 성장이론(요소창출)
- 성장기 → 1차 전환점 → 구조조정기 → 2차 전환점 → 성숙기

**산업 간 성장률 격차**

- Petty 법칙 : 경제 발전에 따라 산업구조 변화
- Hoffman 법칙 : 발전에 따라 생산재 산업 비중 증가

### 02 산업연관 분석

**산업연관표 (**가로 = 배분구조, 세로 = 투입구조)

- 산업간의 연관관계를 수량적으로 파악
- 중간생산물 거래도 포괄
- 총투입액 = 총산출액 = 총수요액 - 수입

세로방향 지표

- 중간투입계수 = 중간투입액 / 총투입액
- 부가가치계수 = 보가가치액 / 총투입액

가로방향 지표

- 생산유발계수 : 전방연쇄효과(모→특), 후방연쇄효과(특→모)
- 수입유발계수, 고용유발계수, 부가가치 유발계수

### 03 순환 분석

**제품수명주기이론 :** 도입기 - 성장기 - 성숙기 - 쇠퇴기

**Lynch사의 경기순환** : 강세 (초기-중기-후기) - 약세 (초기-후기)

### 04 산업경쟁력 분석

의의 : 한 국가의 특정 산업이 다른 국가와의 경쟁에서 성과를 얻는 정도

이론 : 국제무역이론, 경쟁우위론, 경쟁력 분석

분석모형

- 경쟁자산 : 축적 정도
- 시장구조 : 협력, 분업도, 산업특성, 규제, 정책
- 산업성과 : 현재의 경쟁력
- 기업의 역할 : 경영능력, 투자개발
- 정부의 역할 : 인프라 확충, 인력개발
- 경쟁국의 전략

### 05 산업정책 분석

**산업정책 특징**

- 총공급관리정책 ( 산업정책 ↔ 통화/재정정책)
- 후발국에서 강조, 국가별 경제발전 수준에 따라 달라짐

**산업구조정책**

- 산업 간 구성비율을 조정
- 일반적 / 기능별 / 지역별 / 산업별 / 기업열 정책
- 유인 / 규제 / 비전 제시정책

**시장 경쟁강도 측정**

- 시장집중률지수 (CR_k) : 상위 k개 기업의 시장점유율 합
- 허핀달지수 (HHI) : 클수록 기업규모의 불균등정도가 큼

## 6. 리스크 관리 (8)

### 01 리스크 관리 개론

리스크 : 미래수익의 불확실성

재무위험 : 시장위험, 신용위험, 운영위험, 유동성위험, 법적위험

**베어링은행 사례**

- 파생상품 담당자의 권한 밖의 불법거래로 베어링은행이 파산
- 내부관리 통제의 필요성 대두

**메탈게젤샤프트사 파산사건**

- 선도 공급계약에서의 Rolling Hedge의 결합 간과로 파산
- 선물 시장에 대한 이해 부족

**오렌지카운티 파산사건**

- 재정담당관이 이자율 예측에 실패하여 지방 재정이 파산
- VaR등을 이용한 리스크 측정의 필요성 대두

### 02 시장리스크

**VaR (위험가치)**

정의 : 시장이 불리한 방향으로 움직일 경우 일정기간동안의 최대손실 가능액

ex) 1일동안 VaR이 신뢰구간 95%에서 10억으로 추정됨

종류 : 부분가치 평가 (델타) / 완전가치 평가 (역사적, 스트레스, 몬테카를로)

의의 : 리스크를 수치화함

사용 : RAROC = 순수익 / VaR (성과측정 지표)

**델타분석법**

관련 리스크요인을 결정 → 요인의 변동성과 상관관계 추정 → 부분가치평가

- 델타 노말 분석법 : 선형 관계로 가정

$$
주식 VaR = (V \cdot Z_{\alpha} \cdot \sigma \cdot \sqrt{T}) \cdot \beta \\
채권 VaR = (V \cdot Z_{\alpha} \cdot \sigma \cdot \sqrt{T}) \cdot MD \\
옵션 VaR = (V \cdot Z_{\alpha} \cdot \sigma \cdot \sqrt{T}) \cdot \Delta \\
-\\ 95\% =1.65, \quad 99\% = 2.33

$$

(위험노출금액, Z값, 1일 변동성, VaR 추정기간 - 단, 기간은 주말을 제외한 거래일 기준)

- 델타 감마 분석법 : 수익구조가 비선형인 경우(ex) 옵션) 보완하기 위해 사용

$$
\text{VaR}_{\text{포트폴리오}} = \sqrt{\sum_{i=1}^n \sum_{j=1}^n w_i w_j \text{VaR}_i \text{VaR}_j \rho_{ij}}
$$

**역사적 시뮬레이션 분석법 :** 과거의 데이터로 완전가치평가

**몬테카를로 시뮬레이션 분석법 :** 위험요인의 확률모형으로 완전가치평가

**스트레스 검증법 :** 최악의 상황 (or 20% 이상의 시장변동성)에서 포트폴리오의 변화 측정, 보조적 이용

→ 셋 다 정규분포를 전제하지 않아 비선형도 측정가능

### 03 신용리스크

특징 : 신용리스크는 정규분포를 이루지 않아 비모수적으로 측정 (ex) 채권)

의의 : ‘예상 외 손실’ 을 의미

**EDF 모형**

의의 : 미래 특정시점의 도산가능성(자산가치 < 부채인 시점)을 예측

- 기대채무불이행빈도 (EDF)
- 부도거리 = (기대자산가치 - 부채가치) / 표준편차 → 채무불이행점으로부터의 거리

**Default 모형**

의의 : 부도발생을 신용손실로 간주

측정 : 기대손실금액, 변동성금액

- EAD : 신용리스크에 노출될 금액
- 부도율 : 일정 기간동안 부도가 날 확률
- LGD : 부도 시 경제적 손실의 크기

기대손실금액

$$
EL = EAD \cdot 부도율\cdot LGD
$$

변동성금액

$$
\sigma_{EL} = EAD \cdot \sqrt{부도율(1-부도율)}\cdot LGD
$$

**MTM 모형**

의의 : 부도발생 + 신용등급변화를 신용손실로 간주

측정 : Credit VaR
