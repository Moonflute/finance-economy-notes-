# 용어사전

중요 개념을 빠르게 리마인드하기 위한 한 줄 사전이다. 각 챕터를 읽다가 용어의 위치를 잃었을 때 여기서 먼저 감을 잡고, 세부 설명은 해당 본문으로 돌아가 확인한다.

## 경제

GDP : 일정 기간 한 나라 안에서 새로 생산된 최종 재화와 서비스의 시장가치.

명목 GDP : 당해 연도 가격으로 계산한 GDP로, 물가 변동이 함께 반영된다.

실질 GDP : 기준연도 가격으로 계산한 GDP로, 생산량 변화에 초점을 둔다.

GDP 디플레이터 : 명목 GDP를 실질 GDP로 나눈 값으로 경제 전체 물가수준을 나타낸다.

GDP 갭 : 실제 GDP와 잠재 GDP의 차이로 경기 과열 또는 침체 압력을 판단한다.

잠재 GDP : 노동과 자본을 정상적으로 활용할 때 지속 가능한 산출 수준.

총수요 AD : 가계, 기업, 정부, 해외가 각 물가수준에서 구매하려는 총지출.

총공급 AS : 기업들이 각 물가수준에서 생산하려는 총산출.

수요견인 인플레이션 : 총수요 증가가 물가 상승을 유발하는 인플레이션.

비용인상 인플레이션 : 원자재, 임금 등 생산비 상승이 물가 상승을 유발하는 인플레이션.

스태그플레이션 : 경기 침체와 물가 상승이 동시에 나타나는 상태.

필립스 곡선 : 단기적으로 실업률과 인플레이션 사이의 상충관계를 설명하는 곡선.

자연실업률 : 경기 변동과 무관하게 구조적·마찰적 요인으로 존재하는 실업률.

IS 곡선 : 생산물시장의 균형을 나타내며 이자율과 국민소득의 음의 관계를 보인다.

LM 곡선 : 화폐시장의 균형을 나타내며 이자율과 국민소득의 양의 관계를 보인다.

통화정책 : 중앙은행이 기준금리, 공개시장운영, 지급준비율 등을 통해 통화량과 금리를 조절하는 정책.

재정정책 : 정부가 지출과 조세를 통해 총수요와 소득을 조절하는 정책.

구축효과 : 정부지출 증가가 금리를 올려 민간투자를 밀어내는 현상.

승수효과 : 지출 증가가 소득과 소비를 연쇄적으로 증가시키는 효과.

구매력평가 PPP : 장기적으로 환율이 각국 물가수준의 차이를 반영한다는 이론.

이자율평가 IRP : 국가 간 금리 차이가 선물환율 또는 기대환율에 반영된다는 이론.

경상수지 : 상품, 서비스, 본원소득, 이전소득 거래의 대외수지.

자본수지 : 자본이전과 비생산·비금융자산 거래를 기록하는 국제수지 항목.

경기선행지표 : 실제 경기보다 먼저 움직여 향후 경기 방향을 예고하는 지표.

경기동행지표 : 현재 경기 흐름과 함께 움직이는 지표.

경기후행지표 : 경기 변동 이후 뒤따라 움직이는 지표.

## 금융

주식 : 기업의 소유권 지분을 나타내며 배당과 시세차익을 기대하는 증권.

보통주 : 의결권과 잔여재산청구권을 가진 일반적인 주식.

우선주 : 의결권은 제한될 수 있으나 배당이나 잔여재산 분배에서 우선권을 갖는 주식.

PER : 주가를 주당순이익으로 나눈 배수로 이익 대비 주가 수준을 본다.

PBR : 주가를 주당순자산으로 나눈 배수로 장부가치 대비 주가 수준을 본다.

EV/EBITDA : 기업가치를 EBITDA로 나눈 배수로 영업현금창출력 대비 가치 수준을 본다.

DCF : 미래 현금흐름을 적정 할인율로 현재가치화해 기업가치를 평가하는 방법.

DDM : 미래 배당을 할인해 주식가치를 평가하는 배당할인모형.

CAPM : 무위험수익률, 베타, 시장위험프리미엄으로 기대수익률을 설명하는 모형.

베타 : 개별자산 수익률이 시장수익률에 얼마나 민감하게 움직이는지 나타내는 지표.

알파 : CAPM 등 기준모형으로 설명되지 않는 초과수익.

채권 : 발행자가 투자자에게 원금과 이자를 지급하기로 약속한 채무증권.

표면금리 : 채권 액면가에 대해 약정된 이자율.

만기수익률 YTM : 채권을 만기까지 보유할 때 기대되는 내부수익률.

듀레이션 : 채권 현금흐름의 가중평균 회수기간이자 금리 민감도 지표.

수정듀레이션 : 금리 1%p 변화에 대한 채권가격 변화율을 근사하는 지표.

볼록성 : 채권가격과 수익률 관계의 곡률로 듀레이션의 오차를 보정한다.

수익률곡선 : 만기별 채권수익률을 연결한 곡선으로 금리 기대와 위험프리미엄을 반영한다.

신용스프레드 : 신용위험이 있는 채권수익률과 무위험채권수익률의 차이.

선물 : 미래 특정 시점에 기초자산을 정해진 가격으로 사고팔기로 한 표준화 계약.

베이시스 : 선물가격과 현물가격의 차이.

옵션 : 기초자산을 정해진 가격에 사거나 팔 수 있는 권리.

콜옵션 : 기초자산을 정해진 가격에 살 수 있는 권리.

풋옵션 : 기초자산을 정해진 가격에 팔 수 있는 권리.

내재가치 : 옵션을 즉시 행사했을 때 얻을 수 있는 경제적 가치.

시간가치 : 만기까지 남은 불확실성에 대해 시장이 부여하는 옵션 가치.

델타 : 기초자산 가격 변화에 대한 옵션가격의 민감도.

감마 : 기초자산 가격 변화에 대한 델타의 민감도.

세타 : 시간 경과에 따른 옵션가격의 민감도.

베가 : 변동성 변화에 대한 옵션가격의 민감도.

스왑 : 두 당사자가 미래 현금흐름을 교환하기로 하는 장외파생계약.

ETF : 특정 지수나 자산군을 추종하며 거래소에서 주식처럼 거래되는 펀드.

ELS : 주가지수나 개별주식을 기초자산으로 수익조건이 정해진 주가연계증권.

REITs : 부동산이나 부동산 관련 자산에 투자하고 임대료·매각차익을 배당하는 투자기구.

ABS : 기초자산의 현금흐름을 바탕으로 발행되는 자산유동화증권.

MBS : 주택담보대출채권을 기초자산으로 발행되는 유동화증권.

## 재무설계

재무설계 : 개인의 재무·비재무 목표 달성을 위해 자원을 체계적으로 배분하고 관리하는 과정.

위험성향 : 투자자가 손실 가능성을 심리적으로 얼마나 받아들일 수 있는지 나타내는 특성.

위험수용능력 : 소득, 자산, 투자기간 등 객관적 조건상 감내 가능한 위험 수준.

자산부채상태표 : 특정 시점의 자산, 부채, 순자산을 정리한 개인 재무제표.

현금흐름표 : 일정 기간의 현금 유입과 유출을 정리해 저축·투자 여력을 파악하는 표.

유동성비율 : 단기 지출이나 비상상황에 대응할 현금성 자산의 충분성을 보는 비율.

부채상환비율 : 소득 대비 원리금 상환 부담을 평가하는 지표.

저축률 : 소득 중 소비하지 않고 저축·투자하는 비율.

생애주기 재무설계 : 연령과 가족상황 변화에 맞춰 주택, 교육, 은퇴, 상속 목표를 조정하는 설계.

은퇴설계 : 은퇴 후 필요소득, 기대수명, 물가, 투자수익률을 고려해 준비자금을 계산하는 과정.

국민연금 : 국가가 운영하는 공적연금으로 노령, 장애, 사망 위험에 대비한다.

퇴직연금 DB : 사용자가 운용 책임을 지고 근로자의 퇴직급여가 사전에 정해지는 제도.

퇴직연금 DC : 근로자가 운용 책임을 지고 부담금과 운용성과에 따라 급여가 달라지는 제도.

IRP : 개인형 퇴직연금으로 퇴직급여 이전과 추가 납입을 통해 노후자금을 관리하는 계좌.

ISA : 여러 금융상품을 한 계좌에서 운용하고 요건 충족 시 세제혜택을 받는 개인종합자산관리계좌.

금융소득 : 이자소득과 배당소득을 합친 소득.

종합과세 : 여러 소득을 합산해 누진세율을 적용하는 과세 방식.

분류과세 : 특정 소득을 다른 종합소득과 분리해 별도로 과세하는 방식.

분리과세 : 원천징수 등으로 과세관계를 종결하거나 별도 세율로 과세하는 방식.

양도소득 : 자산 양도로 발생한 차익에 대한 소득.

상속세 : 사망으로 재산이 이전될 때 상속재산에 부과되는 세금.

증여세 : 생전 무상 이전으로 재산을 취득한 자에게 부과되는 세금.

세후수익률 : 세금을 차감한 뒤 실제 투자자가 얻는 수익률.

## 법률

자본시장법 : 금융투자상품과 금융투자업, 공시, 불공정거래 등을 규율하는 법률.

금융투자상품 : 원본손실 가능성이 있는 투자성 금융상품.

증권 : 채무증권, 지분증권, 수익증권 등 권리가 표시된 금융투자상품.

파생상품 : 기초자산의 가격, 이자율, 지표 등에 따라 가치가 변하는 계약.

전문투자자 : 투자 위험 감수능력과 전문성이 높다고 보아 일부 보호규제가 완화되는 투자자.

일반투자자 : 전문투자자가 아닌 투자자로 더 강한 투자자보호 규제를 받는다.

적합성 원칙 : 투자자의 지식, 경험, 재산상황, 투자목적에 맞는 상품만 권유해야 한다는 원칙.

적정성 원칙 : 권유가 없더라도 투자자에게 부적정한 상품이면 위험을 고지해야 한다는 원칙.

설명의무 : 금융상품의 구조, 위험, 수수료, 손실 가능성 등을 이해할 수 있게 설명해야 하는 의무.

불공정영업행위 : 금융상품 판매 과정에서 우월적 지위나 부당한 조건을 이용하는 행위.

부당권유 : 오인하게 하거나 단정적 판단을 제공하는 등 부적절한 방식의 권유.

청약철회권 : 일정 요건과 기간 안에 금융상품 계약 청약을 철회할 수 있는 소비자 권리.

위법계약해지권 : 판매규제 위반으로 체결된 계약을 소비자가 해지할 수 있는 권리.

미공개중요정보 이용 : 공개되지 않은 중요한 정보를 이용해 증권거래를 하는 불공정거래.

시세조종 : 인위적으로 가격이나 거래량을 조작해 시장가격을 왜곡하는 행위.

부정거래 : 기망, 풍문, 위계 등을 이용해 투자자를 속이거나 시장을 교란하는 행위.

내부통제 : 법규 준수와 위험관리를 위해 회사 내부에 마련하는 기준과 절차.

이해상충 : 회사, 임직원, 고객 사이의 이익이 충돌하는 상황.

정보차단장치 : 이해상충 방지를 위해 부서 간 정보 흐름을 제한하는 내부통제 장치.

민법 : 사인 간 권리·의무 관계의 기본 원칙을 정한 법.

물권 : 특정 물건을 직접 지배해 이익을 얻는 배타적 권리.

채권 : 특정인에게 일정 행위를 청구할 수 있는 권리.

계약 : 둘 이상의 의사표시 합치로 권리와 의무를 발생시키는 법률행위.

상법 : 기업 활동과 회사, 상행위에 관한 법률관계를 규율하는 법.

주식회사 : 주주가 출자한 자본을 바탕으로 하고 주주는 인수가액 한도에서 책임지는 회사.

유한책임 : 투자자가 출자한 범위 안에서만 책임을 부담하는 원칙.

상속순위 : 법정상속인이 상속받는 순서로 직계비속, 직계존속, 형제자매, 4촌 이내 방계혈족 순서가 기본이다.

유류분 : 일정 상속인에게 법적으로 보장되는 최소한의 상속재산 몫.
